## 3. P0 / P1 / P2 정의 (통합)

### P0 — 즉시 폐기 (REGENERATE)

```text
One Figure = One 계측 목적
가시설 3분할: 배면 지반 | 벽체 | 굴착측
띠장·버팀보·앵커 두부의 지중·배면 지반 배치 금지
어스앵커 하중계 = 굴착측 두부, 반력판–하중계–앵커헤드 사이, 앵커 설치각과 동축
지하수위계 ≠ 간극수압계 (동일 관·동일 수위선 금지)
센서·측점·계측기·시스템 혼동 (침하핀 라벨을 지표침하계에 붙임 등)
단면·입면·평면 무라벨 혼합 (VIEW-01)
```

### P1 — 전면 수정

```text
건물 포함 Figure: 출입문·1층 = 지표면 (P0-1 조건부)
IPI·지하수위 관로 GL 연속 (P0-2)
데이터로거 = 함체(인클로저) — 범례·빈 박스·접속함만 금지 (센서≠로거 혼동은 P0)
설치도에 서버·그래프·관리기준 혼입
```

### P2 — 부분 수정·스타일

```text
로거 외형 세부 (CR1000X는 예시일 뿐)
IPI 본도 지시선 단축 「지중경사계」「IPI」
색상·선 굵기 미세 · 출판 visualReview
```

**코드:** REJECT R-01~R-83 · DIS-01~80 · DP-01~80는 **P1 이하 상세** — **P0 위반 시에만** 생성·승인 금지.

## 5. Cursor 프롬프트 — 최상단 고정 (10줄)

→ [36 §1.0](./36-AI-이미지-생성-엔지니어링-프롬프트-가이드.md) · [51 §9](./51-계측-도면-검수-공통-원칙.md) — **아래 블록을 맨 앞에 붙인 뒤** `이 그림의 계측 목적은 단 하나다: ____` 한 줄 추가.

```text
이 그림은 토목 계측 기술도면이다.
한 그림에는 하나의 계측 목적만 담는다.
센서·측점·계측기·시스템을 혼동하지 않는다.
센서는 실제 설치 위치에만 배치한다.
기준점·측정 방향·측정 물리량을 반드시 표시한다.
가시설 단면은 배면 지반 | 벽체 | 굴착측으로 나눈다.
띠장·버팀보·앵커 두부를 흙 속에 넣으면 폐기한다.
어스앵커 하중계는 굴착측 두부의 반력판–하중계–앵커헤드 사이, 앵커 축과 동축이다.
수위계와 간극수압계를 같은 장치로 그리면 폐기한다.
틀릴 것 같으면 그리지 말고 별도 그림으로 분리한다.
```
