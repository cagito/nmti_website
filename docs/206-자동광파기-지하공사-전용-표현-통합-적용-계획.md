# 자동광파기(ATS) — 지하공사 전용 표현 통합 적용 계획 (ATS-SUB-01)

**수립:** 2026-06-30  
**요청:** 자동광파기는 **지반 하부에 공사할 때**를 제외하고 Figure·본문·프롬프트에 넣지 않는다. 실무에서도 범용 수단이 아니다.  
**상위 정책:** [127 DISP-ATS-01](./127-변위계측-자동광파기-남용방지-및-와이어식-우선-표준.md) **강화** — 본 계획이 **127 §2 DISP-04~08** 을 구체화한다.  
**통합:** [128 디자인 원칙 통합](./128-디자인-원칙-문서-통합-반영-실행계획.md) · [TERMINOLOGY](./TERMINOLOGY.md)

## 0. 한 줄 정책 (ATS-SUB-01)

```text
자동광파기(ATS)·Total Station 본체는 「지반 하부 공사」Figure·본문에만 허용한다.
지상 구조물·사면·교량·건축·항만·철도 노반·유지관리 Figure에는 ATS를 그리지·쓰지 않는다.
프리즘(반사 측점)만 필요한 경우 ATS 본체 없이 측점·핀·수준만 표기할 수 있다.
ATS 장비 설명은 sensors/automated-total-station(IMG-042) 노드에만 정식 hero로 둔다.
```

**등급:** **P1** (표현·교육 정책) — 물리 오류 P0와 별도. 위반 시 **DOC_FIX** → Figure는 **REGENERATE**.

## 2. 「지반 하부 공사」정의 (허용 맥락)

### 2.1 허용 (ATS 본체·시준선·광학망)

| 맥락 | 노드·Figure 예 | 근거 |
|------|----------------|------|
| **터널 굴착·NATM** | `fields/tunnel/*` · IMG-008·061·111 | 지하 굴착·지보 |
| **터널 상부 지표 영향** | IMG-010 (터널 상부 침하·천단 연계) | 굴착 영향권 계측 |
| **흙막이·깊은 굴착** | `fields/retaining-excavation` (**굴착 본체**) · IMG-002·006 | 지하 굴착 공사 |
| **ATS 센서 설명** | `sensors/automated-total-station` · **IMG-042** | 장비 카탈로그 1종 |

### 2.2 금지 (ATS 본체·「자동광파기」라벨·시준선 hero)

| 맥락 | 대체 수단 |
|------|-----------|
| 사면·비탈면 (`fields/slope`) | IPI · 지표경사계 · **프리즘 측점**(광학망 목적 아님) · 와이어(배면) |
| 교량 (`fields/bridge`) | GNSS(103) · 처짐계/LVDT(104) · 지표침하계 · 경사계 |
| 건축 (`fields/building`) | LVDT · 와이어식 · 레이저 · 균열계 · 경사계 |
| 인접건물 (`adjacent-building`) | 균열계 · 구조물경사계 · **프리즘 측점** — **ATS 영향권 밖 배치도 금지** |
| 구조물 안전·항만·댐·철도 노반 | 센서형 우선 · GNSS(거시) |
| 전체 개념도 callout | ATS 칸 **삭제** |

### 2.3 경계 (문서에 명시)

- **프리즘·측점·천단핀·수준 BM** = 측량 **표식** — ATS 없이 표기 **가능** (도시철도 PDF 수동 측량 맥락).
- **「광파기」수동 1회 측량** vs **「자동광파기」연속망** — 후자만 ATS-SUB-01 적용 대상.
- `retaining-excavation/adjacent-building` = **지상 건물** — **지하 공사 아님** → ATS **금지**.
