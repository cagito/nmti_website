## 4. 추가 구현 계획

### Phase 0 — [58번 계획](58-교량-처짐-케이블장력-신규-추가-계획.md) 실행 (P0)

| nodeId | 라벨 | Figure |
|--------|------|--------|
| `fields/bridge/deflection` | 처짐 | IMG-103 |
| `sensors/deflection-gauge` | 처짐계 | IMG-104 |
| `fields/bridge/cable-tension` | 케이블 장력 | IMG-105 |
| `sensors/cable-tension-meter` | 케이블장력계 | IMG-106 |

**준공보고서 정합 포인트 (콘텐츠에 반영):**

- 처짐 관리기준: **L/600** 예시(§5.3.6) — 그래프 Y=처짐량 δ, 「현장별 설계 적용」
- 장력계: AC-310 / GV-2440 — **주파수 분석→장력** (§2.4.19~20)
- 신축이음: 준공보고서 「장력계」와 **하중계(앵커) 혼동 금지** — 58·60 표준과 동일
