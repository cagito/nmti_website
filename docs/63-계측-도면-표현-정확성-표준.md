# 계측 도면 표현 정확성 표준 (정본)

**등급:** **P0 보조** — **그럴듯하지만 부정확한** AI/Cursor 표현 차단  
**역할:** [51 §3 DP-51~60](./51-계측-도면-검수-공통-원칙.md) · [§4 DIS-51~60](./51-계측-도면-검수-공통-원칙.md) 상세  
**범위:** 대표 단면·전체 개념도·선·케이블·측량·부착·자동/수동·안정층·설치도 혼입·변위계류

> **한 줄:** **단순하게 그리되, 선 하나·센서 하나도 의미 없이 넣지 마라.**  
> **재작성 전 문서 정본** — 지금 Figure 생성·승인하지 않음.

## 0. 강제 지시문 (프롬프트 · 재작성 시)

```text
대표 단면도에는 실제 가능한 단면만 넣어라.
전체 개념도라도 센서 위치는 틀리면 안 된다.
수위선, 케이블, 시준선, 기준선은 서로 다르게 그려라.
케이블을 구조부재처럼 그리지 마라.
측량 장비와 센서를 분리해라.
구조물 센서는 부착면을 명확히 해라.
자동형과 수동형을 같은 방식처럼 그리지 마라.
안정층은 기준 조건으로 작동해야 한다.
설치도에 관리기준·알람·서버를 넣지 마라.
모든 변위계를 같은 장비처럼 그리지 마라.
```

## 1. DP-51 — 대표 단면 = 실제 가능 구성 (REP-01)

**원칙:** 대표 단면도 = **한 단면에 존재 가능한** 구성 — 개념 모음집 **아님**

**금지:** 앵커식+버팀보식+옹벽+사면+성토 **한 단면 혼합**

**분리:** 앵커식 / 버팀보식 / 주변지반 — 별도 Figure · 실제 혼합형만 **`혼합형 예시`** 명시

(연계: IMG-002 · MIX-01 · [52](./52-IMG-002-전면재작성-프롬프트-정본.md))

---

## 2. DP-52 — 전체 개념도도 설치 위치 정확 (CONC-01)

**원칙:** 전체 개념도 ≠ **보기 좋은 임의 배치** — 각 센서 **설치 가능 위치만**

**폐기 예:** 토압계 굴착측 · 앵커 LC 지중 · 수위계 벽체 부착 · 균열계 균열 **평행**

(연계: DP-04 · INST · IMG-001 G-01~G-15)

---

## 3. DP-53 — 선 의미 단일 (LNT-01)

| 선 | 의미 |
|----|------|
| **실선** | 구조물·센서·계측선 |
| **파란 점선** | 수위선 (G.W.L) |
| **빨간 점선** | 위험선·**추정** 활동면 |
| **회색 점선** | 보조 기준선 |
| **청록 화살표** | 변위·측정 방향 |

**금지:** 빨강=활동면인지 경보인지 불명 · 파랑=수위인지 데이터 흐름인지 · 청록 점선=시준인지 케이블인지

(연계: DP-35 · DP-48 · CLR-01)

---

## 4. DP-54 — 케이블 ≠ 구조 부재 (WIR-01)

| 요소 | 선형 |
|------|------|
| **강연선** | 다발 또는 **굵은 사선** |
| **버팀보** | 구조부재 **굵은 실선** |
| **케이블** | **얇은 곡선** |
| **시준선** | 가는 점선 |
| **기준선** | 얇은 보조선 |

**금지:** 데이터 케이블=앵커 강연선 · 시준선=버팀보 · 케이블=지보재

---

## 5. DP-55 — 측량 장비 ≠ 센서 (SVY-01)

| 구분 | 역할 |
|------|------|
| **센서** | 물리량 **직접** 계측 |
| **측량 장비** | 측점 **좌표·표고** 관측 |

**원칙:** 광파기 본체=**안정 지점** · 프리즘=**거동 대상** · 침하핀=**측량 측점**

**금지:** 프리즘→로거 · 광파기 건물 위 · 레벨=지반 변위 센서

(연계: DP-03 · DP-05 · R-04 · R-05 · IMG-043)

---

## 6. DP-56 — 구조물 센서 부착면 (ATCH-01)

| 장비 | 부착 |
|------|------|
| **균열계** | 균열 **양측 모재** · 균열에 **직각** |
| **경사계** | 견고한 **외벽·기둥·브래킷** |
| **변형률계** | 강재·콘크리트 **표면·매립** |
| **프리즘** | 외벽·기둥 **측점** |

**금지:** 센서 공중 부양 · 균열선 위만 · 경사계 창문·마감·허공

(연계: DP-04 · DP-06 · IMG-005)

## 7. DP-57 — 자동·수동 시각 분리 (AMN-01)

| 유형 | 표현 |
|------|------|
| **자동형** | 케이블·로거·전원 |
| **수동형** | 측점·측량기·**기록 절차** |

**금지:** 침하핀→로거 · 수동 관측공=자동센서 · 자동센서인데 케이블·함체 없음

(연계: AUTO-01 · SETTLE-01 · AMN-01 · MAN-01 · IMG-070)

## 8. DP-58 — 안정층 = 기준 조건 (STBL-01)

**원칙:** 안정층 = **기준 심도·근입 대상** — 장식 라벨 **금지**

**필수:** IPI Base **안정층 도달** · 기준점 **영향권 밖** · 침하 기준=**부동점**

**금지:** 안정층 라벨만·센서 미도달 · 기준점 연약층 · 근입 **임의 m** 일반화

(연계: [51 §5](./51-계측-도면-검수-공통-원칙.md) · DIM-01 · R-05 · R-09)

---

## 9. DP-59 — 설치도 ≠ 관리기준 (GRF-01)

| Figure | 내용 |
|--------|------|
| **설치도** | 위치·심도·방향 |
| **해석도** | 그래프·관리기준 |
| **시스템도** | 경보·알림 |

**금지:** 설치 단면에 **관리기준 그래프·알람·서버·모바일** (DP-33·SIM-03 보강)

## 10. DP-60 — 변위계류 장비 구분 (CLS-DISP)

| 장비 | 핵심 |
|------|------|
| **균열계** | 균열폭 |
| **신축계** | 두 점 **상대변위** |

| **지중경사계** | 심도별 **수평변위** |
| **구조물경사계** | 기울기 **θ** |

**금지:** 위 장비 **동일 아이콘** · 「일반 변위계」 뭉개기

(연계: CLS-01 · DP-43 · R-07)
