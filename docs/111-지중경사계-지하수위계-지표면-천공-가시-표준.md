# 지중경사계·지하수위계 — 지표면 천공 가시 표준 (정본)

**규칙 ID:** **BORE-GL-01** (P0-2 / C0d 확장)  
**등급:** **P0 치명** — 단면·설치도 **전 Figure**  
**연계:** [27-지표면-건물-안착-원칙 §2 C0d](./27-지표면-건물-안착-원칙.md) · [09-IPI 표준](../ImageWorks/NMTI_Engineering_Image_Prompt_Package_v1/09_지중경사계_센서형-다단식_표현_표준.md) · [INSTRUMENTATION §3.3·3.4](./INSTRUMENTATION_DRAWING_RULES.md)

> **한 줄:** 지중경사계·지하수위계는 **지표면(GL)에서 천공**하고, 그림에서 **지표면까지 천공 개구부가 반드시 보여야** 한다.

---

## 0. 강제 지시문 (프롬프트 맨 위 · 복붙)

```text
센서형 다단식 지중경사계와 지하수위계는 지표면(GL)에서 천공한다.
단면도에서 지표면까지 천공부위(케이싱 상단·well cap·지표 보호함)가 반드시 가시적으로 표현되어야 한다.
지층 중간 깊이에서만 관이 시작하거나, 지표면 개구부 없이 지하에만 그리면 폐기한다.
관측공·케이싱은 GL에서 아래로 연속이다. 공중 부양·GL 끊김 금지.
```

---

## 1. 적용 대상

| 계측기 | 적용 Figure |
|--------|-------------|
| **센서형 다단식 지중경사계** | IMG-001·002·015·025~029·027·096·064·070·077 등 **모든 지중 천공 단면** |
| **지하수위계** | IMG-030·062·096·002·항만·사면·가시설 단면 **전부** |

**해당 없음:** 간극수압계(밀폐 천공·필터 tip) — GL 개방 관측공과 **이형** (§3).

---

## 2. 필수 표현 (가시 요소)

### 2.1 공통

| # | 필수 |
|---|------|
| B1 | **「지표면」** 또는 **GL** 라벨·경계선이 단면에 명시 |
| B2 | 천공 관로가 **지표면에서 시작**하여 아래로 **연속** |
| B3 | **지표면에서 천공 개구부가 그림에 보임** — 상단이 GL 아래에 묻혀 보이지 않음 |
| B4 | 지층 **중간 깊이에서만** 관·케이싱이 갑자기 시작 **금지** |

### 2.2 센서형 다단식 지중경사계

| 요소 | 지표면에서 보여야 할 것 |
|------|-------------------------|
| **케이싱(경사관)** | 수직 관이 **GL에서 관통**하여 지하로 연속 |
| **지표 보호함·캡** | 케이싱 상단 **보호함·캡·맨홀** — GL 또는 GL 직상 |
| **내부** | GL 아래 **다점 센서 노드** + 케이블 (침하계형 단일 막대 금지) |

### 2.3 지하수위계

| 요소 | 지표면에서 보여야 할 것 |
|------|-------------------------|
| **관측공·보호관** | **GL에서 천공** · 개방 관로가 지층 속으로 연속 |
| **well cap / surface protective casing** | **지표면에 부착**된 캡·보호관 상단 — **필수 가시** |
| **스크린·필터** | GL **아래** 구간 — 캡과 구분 |
| **G.W.L** | GL **아래** 관측공 **내부** 수면 |

---

## 3. 금지 (C0d · BORE-GL-01)

| 금지 | 이유 |
|------|------|
| 케이싱·관측공이 **지층 중간**에서만 시작 | 천공·설치 현실과 불일치 |
| 지표면 **개구부·캡 없음** — 지하에만 관 표시 | 천공부위 비가시 |
| 관로가 **지표면 위 공중 부양** | GL 미접속 |
| GL에서 **끊긴** 관·케이싱 | 연속성 위반 |
| 지하수위계를 **벽체 부착 센서**처럼 표현 | 장치 혼동 |
| IPI·지하수위를 **동일 관** 기호 | EXC-03 |

---

## 4. 검수 Q (Q-BORE-GL)

1. **지표면** 선이 보이는가?
2. IPI 케이싱 **상단·보호함**이 **지표면에서** 보이는가?
3. 지하수위계 **well cap·보호관 상단**이 **지표면에서** 보이는가?
4. 관로가 GL **아래로만** 연속되는가? (중간 시작·부양 없음)
5. 간극수압계와 **동형 관측공**이 아닌가?

**하나라도 「아니오」→ REGENERATE.**
