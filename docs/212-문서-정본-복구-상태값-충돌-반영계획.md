**상태값 원칙** (180·181·prompt 공통):

```text
현재 상태 = PASS | MAJOR_FIX | REGENERATE 중 하나.
과거 REGENERATE → CANONICAL_STATUS 「과거·resolved」 또는 ARCHIVE.
한 문서에서 PASS와 REGENERATE를 동시에 「현재」로 쓰지 않는다.
PASS여도 prohibitedErrors는 삭제하지 않는다.
```
