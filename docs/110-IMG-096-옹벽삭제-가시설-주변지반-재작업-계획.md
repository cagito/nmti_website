## 2. 반드시 삭제 (v4 체크리스트)

```text
삭제 대상
├ 옹벽 (전면 삭제)
├ 옹벽 기초
├ 옹벽 배면 토압 개념
├ 불필요한 사면 원호·잠재 슬립면
├ H/2H를 옹벽 높이처럼 보이게 하는 표현
├ 성토부 계측처럼 보이는 Sand Mat, 침하판 중심 구성
├ 데이터 흐름도 · 서버 · 모바일 · 알람 UI
└ 센서 공중 부유 아이콘
```

**유지 가능:** 굴착선 · **H = 굴착깊이** · **H~2H = 계측 배치 검토 범위 예시** (고정 기준 아님)

## 3. 필수 계측 배치

| 계측 항목 | 권장 라벨 | 역할 |
|-----------|-----------|------|
| 배면 수평변위 | **센서형 다단식 지중경사계** | 굴착에 따른 배면 지반 수평변위 |
| 지표침하 | **지표침하 측점** / **지표침하핀** / 침하 측점 | 배면 지표 침하 (**지표침하계 금지**) |
| 지하수위 | **지하수위계** | 배면 관측공·수위선 |
| 간극수압 | **간극수압계** | 특정 심도·필터+차수 |
| (선택) 인접 구조물 | 균열계 · 구조물경사계 | 「선택 계측」으로 분리 |
| (선택) 자동 측량 | 프리즘 · 자동광파기 | 측량 측점 — 로거 직결 X |
