## 1. 공통 PASS 5게이트

모든 Figure는 아래 **5개를 모두** 만족해야 `reviewGrade: PASS` 가능.

| # | 게이트 | 검수 질문 |
|---|--------|-----------|
| G1 | **설치 위치** | INSTRUMENTATION · image-knowledge와 일치하는가? |
| G2 | **측정 물리량** | 노드·캡션·라벨의 물리량이 계측기 종류와 일치하는가? |
| G3 | **방향** | 힘·변위·수두·침하·장력·압축력 화살표·감지면이 맞는가? |
| G4 | **기준·축** | 기준점·대상점·측정축이 해석상 성립하는가? |
| G5 | **표현** | 확정/추정/예시·범례·대외 문구가 구분되는가? |

**즉시 FAIL (5게이트 외):** LOGO-01 · LEGEND-01 · EXT-01 · FOOT-01 · P0-4~10
