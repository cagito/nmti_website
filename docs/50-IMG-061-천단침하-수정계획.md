```text
- 터널 내 자동광파기·수준으로 기준점 대비 천단 표고 변화를 측정합니다.
- 외부 수준점은 수준측량 네트워크 기준으로 활용하되, 천단과 물리적 와이어로 연결하지 않습니다.
- 실링 숏크리트 후 측점을 설치하고 다음 굴착 전 초기치를 확정합니다(KCS 3.5.3).
- 내공변위·지표침하와 동일 단면·시간축에서 통합 검토합니다.
```
