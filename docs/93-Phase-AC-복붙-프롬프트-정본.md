## §1 IMG-007 — 터널 계측 전체 개념도 (REGENERATE)

```text
터널 계측 전체 개념도를 수정한다.

현재 그림은 터널 내공변위, 천단침하, 지중변위, 록볼트 축력, 숏크리트 응력, 강지보 응력, 지표침하를 한 단면에 넣었으나 각 계측 항목의 기준점, 측점, 측선, 설치 위치가 명확히 구분되지 않는다.

수정사항:
1. 계측 항목별로 "무엇을 측정하는지"를 명확히 분리한다.
2. 천단침하는 터널 내부 천단 측점의 표고 변화로 표현한다.
3. 내공변위는 좌우 측벽 또는 어깨부 측점 간 거리 변화로 표현한다.
4. 지중변위는 터널 주변 지반 또는 암반 내 다점변위계로 별도 표현한다.
5. 록볼트 축력은 록볼트 두부 하중 또는 계측용 록볼트로 구분한다.
6. 숏크리트 응력·강지보 응력은 각각 숏크리트 내부/표면, 강지보 부재에 설치된 계측기로 구분한다.
7. 각 계측항목에 측정 방향과 단위를 표시한다.
8. 하나의 센서가 터널 전체 거동을 모두 판단하는 것처럼 표현하지 않는다.

금지:
- 지중변위계를 록볼트처럼 표현하지 말 것.
- 천단침하와 내공변위를 같은 측선으로 표현하지 말 것.
- 숏크리트 응력계와 강지보 응력계를 같은 센서처럼 표현하지 말 것.

1920x1080, Korean civil instrumentation line drawing, white background.
```

## §3 IMG-023 — 철도 노반 계측 개념도 (REGENERATE)

```text
철도 노반 계측 개념도를 수정한다.

현재 그림은 레일 변위, 궤도 변위, 노반 침하, 지반 변위, 진동 계측이 한 그림에 혼합되어 있어 계측 대상이 불명확하다.

수정사항:
1. 계측 대상을 "궤도", "도상", "노반", "하부 지반", "인접 구조물"로 구분한다.
2. 궤도틀림은 레일의 고저, 방향, 궤간, 수평틀림으로 구분한다.
3. 노반침하는 침하 측점 또는 자동광파기·수준측량 개념으로 표현한다.
4. 진동계는 레일, 침목, 노반, 구조물 중 어느 위치에서 측정하는지 명확히 표시한다.
5. 지반 변위계는 노반 하부 지반 변형을 보는 계측으로 별도 구분한다.
6. 레일 변위와 노반 침하를 같은 계측값처럼 표현하지 않는다.
7. 데이터 그래프는 "궤도틀림", "침하", "진동"을 각각 분리한다.

금지:
- 레일 변위와 노반침하를 같은 센서로 표현하지 말 것.
- 진동계 단위를 침하량 단위와 섞지 말 것.
- 철도 계측 전체를 하나의 데이터 로거 흐름도로 단순화하지 말 것.

1920x1080, Korean CAD.
```

## §4 IMG-024 — 댐 계측 개념도 (REGENERATE)

```text
댐 계측 개념도를 전면 수정한다.

현재 그림은 누수와 침투를 제체 내부에 설치한 센서가 직접 계측하는 것처럼 보이므로 부정확하다.

수정사항:
1. 댐 단면은 상류측, 제체, 코어, 필터, 드레인, 기초, 하류측으로 구분한다.
2. 상류수위는 별도 수위계로 표시한다.
3. 간극수압계는 제체 또는 기초부 지정 심도에 설치한다.
4. 누수량 계측은 하류측 배수로, 드레인 출구, 집수정, 삼각위어, 유량계에서 측정하는 것으로 표현한다.
5. 탁도계가 있다면 누수수의 탁도 변화 확인용으로 하류 배수계통에 배치한다.
6. 침투선은 개념선으로 표시하되, "실측값"처럼 보이지 않게 한다.
7. 침하와 수평변위는 별도 계측 항목으로 표시한다.
8. 지진계는 제체 또는 기초부 동적응답 계측으로 분리한다.

금지:
- 누수계를 제체 내부에 박힌 센서처럼 그리지 말 것.
- 침투선과 지하수위를 같은 개념으로 표현하지 말 것.
- 간극수압계가 누수량을 직접 측정하는 것처럼 표현하지 말 것.
```

## §5 IMG-031 — 간극수압계 설치도 (MAJOR_FIX)

```text
간극수압계 설치도를 수정한다.

현재 그림은 간극수압계와 관측공 수위계가 혼동될 수 있으며, 필터구간·그라우트·차수구간이 불명확하다.

수정사항:
1. 간극수압계는 특정 심도의 간극수압을 측정하는 센서로 표시한다.
2. 센서 주변에는 필터사 또는 투수성 필터구간을 짧고 명확히 표시한다.
3. 필터구간 상하부에는 벤토나이트 또는 그라우트 차수구간을 표시한다.
4. 케이블은 보호관을 통해 지표면까지 연결한다.
5. 지하수위계와 비교할 경우, 지하수위계는 개방형 관측공 수위 측정으로 따로 표현한다.
6. 간극수압계 측정값은 수압 또는 수두로 환산 가능하되, 자유수면 수위와 동일한 개념이 아님을 명시한다.
7. 장심도 필터로 여러 지층을 연결하는 형태는 피한다.

금지:
- 간극수압계를 단순 수위계처럼 그리지 말 것.
- 필터구간을 전체 관측공 길이처럼 길게 표현하지 말 것.
- 그라우트가 센서 감지부를 막는 것처럼 표현하지 말 것.

1920x1080, Korean technical drawing.
```

## §6 IMG-033 — 층별침하계 개념도 (REGENERATE)

```text
층별침하계 개념도를 수정한다.

현재 그림은 자석링, 기준관, 측정 프로브의 역할이 불명확하여 각 층에 센서가 고정 설치되어 자동 측정되는 것처럼 오해될 수 있다.

수정사항:
1. 기준관은 고정된 측정 통로로 표현한다.
2. 자석링은 각 지층 또는 측정 심도에 설치되어 주변 지반과 함께 이동하는 링으로 표현한다.
3. 측정 프로브는 기준관 내부를 따라 내려가며 자석링 위치를 탐지하는 장치로 표현한다.
4. 층별침하량은 초기 자석링 위치와 현재 위치의 차이로 산정한다고 표시한다.
5. 총침하와 층별침하를 구분한다.
6. 기준점은 지표의 임의 지점이 아니라 안정 기준점 또는 기준 심도로 구분한다.
7. 자동계측 방식과 수동 프로브 측정 방식을 혼동하지 않는다.

금지:
- 자석링을 전기식 센서처럼 데이터 로거에 직접 연결된 것으로 표현하지 말 것.
- 각 층 침하량을 단순 막대그래프로만 표현하고 기준 위치를 생략하지 말 것.
- 기준관이 지반과 함께 침하하는 것처럼 표현하지 말 것.

```

## §7 IMG-036 — 변형률계 설치 개념도 (MAJOR_FIX)

```text
변형률계 설치 개념도를 수정한다.

현재 그림은 변형률계를 설치하면 구조 응력을 직접 평가할 수 있는 것처럼 단순화되어 있다. 변형률계는 설치 방향, 위치, 온도보정, 재료상수, 단면 위치가 함께 정의되어야 한다.

수정사항:
1. 철근 매립형, 콘크리트 표면 부착형, 강재 표면 부착형을 구분한다.
2. 변형률계의 측정축 방향을 명확히 표시한다.
3. 부재 축방향 변형률, 휨 변형률, 전단 변형률을 구분한다.
4. 온도계 또는 더미 게이지를 함께 표시하여 온도보정 필요성을 표현한다.
5. 응력 환산은 재료 탄성계수와 보정 조건이 필요하다고 표시한다.
6. 단일 변형률계 하나로 부재 전체 응력을 대표하지 않도록 한다.
7. 보호커버, 케이블 보호, 방수·방진 처리를 표시한다.

금지:
- 변형률계 수치가 곧 구조물 안전판정값인 것처럼 표현하지 말 것.
- 콘크리트 응력과 강재 응력을 같은 방식으로 단순 환산하지 말 것.
- 게이지 방향을 표시하지 않은 변형률계 그림을 사용하지 말 것.

1920x1080, Korean technical drawing.
```

## §8 IMG-079 — 숏크리트 응력·변형 계측 개념도 (REGENERATE)

```text
숏크리트 응력·변형 계측 개념도를 수정한다.

현재 그림은 숏크리트 계측기가 지반압 전체를 대표하는 장치처럼 보인다. 숏크리트 계측은 국부 응력 또는 변형을 측정하는 것이며, 지보재 전체 안정성을 단일 센서로 판단할 수 없다.

수정사항:
1. 숏크리트 응력계는 숏크리트 내부 매립 위치에 표시한다.
2. 변형률계는 숏크리트 표면 부착 또는 매립 위치를 구분한다.
3. 측정 방향을 방사방향, 접선방향, 축방향 등으로 구분한다.
4. 록볼트, 강지보, 숏크리트 계측은 서로 다른 지보재 계측으로 분리한다.
5. 지반하중 화살표는 개념하중으로만 표시하고, 숏크리트 응력계가 전체 지반압을 직접 측정하는 것처럼 표현하지 않는다.
6. 계측값은 국부 응력·변형 변화와 시간 경과 추세로 해석한다고 표시한다.

금지:
- 숏크리트 응력계 하나로 터널 전체 안정성을 평가하는 그림을 만들지 말 것.
- 숏크리트 응력계와 토압계를 같은 장비처럼 표현하지 말 것.
- 지반압 화살표와 센서 측정값을 1:1로 연결하지 말 것.

1920x1080, Korean tunnel shotcrete instrumentation drawing.
```

## §9 IMG-081 — 기둥 축소량 계측 개념도 (REGENERATE)

```text
기둥 축소량 계측 개념도를 전면 수정한다.

기둥 축소량은 단순 수직 변위계 또는 로드 하나로 모든 층의 축소량을 직접 측정하는 문제가 아니다. 시공단계, 층별 타설 시점, 콘크리트 재령, 탄성축소, 크리프, 건조수축, 하중 증가를 함께 고려해야 한다.

수정사항:
1. 기둥 축소량은 층별 누적 수직변위로 표현한다.
2. 기준층 또는 기준점은 구조물 전체와 함께 움직일 수 있으므로 절대 기준처럼 표현하지 않는다.
3. 각 층별 측점 또는 매립형 변형률계 위치를 표시한다.
4. 콘크리트 탄성축소, 크리프, 건조수축, 시공하중 증가를 구분한다.
5. 계측방법은 변형률계, 매립 게이지, 층별 표고 측량, 레벨 측정, 기준점 비교 등으로 구분한다.
6. RF층을 단순 기준으로 두고 모든 층 축소량을 산정하는 표현은 피한다.
7. 그래프는 층별 시간 경과 축소량 예시로 하되, "예시"라고 표시한다.

금지:
- 기둥 내부에 설치한 수직 로드 하나가 모든 층 축소량을 직접 측정하는 것처럼 그리지 말 것.
- RF층을 절대 기준점으로 표현하지 말 것.
- 크리프·건조수축·탄성축소를 모두 같은 변위로 단순화하지 말 것.

1920x1080, Korean building structure instrumentation drawing.
```

## §10 IMG-059 — 관리기준 설정 개념도 (REGENERATE)

```text
관리기준 설정 개념도를 전면 수정한다.

현재 그림은 모든 센서의 관리기준을 하나의 표와 동일한 경보 로직으로 처리하는 것처럼 보인다. 계측 항목별로 기준의 방향, 단위, 누적값, 속도, 관리단계, 시공단계 영향이 다르므로 일반화하면 안 된다.

수정사항:
1. 관리기준은 센서별이 아니라 계측 항목별로 설정한다고 표현한다.
2. 변위, 침하, 수위, 간극수압, 하중, 응력, 진동은 각각 기준 방식이 다르다고 구분한다.
3. 누적값 기준과 변화속도 기준을 분리한다.
4. 양방향 기준이 필요한 항목과 단방향 기준이 필요한 항목을 구분한다.
5. 시공단계별 기준과 유지관리 기준을 분리한다.
6. 초기치 설정, 이상값 확인, 계측기 점검, 현장 확인 절차를 기준 초과 대응 흐름에 포함한다.
7. 관리기준 수치는 "예시" 또는 "현장별 기준 적용"으로 표시한다.
8. 기준 초과 시 자동 삭제·보정이 아니라 보고·검토·현장 확인으로 연결한다.

금지:
- 모든 센서에 동일한 정상/주의/경고 로직을 적용하지 말 것.
- 출처 없는 관리기준 수치를 넣지 말 것.
- 기준 초과를 곧바로 구조물 위험으로 단정하지 말 것.
```
