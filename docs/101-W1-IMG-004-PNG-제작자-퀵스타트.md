## 4. 등록·서명 (PowerShell)

```powershell
cd x:\website\homepage

npm run register:figure -- --id IMG-004 `
  --input assets/images/technology/source/IMG-004_어스앵커-하중계-설치-개념도_앵커두부정착구.png `
  --method ai-reviewed `
  --reviewer "검수자이름" `
  --visual-grade PASS `
  --notes "v3 ai-reviewed — redline v2 PASS"

npm run build:images
npm run sign:phase-a -- --id IMG-004
npm run verify:local
```
