## 2. 지적 사항 ↔ 기술 판정 ↔ 현재 대응

### 2.1 전단면 360° 폐합 (P1~P8·Extension Tube·바닥 연결)

| 항목 | 내용 |
|------|------|
| **지적** | P1~P8이 360° 원형으로 바닥까지 Extension Tube·브래킷 연결 — 도로·철도·장비 통행 불가 |
| **공학 판정** | **치명 오류.** KDS 내공변위는 **벽면(아치·측벽) 변형** 항목이며, **인버트·노반**은 통행·궤도 구간으로 Kit 체인을 둘 수 없음 |
| **현재 대응** | P1~**P5**만 **상부 아치**에 배치. 하부 **「노반 (도로·철도)」「내공변위 미계측 구간」** 라벨. P6~P8 **없음** |
| **코드** | `scripts/lib/tunnel_convergence_draw.py` — `ARCH_ANGLES` P1~P5, `FLOOR_Y` 노반만 |

### 2.2 하부 P5~P6 변위 프로파일 (가상 데이터)

| 항목 | 내용 |
|------|------|
| **지적** | 프로파일 그래프에 P5·P6(바닥) mm 변위가 표시 — 존재하지 않는 센서 데이터 |
| **공학 판정** | **치명 오류.** 하부 미계측 구간에 프로파일·수치를 넣으면 해석·영업 신뢰 붕괴 |
| **현재 대응** | 우측 패널 **「전단면 변위 프로파일 (상부 아치 P1~P5)」** — **반원(아치) 구간만**. invert·P6~P8 **미표시** |
| **잔여 주의** | 차트에서 P3→…→P5→P3 **아치 상 연결선**은 「폐합」처럼 보일 수 있음 → **노반을 관통하지 않는 아치 체인**임을 캡션·본문에서 명시 (§2.3) |

### 2.3 Closed Loop 연산 vs 개방 아치 체인

| 항목 | 내용 |
|------|------|
| **지적** | 360° 폐합 가정 시에만 성립하는 역계산 — 하부 개방 현장과 불일치 |
| **공학 판정** | **타당.** 현장은 **개방 경계** 아치 체인(첫·끝 측점 고정 또는 상대 기준)으로 ΔX·ΔY 누적 |
| **현재 대응** | 콘텐츠 원리: Kit **연속 배치** → 측점별 좌표 **누적** (폐합·360° 언급 **없음**) |
| **보강 (2026-06-25)** | 원리 문단에 **「하부 노반과 폐합(Closed Loop) 가정 금지·개방 경계」** 명시 |

### 2.4 Extension Tube 돌출·건축한계·유지관리

| 항목 | 내용 |
|------|------|
| **지적** | 거친 숏크리트면에서 튜브가 내공 돌출 → 시공 장비·운영 차량 Clearance 침범 |
| **공학 판정** | **타당.** Figure는 **라이닝 밀착·지지 브래킷** 전제; 과장된 돌출·중앙 부유 Kit 금지 |
| **현재 대응** | 튜브는 **아치 라이닝 따라** 체인; 중앙 허브·부유 센서 **없음** · MUX→로거 **측면 배선** |
| **보강 (계획)** | Figure에 **「라이닝 밀착·건축한계 준수」** 캡션 한 줄 (Phase 2) |
