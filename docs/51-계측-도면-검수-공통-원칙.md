## 2. 앵커·버팀보·띠장 배치 절대 규칙 (P0-6 · SOE-ABS)

> **⛔ P0 치명** — [INSTRUMENTATION §3.1·§3.2·§3.7](./INSTRUMENTATION_DRAWING_RULES.md) · [14 §2](./14-흙막이-굴착-계측-개념도-AI-생성-가이드라인.md)

### 2.1 핵심 3종 (띠장 · 버팀보 · 앵커)

| 구성요소 | 올바른 위치 | 절대 금지 |
|----------|-------------|-----------|
| **띠장** | **굴착측** 벽체면 (H-빔·띠장이 벽체 굴착면에 부착) | **배면 지반(토사) 속** · 벽체 배면 토사 내부 |
| **버팀보** | **굴착측** 내부 공간 (굴착 공동을 가로지름) | 지반 속 · **배면 지반** 속 · 건물 기초 아래 |
| **어스앵커 두부** | **굴착측** 벽체면 (반력판·헤드만 노출; **앵커 사선 축과 동축**) | 정착장 **내부** · 지반 속 · 배면에 두부만 떠 있음 |
| **어스앵커 하중계** | **굴착측 두부** 반력판–하중계–앵커헤드 사이, **앵커 설치각과 동축** | 자유장·정착장·그라우트 내부 |
| **강연선** | **배면 지반** 방향으로 천공·정착 | 버팀보처럼 **수평 지보재** · 굴착측으로만 향함 |
| **앵커 정착장** | 배면 지반 **내** 설계 심도 | **굴착측** 노출부 |

### 2.2 계측기·부속 (동일 3분할 준수)

| 구성요소 | 올바른 위치 | 절대 금지 |
|----------|-------------|-----------|
| **버팀보 하중계** | 띠장–버팀보 접합부 | 버팀보 중앙, 배면 지반 속 |
| **어스앵커 하중계** | **굴착측 두부** 반력판–하중계–앵커헤드 사이, **앵커 축과 동축** | 자유장·정착장·그라우트 내부 |
| **토압계** | 벽체 **배면** 접면 | 굴착측 띠장 위 |
| **지중경사계** | 벽체 배면 또는 영향권 내 천공 | 벽체 전면 굴착공간 중앙 |

### 2.3 Q-SOE 검수 (3항목)

1. **띠장**이 벽체 **굴착측 면**에만 있는가? (배면 토사 **X**)
2. **버팀보**가 **굴착 공동** 안에만 있는가? (지반 **X**)
3. **앵커** 강연선이 **배면(←)** 으로 정착하고, **하중계**가 **두부(반력판–헤드)** 에만 있는가?

**하나라도 「아니오」→ REGENERATE (P0-6).**

## 2.4 IMG-001 — 가시설 계측 전체 개념도 (전용 · P0)

> **역할:** `fields/retaining-excavation` **category hero** · [prompts/IMG-001](../ImageWorks/NMTI_Engineering_Image_Prompt_Package_v1/prompts/IMG-001_가시설_계측_전체_개념도.md)  
> **⛔ 본 절은 「가시설 계측 전체 개념도」 생성·검수 시 필수.** 위반 시 **생성하지 않음 · REJECT**

### 2.4.0 3영역 선행 (생성 게이트)

이 그림은 **「가시설 계측 전체 개념도」**이다. 흙막이 단면은 **먼저** 다음 3영역으로 나눈다.

```text
[배면 지반] | [흙막이 벽체] | [굴착측]
```

**3영역을 구분하지 못하면 이미지를 생성하지 않는다.** (P0-5 · R-01)

※ 인접 건물이 있으면 좌→우 전체: `인접 건물 | 배면 지반 | 흙막이 벽체 | 굴착측` (FIG-03)

### 2.4.1 절대 배치 규칙 (15항)

| # | 규칙 |
|---|------|
| **G-01** | **띠장**과 **버팀보**는 반드시 **굴착측 벽체면**에만 배치 |
| **G-02** | 띠장·버팀보를 **배면 지반 속**에 넣지 않음 |
| **G-03** | **버팀보 하중계** = **띠장–버팀보 접합부** 또는 **축력 전달부** (보 **정중앙 금지**) |
| **G-04** | **어스앵커 하중계** = 반드시 **굴착측 앵커 두부** |
| **G-05** | 어스앵커 하중계를 **자유장·정착장·그라우트 내부·배면 지반 속**에 넣지 않음 |
| **G-06** | 어스앵커 = 벽체 **관통** → **배면 지반** 방향 — **자유장**과 **정착장** **구분** 표시 |
| **G-07** | 앵커 두부 조립 순서 = **`반력판 → 하중계 → 앵커헤드 → 강연선`** |
| **G-08** | **토압계** = 벽체 **배면·지반 접면** — 감지면 방향 **배면 지반 → 벽체** |
| **G-09** | **지하수위계** = **GL에서 천공** — 관측공·**well cap 지표면 가시** · **스크린**·**G.W.L** (BORE-GL-01) |
| **G-10** | **간극수압계** = 특정 심도 **센서** — **짧은 필터** 구간 + 상·하부 **차수(seal)** 구간 |
| **G-11** | 지하수위계 ≠ 간극수압계 — **동일 긴 관측공** 형태로 그리지 않음 (EXC-03) |
| **G-12** | 침하 **핀·측점**이면 **`지표침하계` 라벨 금지** → **`지표침하 측점(침하핀)`** |
| **G-13** | **지표침하 측점(침하핀)** 은 **데이터로거에 직접 연결하지 않음** |
| **G-14** | **자동 센서형 지표침하계**만 **케이블 → 계측함** 연결 표현 |
| **G-15** | **데이터로거 ≠ 센서** — **`현장 계측함 / 데이터로거`** 로 **작게** 표시 · **센서 목록·범례와 분리** (P0-3 · R-10) |
