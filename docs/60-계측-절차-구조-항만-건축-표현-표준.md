# 계측 절차·구조·항만·건축 표현 표준 (정본)

**등급:** **P1** — ZIP 208장 검수 **3차 원칙** (문서 정본 · **Figure 재작성 전 참조용**)  
**역할:** [51 §3 DP-21~30](./51-계측-도면-검수-공통-원칙.md) · [§4 DIS-21~30](./51-계측-도면-검수-공통-원칙.md) 상세  
**범위:** 수동·원격·스마트(070~073) · 진동·지진 · 경사/변위 · 처짐 · 장력 · 항만(098) · 건축(100·081~082·099)

## 0. 강제 지시문 (프롬프트 · Figure 재작성 시 맨 앞)

```text
수동계측은 측정 기준과 기록 절차를 그려라.
원격계측은 통신보다 로컬 저장과 재전송을 먼저 그려라.
스마트계측은 색상 경보가 아니라 검증·확인·조치이력을 그려라.
지진·진동은 3축, 시간파형, 최대값, 주파수를 분리해라.
경사와 변위를 같은 값처럼 그리지 마라.
구조물 지진계는 층별 응답 차이를 표현해라.
처짐계는 기준선, 지점, 하중조건 없이 그리지 마라.
장력계는 고정단, 정착단, 축방향 힘 전달 경로를 그려라.
항만 계측에서 조위, 지하수위, 간극수압을 같은 선으로 그리지 마라.
건축 구조물 계측은 층별·부재별·방향별로 분해해라.
```

## 1. DP-21 — 수동 = 기준·기록 절차 (MAN-01)

**금지:** 작업자·장비만 강조하는 홍보 장면 · 기준점 없는 수동 계측 · 수동값→즉시 공식값

**필수:** 측정 기준점 · 초기치 · 측정 위치·방향 · 측정 주기 · **기록표** · 재측정·검측 절차

**대상:** IMG-070 · 수동 리드아웃·수동 수위·수동 경사 Figure

---

## 2. DP-22 — 원격 = 로컬 저장 우선 (REM-02)

**금지:** `센서 → LTE → 서버` 단축 · 통신선만 예쁜 연결

**필수 순서:**

```text
센서 → 로거 로컬 저장 → 통신 → 서버 수신확인
```

**병기:** 통신두절 상태 · 재전송 · 시간 동기화 · 누락 데이터 확인

**대상:** IMG-072 · 048 · 원격 자동계측 흐름도

---

## 3. DP-23 — 스마트 = 운영 절차 (SMT-02)

**금지:** 정상·주의·경고 **색상만** · 기준 초과→자동 위험 · 작업중지 단독 판단 · 조치 이력 없음

**필수:** 데이터 품질검증 · 관리기준 판정 · 관련 센서 비교 · 담당자 확인 · **조치 이력** · 경보 해제 조건

**대상:** IMG-073 · 102 · 스마트·경보 Figure

---

## 4. DP-24 — 진동·지진 = 3축·파형·주파수 (VIB-02)

**금지:** 흔들리는 선 하나 · PGA·PPV 혼동 · 가속도·속도 단위 혼용 · 3축을 단방향처럼

**필수:** 3축 X/Y/Z · 가속도 또는 속도 **단위** · 시간파형 · 최대값 · 우세주파수 · 이벤트 시각 · 트리거

**대상:** IMG-076 · 095 · 097 · 지진·가속도계 Figure

---

## 5. DP-25 — 경사 ≠ 병진 (KIN-01)

| 구분 | 물리량 |
|------|--------|
| **경사계** | 회전각 θ · 기울기 |
| **변위계** | 상대거리 변화 |
| **프리즘/광파기** | 좌표 변위 |
| **GNSS** | 절대·상대 좌표 변위 |

**금지:** **경사 θ** 와 **수평변위 Δx** 를 같은 값처럼 표현

**대상:** IMG-005 · 043 · 구조물경사·변위 Figure

---

## 6. DP-26 — 구조물 지진 = 층별 응답 (SSE-01)

**금지:** 센서 1개로 전체 지진응답 대표 · 층·방향 없는 지진계 배치

**필수:** 기초부·중간층·최상층 센서 · 층간응답 · 증폭비 · 시간동기화 · X/Y/Z

**대상:** 건물·교량 동적 응답 Figure

---

## 7. DP-27 — 처짐 = 기준선·지점·하중 (DEFL-01)

**금지:** 처짐계만 매달기 · 기준선 없는 δ · 하중 조건 없는 그래프 · **처짐=침하** 동일 표기

**필수:** 지점 위치 · 경간 · 하중 위치 · **기준선** · 처짐 방향 · 중앙부 처짐 · 단부 회전 가능성

**대상:** IMG-099 · 교량·보·슬래브 처짐 — [INSTRUMENTATION §3.27](./INSTRUMENTATION_DRAWING_RULES.md)

---

## 8. DP-28 — 장력 = 축방향 힘 경로 (TEN-01)

**금지:** 케이블 중간 임의 부착 · 고정단·정착단 없음 · LC가 전체 장력 무조건 대표

**필수:** 고정단 · 정착단 · **장력 방향(축)** · 센서 위치 · 편심 · 온도 영향 · 초기 장력

**대상:** 케이블·교량 장력계 Figure

---

## 9. DP-29 — 항만 = 조위·지하수·간극수압 분리 (HAR-05)

| 항목 | 의미 |
|------|------|
| **조위** | 해측 수위 |
| **지하수위** | 배면 관측공 수위 |
| **간극수압** | 특정 심도 u |
| **누수량** | 배수로·집수정·위어 유량 |

**금지:** 해수면=G.W.L=간극수압 **동일선** · 케이슨 내부 누수 센서 · 조위→배면 수압 **인과 단정**

**대상:** IMG-098 · 064 · 084 — [INSTRUMENTATION §3.26](./INSTRUMENTATION_DRAWING_RULES.md)

---

## 10. DP-30 — 건축 = 층·부재·방향 분해 (BLD-DEC-01)

**금지:** 건물 전체 경사를 센서 1개로 · 기둥축소=단일 변위계 · 균열계=경사계 동일 처리

**필수:** 층별 측점 · 기둥·보·슬래브 구분 · 경사·균열·변형률·침하 **분리** · 계측 방향 · 시공단계 · 온도·크리프·건조수축

**대상:** IMG-100 · 081 · 082 · 037 · 099
