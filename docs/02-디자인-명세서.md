## 카드 아이콘 라인 색

카드 제목(`<h3>`) 색은 통일하고, **`.card-icon` SVG stroke만** 카드 순서에 따라 다르게 적용한다.  
배경색·박스는 넣지 않는다 (아이콘 라인만 색상 변경).

### 구현 위치

| 항목 | 파일 | 위치 |
|------|------|------|
| 색상 토큰 정의 | `css/style.css` | `:root` — `--card-icon-1` ~ `--card-icon-6` |
| 카드별 적용 | `css/style.css` | `.card-icon { color: var(--card-accent) }` + `nth-child` 규칙 |
| SVG | `index.html` | `<symbol>` — `stroke="currentColor"` (CSS `color` 상속) |

### 색상 팔레트 (6색 순환)

| 순서 | CSS 변수 | HEX | 색상 |
|------|----------|-----|------|
| 1 | `--card-icon-1` | `#2563eb` | Blue |
| 2 | `--card-icon-2` | `#0891b2` | Cyan |
| 3 | `--card-icon-3` | `#0d9488` | Teal |
| 4 | `--card-icon-4` | `#7c3aed` | Violet |
| 5 | `--card-icon-5` | `#d97706` | Amber |
| 6 | `--card-icon-6` | `#475569` | Slate |

### 섹션별 적용 규칙

동일 팔레트를 **그리드 내 카드 순서(nth-child)** 로 매핑한다.

| 섹션 | 그리드 클래스 | 순환 규칙 | 카드 수 |
|------|---------------|-----------|---------|
| 토목 계측 6분야 | `.field-grid` | `6n+1` … `6n+6` | 6 |
| 현장별 문제·해결 | `.ps-grid` | `6n+1` … `6n+6` | 6 |
| 프로그램 기능 | `.program-features` | `3n+1` … `3n+3` | 3 |
| 사업분야 | `.service-grid` | `6n+1` … `6n+6` | 6 |
| 회사 소개 팩트 | `.fact-grid` | `4n+1` … `4n+4` | 4 |
| 등록·인증·특허 | `.cert-grid` | `6n+1` … `6n+6` | 6 |
| 연락처 | `.contact-grid` | `1`, `2` | 2 |

### 예: 토목 계측 6분야 (`#fields`)

| 순서 | 제목 | 아이콘 색 |
|------|------|-----------|
| 1 | 가시설·흙막이 | `#2563eb` |
| 2 | 교량·구조물 | `#0891b2` |
| 3 | 터널·지하 | `#0d9488` |
| 4 | 비탈면·사면 | `#7c3aed` |
| 5 | 지하수·연약지반 | `#d97706` |
| 6 | 자동화·유지 | `#475569` |

### 색상 변경 방법

1. `:root`의 `--card-icon-1` ~ `--card-icon-6` HEX 값 수정 (전역)
2. 특정 카드만 변경 시 해당 `nth-child` 블록에 `--card-accent: #xxxxxx` 직접 지정
