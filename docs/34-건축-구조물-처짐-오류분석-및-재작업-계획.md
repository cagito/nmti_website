## 1. 문제 요약

`fields/building/deflection`(페이지 제목 **「처짐」**) hero에 **IMG-050**(침하 그래프 예시)이 바인딩되어 있습니다. Figure 좌측은 **연약지반 성토·침하계** 설치 개념도, 우측은 **시간-침하 곡선** 그래프로, 건축 구조물 **처짐(Deflection)** 과 **지반 침하(Settlement)** 가 완전히 혼동된 상태입니다.

| 구분 | 현황 |
|------|------|
| hero 바인딩 | ❌ **DEF-01** — `IMG-050` = 토목 침하 Figure, ≠ 건축 구조물 처짐 |
| 단면 배경 | ❌ **DEF-02** — 성토체(흙더미)·연약지반 — RC 보·슬래브 골조 **부재** |
| 센서 표현 | ❌ **DEF-03** — 침하계·침하판(지반) — LVDT·레이저 처짐계 **부재** |
| 그래프 | ❌ **DEF-04** — Y축 침하량·120일 침하곡선·임의 -50mm — 처짐량·L/250·L/360 **부재** |
| 본문·바인딩 | ⚠ **M1** — `content-data`·`dictionary`에 침하계·settlement-gauge 잔존 |
| IMG-050 자체 | ✅ 터널·연약지반·성토 **sectionImages** 용도로는 **유지** (hero에서만 분리) |

**판정:** `fields/building/deflection` hero **REGENERATE** — `IMG-099` 신규 제작·교체 전 **문맥 오류 노출**.

## 2. 발견된 오류 (치명 → 중요)

### 2.1 DEF-01 — 분야 hero에 침하(Settlement) Figure 재사용 (치명)

**관찰:**

- `dictionary.js`: `fields/building/deflection` → `imageId: 'IMG-050'`
- IMG-050 파일명·캡션: `침하-그래프-예시_시간침하곡선예측선`
- 프롬프트 필수 라벨: **침하계, 침하량, 성토, 기준점** — 토목 연약지반 성토 전용

**공학적 모순:**

- 건축 **처짐** = 보·슬래브 등 **구조 부재의 휨 변형** (하중·강도·크리프)
- Figure 내용 = **지반·성토체의 수직 하강(침하)** — 계측 원리·센서·그래프 축이 전혀 다름

**조치:**

- hero → **`IMG-099`** (신규) 로 교체
- **IMG-050**은 `fields/tunnel/*`·연약지반 등 **침하 맥락 sectionImages** 전용 유지

**규칙 근거:** HAR-01(항만)·BRI-01(교량)과 동형 — **분야 전용 hero 금지 재사용**

---

### 2.2 DEF-02 — 성토·연약지반 단면 (치명)

**관찰:** 좌측 개념도 = 사다리꼴 **성토체** + 지반면 + 수직 막대·침하판

**올바른 표현 (건축 처짐):**

| 요소 | 표현 |
|------|------|
| 골조 | RC **기둥(Column)** + **보(Beam)** 또는 **슬래브(Slab)** 단면 |
| 측점 | 보 **중앙부(Mid-span)** — 최대 처짐 위치 |
| 시공 맥락 | 거푸집·동바리·타설 단계 (선택 콜아웃) |
| 금지 | 성토체·연약지반·지반면 기준 침하판 |

---

### 2.3 DEF-03 — 침하계·침하판 vs 처짐 계측기 (치명)

**관찰:** 지반에 매립된 **침하계(침하판)** + 기준점(지표) 구조

**올바른 표현:**

| 기호 | 계측기 | 설치 위치 |
|------|--------|-----------|
| ① | **LVDT(변위계)** 또는 레이저 처짐계 | 보 하단 중앙 — 수직 변위 |
| ② | 와이어식 처짐 센서 (선택) | 동일 스팬 |
| ③ | **구조물경사계(Tiltmeter)** (선택) | 보-기둥 접합부 — 기울기 연계 |
| ④ | **자동광파기(ATS)** + 프리즘 (선택 inset) | 절대 3D 변위 — BLD-03 |

**금지:** 침하계·침하판·층별침하계(IMG-032·033)를 건축 처짐 hero에 사용

---

### 2.4 DEF-04 — 침하 그래프 vs 처짐 그래프 (치명)

**관찰 (IMG-050 우측):**

- Y축: **침하량(mm)** — 0 ~ -80 (아래=침하)
- X축: **시간(일)** 0~120 — **1차 압밀·성토 침하** 형태
- 관리기준: 임의 **-50mm** 수평선
- 범례: 예측 **침하**선

**올바른 표현 (건축 처짐):**

| 항목 | TO-BE |
|------|-------|
| Y축 | **처짐량(δ, mm)** — 보 중앙 수직 변위 |
| X축 | **시간(일)** 또는 **시공 단계**(타설·양생·해체·프리스트레스) |
| 곡선 | 급처짐 → 수렴 (크리프·강도 발현) — 설계 예측 처짐선 병기 |
| 관리기준 | **L/250**(활하중 처짐) · **L/360**(균열·소성) 또는 현장 **설계 허용치**(예: 15mm, 20mm) — KDS 건축구조기준 |
| 금지 | 120일 연약지반 침하곡선·「예측 침하」라벨 |

## 5. 목표 Figure — IMG-099 (재작업 후)

### 5.1 레이아웃 (16:9)

```text
[좌 55%] RC 골조 단면 + LVDT·(선택) 경사계  |  [우 45%] 시간-처짐(δ) 그래프 + L/250·L/360 기준선
```

### 5.2 좌측 — 설치 개념도

| 기호 | 요소 | 표현 |
|------|------|------|
| — | 골조 | 기둥 + 단순보 또는 슬래브 — 거푸집/동바리 암시 가능 |
| ① | LVDT(변위계) | 보 **중앙 하단** — 수직 δ 화살표 |
| ② | 구조물경사계 (선택) | 보-기둥 접합 — θ (BLD-02: 변위와 분리 라벨) |
| — | 콜아웃 | 기준점 = 고정 앵커·지지대 (지반 기준점 **아님**) |

### 5.3 우측 — 처짐 그래프 예시

| 요소 | 내용 |
|------|------|
| Y | 처짐량 δ (mm) — 아래 방향 증가 시 **음수** 또는 양수 **일관** (DAM-01 역전 금지) |
| X | 시간(일) 또는 시공 단계 |
| 실선 | 측정값 — 타설 후 급증 → 수렴 |
| 점선 | 설계 예측 처짐 |
| 기준선 | **L/250** (주의) · **L/360** (관리) — 스팬 L 예시 표기 (예: L=12m → 48mm / 33mm) |
| inset (선택) | ATS 프리즘 네트워크 — BLD-03 |

### 5.4 도면 표준 매핑

| 구역 | 필수 | 금지 (현행 IMG-050 hero) |
|------|------|---------------------------|
| 구조 | RC 보·기둥·슬래브 | 성토체·연약지반 |
| 센서 | LVDT·레이저·와이어 처짐 | 침하계·침하판 |
| 그래프 | δ·L/250·L/360 | 침하량·예측 침하·-50mm 임의선 |
| 용어 | 처짐·변위·휨 | 침하·성토·기준점(지표) |
