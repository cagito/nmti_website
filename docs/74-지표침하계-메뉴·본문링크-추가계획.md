**수립:** 2026-06-22  
**우선순위:** **P1** (용어 정합 · SETTLE-01 · 본문 13+건 미연결)  
**선행:** [42-지표침하핀·지표침하계 구분](./42-지표침하핀-지표침하계-구분-및-자동계측-정책.md) · [TERMINOLOGY](./TERMINOLOGY.md) · IMG-032 **PASS**  
**참고 패턴:** [58-처짐계 분리](./58-교량-처짐-케이블장력-신규-추가-계획.md) (메뉴 라벨·KEYWORD·본문 링크 동시 정리)

## 1. 현황·갭 분석

### 1.1 메뉴 vs 실제 용어

| 항목 | 현재 | 문제 |
|------|------|------|
| TREE 라벨 | **침하계** | KDS·IMG-032·준공·계획서는 **지표침하계**(자동 센서) 표기 |
| nodeId | `sensors/settlement-gauge` | ✅ 유지 — [42 §2.2](./42-지표침하핀-지표침하계-구분-및-자동계측-정책.md) 정본 |
| hero Figure | IMG-032 | 라벨 **지표침하계·침하판** — 메뉴명과 불일치 |
| KEYWORD_MAP | `침하계`, `침하판` | **`지표침하계` 없음** → autolink 미동작 |

### 1.2 본문에서 「지표침하계」 사용처 (링크 없음)

| 파일 | 건수 | 맥락 |
|------|------|------|
| `leaves-part1.mjs` | 5 | 흙막이 벽체 — 자동 센서 통합 해석 |
| `leaves-part2.mjs` | 7 | 교각·기초침하 — ATS·측점과 병기 |
| `leaves-extra.mjs` | 1 | IMG-002 개요 |
| `content-data.js` / SEO HTML | 13+ | 빌드 산출 — 소스 수정 후 재생성 |

**반면** 동일 문단에 `침하계`만 쓰인 곳은 KEYWORD로 `settlement-gauge`에 연결됨 → **같은 센서인데 표기만 다르면 링크 유무가 갈림**.

### 1.3 혼동 위험 (반드시 유지할 구분)

| 용어 | 정체 | nodeId / 링크 |
|------|------|----------------|

| **지표침하핀·지표침하 측점** | 수준·ATS **측량 표식** | **링크 금지** (SETTLE-01) · 맥락상 `sensors/automated-total-station`만 선택 |

| **지표·지중침하** (현상·공종) | 터널 분야 항목 | `fields/tunnel/surface-subsidence` |
