## 0. 결론

IMG-008의 기준은 **상부 아치 개방 체인**이 아니라 **터널 전단면 내공변위계**이다. 이전 문서와 프롬프트가 P1~P5 상부 아치 중심으로 정리되면서 전단면 내공변위계의 본질을 훼손했다.

따라서 신규 이미지와 registry가 전단면 기준으로 정합화되기 전까지 기존 이미지는 출판 PASS로 보지 않는다.

## 4. 출판·디자인 기준

상세 디자인 기준은 `docs/126`을 따른다.

| 검수 | 합격 기준 |
|---|---|
| F1 | 그림이 전단면 내공변위계로 읽힘 |
| F2 | 상부 아치 단독 계측도로 보이지 않음 |
| F3 | 대표 측점이 단면 전체를 설명함 |
| F4 | 대표 측선이 전단면 개념을 보여줌 |
| F5 | 천단침하계 또는 상부 체인 계측으로 오인되지 않음 |
| F6 | 서버/통신/UI 등 무관 요소 없음 |
| F7 | 100%·200% 확대 시 라벨과 선이 선명함 |
| F8 | 홈페이지 hero로 사용 가능한 정돈된 디자인 |

하나라도 F1~F5에 불합격하면 `REGENERATE`이다.
