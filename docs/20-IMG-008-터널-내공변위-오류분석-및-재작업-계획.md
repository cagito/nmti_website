## 2. 발견된 오류 (치명 → 중요)

### 2.1 C1 — P6/P7·Extension Tube가 **건축한계 Envelope 침범** ⛔

**코드 근거** (`scripts/lib/tunnel_convergence_draw.py`):

| 상수 | 값 |
|------|-----|
| `FLOOR_Y` | 740 |
| `CLEARANCE_FLOOR_Y` | **678** (`FLOOR_Y - 62`) |
| P6 각도 | **11π/12 (165°)** — 좌 스프링라인 |
| P7 각도 | **π/12 (15°)** — 우 스프링라인 |
| P6/P7 Y (inset 32) | **≈668** — Envelope 바닥(678) **아래** |

**관찰 (Figure):**

- P6·P7·체인·Ball Joint·MUX가 **하부 말굽(스프링라인)** 에 밀집.
- 주황 **건축한계 점선** 내부·경계와 **겹침** — 「도로·철도 통행 구간」으로 읽힘.

**공학적 모순:**

- 내공변위 Kit는 **라이닝 밀착**이되 **건축한계(Gauge) 바깥(여유 공간 쪽)** 또는 **상부 아치**에만 — Envelope **내부**에 측점을 두면 **차량·궤도 간섭** 도면.

**올바른 표현:**

- 측점 = **상부 아치** (천단~상반부) — 스프링라인 **위** (예: 각도 **≥ 50°~130°** 구간만).
- v7 권장: **P1~P5** (천단·좌우 상부) 또는 P6/P7을 **120°·60°** 등으로 **상향** — Envelope **외측(라이닝)** 에만 표기.
- 치수: 「건축한계 Envelope **외측 라이닝**」 콜아웃.

**규칙 근거:** INSTRUMENTATION §3.11 · §3.11.1 C1 · 04 가이드 §치명 #2

### 2.2 C2 — 건축한계 캡션 **역메시지** ⛔

**관찰 (Figure, `_draw_clearance_envelope`):**

```text
건축한계 (Clearance Envelope)
통행·궤도 구간 — 센서·튜브 밀착 설치   ← 오류
```

**공학적 모순:**

- Envelope **안** = **통행·궤도** — 여기에 「센서·튜브 밀착」은 **설치 불가** 메시지.

**올바른 표현:**

```text
건축한계 (Clearance Envelope) — 통행·궤도 (미계측)
센서 Kit = 라이닝 · Envelope 외측 (건축한계 침범 금지)
```

**규칙 근거:** §3.11.1 C2 · 04 가이드 슬로건

### 2.3 C3 — 노반 **아래** Extension Tube 캡션 ⛔

**관찰 (`draw_tunnel_section` L216~222):**

- 「노반 (도로·철도)」「내공변위 미계측 · Open」 **아래**에  
  **「Extension Tube · Ball Joint Bracket · 건축한계 밀착」** — 노반·인버트 **영역**.

**공학적 모순:**

- Extension Tube·Bracket은 **아치 체인**에만 — **미계측 구간** 라벨과 **동일 높이대**에 하드웨어 문구 = **하부 계측 오인**.

**올바른 표현:**

- 해당 캡션 **삭제** 또는 **아치 체인 중앙**(P1~P2 사이)으로 이동.
- 노반 인근 = **「내공변위 미계측 · Kit 없음」** 만.

**규칙 근거:** §3.11.1 C3
