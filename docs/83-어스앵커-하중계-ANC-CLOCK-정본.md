## 0. 모든 도면 공통 적용 원칙

이 문서는 IMG-004 전용이 아니라 **모든 도면의 어스앵커 표현 공통 정본**이다.

다음 Figure에 어스앵커가 배경으로 작게 나오더라도 동일하게 적용한다.

- 흙막이 계측 전체 개념도
- 흙막이 대표 단면도
- 주변건물 계측도
- 주변지반 계측도
- 하중계 설명도
- 앵커가 포함된 모든 가시설·굴착 단면도

**적용 조건:**

```text
그림 안에 앵커 강연선, 앵커 두부, 반력판, 앵커 하중계, 앵커헤드 중 하나라도 있으면 ANC-CLOCK 적용.
```

## 1. 공간 구분

흙막이 단면의 기본 구분은 다음과 같다.

```text
[배면 지반] | [흙막이 벽체] | [굴착측]
```

| 요소 | 위치 |
|------|------|
| **강연선·자유장·정착장·그라우트체** | 배면 지반 내부 |
| **흙막이 벽체** | 배면 지반과 굴착측 경계 |
| **띠장·반력판·앵커 하중계·앵커헤드** | 굴착측 노출 두부 |

금지:

- 앵커 두부를 배면 지반 속에 배치
- 하중계를 자유장·정착장·그라우트 내부에 배치
- 앵커를 버팀보처럼 수평 지보재로 표현
- 강연선을 굴착측으로만 향하게 표현

---

## 2. ANC-CLOCK 시계방향 기준

좌→우 = `배면 지반 | 벽체 | 굴착측`일 때, 시계는 **벽체·앵커 두부를 중심**으로 읽는다.

```text
           12시
      9시        3시 (굴착측)
           6시

강연선·PC strand 정착 방향 = 7시 (좌하 · ← 배면 지반)
하중계 장축                 = 1시~7시 사선
앵커헤드·반력판 중심축       = 1시~7시 사선
```

| 요소 | 올바른 방향 | 의미 |
|------|-------------|------|
| **강연선** | 7시 방향 | 배면 지반 정착 방향 |
| **자유장** | 강연선과 동일 사선 | 비부착 인장 전달 구간 |
| **정착장** | 강연선 사선 연장부 | 배면 지반 내부 정착 구간 |
| **하중계 장축** | 1시~7시 | 앵커 축과 동축 |
| **앵커헤드** | 1시~7시 축상 | 하중계 바깥쪽 두부 |
| **반력판** | 앵커 축에 직각 | 하중계 반력 전달 |

---

## 3. ANC-AXIS 동일축 원칙

다음 요소는 모두 **하나의 곧은 사선축** 위에 있어야 한다.

```text
정착장 → 자유장 → 강연선 → 반력판 중심 → 하중계 중심축 → 앵커헤드
```

**폐기 조건:**

1. 강연선은 사선인데 하중계가 수평
2. 하중계와 앵커헤드가 서로 다른 각도
3. 반력판 중심과 강연선 중심이 어긋남
4. 앵커 두부가 벽체에 수평·수직으로 단순 부착됨
5. 앵커를 버팀보 하중계 구도처럼 그림

---

## 4. T/P 하중 방향 공통 원칙

어스앵커 하중계가 표현되는 모든 도면에서 T와 P는 다음을 따른다.

| 기호 | 물리량 | 방향 |
|------|--------|------|
| **T** | 앵커 인장력 | 강연선 축방향 |
| **P** | 하중계 압축 반력 | 하중계 중심축, 즉 앵커 축방향 |

- T와 P는 반드시 **같은 사선축** 또는 **같은 연장선** 위에 둔다.
- 두 화살표는 방향이 반대일 수 있으나, **각도는 반드시 같다.**
- T와 P의 각도가 다르면 그림은 폐기한다.
- P를 토압처럼 벽체에 수평으로 작용하는 화살표로 그리지 않는다.
- `P=T`만 단독으로 쓰지 않는다.

## 5. 부품 표현 공통 원칙

| 요소 | 표현 |
|------|------|
| **반력판** | 앵커 축에 직각인 기울어진 판 |
| **하중계** | 반력판과 앵커헤드 사이, 사선축과 동축 |
| **앵커헤드** | 하중계 바깥쪽 굴착측, 사선축상 |
| **강연선** | 배면 지반 방향 사선 |
| **자유장·정착장** | 배면 지반 내부에서 구분 |
| **신호 케이블** | 하중계 일부에서 짧게 노출, 강연선과 구분 |

금지:

- 정착너트·육각너트·볼트형 너트 강조
- 하중계를 강연선 중간에 끼움
- 하중계를 정착장·그라우트체 안에 그림
- 케이블을 강연선처럼 굵게 그림
- 로거·서버·모바일 흐름도화

## 6. 모든 도면 프롬프트 붙여넣기 블록

어스앵커가 포함되는 프롬프트 맨 위에 아래를 붙인다.

```text
[ANC-CLOCK / ANC-AXIS — 모든 어스앵커 도면 필수]
어스앵커는 수평 버팀보가 아니다.
강연선은 배면 지반 방향 7시 사선으로 정착된다.
하중계, 앵커헤드, 반력판 중심, 강연선은 모두 하나의 1시~7시 사선축 위에 정렬한다.
하중계는 수평(3시~9시)이 아니다. 버팀보 하중계처럼 그리면 전체 폐기한다.
반력판은 앵커 축에 직각으로 배치한다.
T 인장력과 P 압축 반력은 같은 사선축 또는 같은 연장선 위에 둔다. 두 화살표의 각도가 다르면 폐기한다.
하중계는 지중, 자유장, 정착장, 그라우트 내부에 넣지 않는다.
정착너트·육각너트는 그리지 않는다.
```

## 10. 검수 체크리스트

- [ ] 앵커 강연선이 배면 지반 방향 7시 사선인가?
- [ ] 하중계 장축이 1시~7시 사선인가?
- [ ] 반력판, 하중계, 앵커헤드, 강연선이 동일 사선축인가?
- [ ] 반력판이 앵커 축에 직각인가?
- [ ] 하중계가 굴착측 노출 두부에 있는가?
- [ ] 하중계가 자유장·정착장·그라우트 내부에 없는가?
- [ ] T와 P가 같은 사선축 또는 같은 연장선 위에 있는가?
- [ ] T와 P의 각도가 같은가?
- [ ] 버팀보 하중계 구도와 혼동되지 않는가?
- [ ] 정착너트·육각너트를 강조하지 않았는가?

하나라도 아니면 **REGENERATE**.
