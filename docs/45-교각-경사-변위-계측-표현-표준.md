# 교각 경사·변위 계측 표현 표준 (BRI-PIER)

**수립:** 2026-06-26  
**우선순위:** **P0 (교량 교각 Figure)**  
**대상:** `fields/bridge/pier` · **IMG-012** · INSTRUMENTATION **§3.23.1**  
**정본:** [INSTRUMENTATION §3.23.1](./INSTRUMENTATION_DRAWING_RULES.md) · [46-IMG-012 수정계획](./46-IMG-012-교각-변위-경사-수정계획.md)

> **한 줄:** 교각 Figure는 **경사·상대변위·절대좌표변위·기초침하**를 **구분**하여 표현한다. 변위계는 **기준·상대측정 대상**이 있는 위치에만 둔다.

---

## 1. 목적

교각 계측 개념도에서 다음 네 가지를 **혼동하지 않는다**.

| 유형 | 계측 수단 | 측정량 | Figure 역할 |
|------|-----------|--------|-------------|
| **① 경사·회전** | **구조물경사계** (상·하부) | 기울기 θ, 상·하부 θ **차** → 교각 회전·기초 변형 추정 | 교각 상·하부 표면 부착 + θ |
| **② 상대변위** | **변위계** (LVDT·레이저 등) | 두 점 사이 **상대** 변위 | **받침부** · **신축이음** · 교각–**독립 기준 구조** 사이 |
| **③ 절대 좌표변위** | **자동광파기 + 프리즘** · **GNSS** | 기준국 대비 **3D·절대** 변위 | 교각 **상부 프리즘** + 영향권 **밖** ATS/GNSS |
| **④ 기초 침하** | **지표침하계** · **층별침하계** ([SETTLE-01](./42-지표침하핀-지표침하계-구분-및-자동계측-정책.md)) | 수직 침하·**부등침하** | 기초·말뚹캡 **침하계(센서)** — 침하핀 금지 |

**혼동 시 오류:** 교각 측면에 **단독 수평 변위계** → 측정 기준 불명 · 상대/절대 혼동 · 계측관리계획서와 불일치.

## 2. 측정 체계 (개념)

```text
[영향권 밖 안정지반]                    [교각]
  기준점(BM) ──────────────── 시준선 ──→ 프리즘(교각 상부)  ← ③ 절대 좌표변위
  자동광파기 / GNSS 기준국

  교각 상부 ── 구조물경사계(θ₁) ─┐
                                  ├─ Δθ → 회전·기초 변형 판단  ← ① 경사
  교각 하부 ── 구조물경사계(θ₂) ─┘

  받침부 / 신축이음 ── 변위계 ── 상대 측정  ← ② 상대변위

  기초·말뚹캡 ── 지표침하계(복수) ── 부등침하  ← ④ 기초 침하
```

## 4. Figure 표현 (IMG-012 v2)

### 4.1 좌측 주 단면 (필수)

| # | 요소 | 표현 |
|---|------|------|
| A | 교각 + 상부 일부(거더·교면 실루엣) | 받침·신축이음 위치 표시 |
| B | ① 상·하부 **구조물경사계** | θ₁·θ₂ · **Δθ** 콜아웃 |
| C | ② **상대변위계** | 받침부 또는 신축이음 **양단** 연결 |
| D | ③ **프리즘**(교각 상) + **ATS**(영향권 밖) | 시준선 점선 |
| E | ④ 기초 **지표침하계** 2~3점 | 부등침하 화살표(선택) |
| F | **기준점** | 영향권 밖 · ATS 인접 |

### 4.2 금지 (현행 IMG-012 v1)

- ③ 변위계(수평변위)가 교각 측면에 **단독** 부착
- ④ 기준점이 **기초 바로 옆**
- 우측 **센서→로거→관리기준→DB** 흐름도
- 그래프 Y축 **0.5°·10mm** 등 **보편 기준값**
- 「경사계」단독 — **구조물경사계** 전칭

### 4.3 우측 패널 (v2)

**계측 항목 요약** (텍스트만, 그래프 선택):

1. 교각 상·하부 경사 (구조물경사계)
2. 교각 회전·기초 변형 (Δθ)
3. 받침·신축 상대변위 (변위계)
4. 절대 좌표변위 (ATS·프리즘 / GNSS)
5. 기초 침하·부등침하 (지표침하계)
6. 관리: **현장별 관리기준 적용**

(선택) 경사·변위 **추세 개념** 그래프 — 축만, 임계 **수치 없음**
