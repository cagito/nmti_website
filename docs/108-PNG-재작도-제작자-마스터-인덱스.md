## 등록 템플릿

```powershell
npm run rework:canonical -- --id IMG-002
# PNG → 출력 경로에 저장 후
npm run rework:done -- --id IMG-002 `
  --input assets/images/technology/source/IMG-002_흙막이-계측-설치-대표-단면도.png `
  --reviewer "검수자"
npm run verify:content
```
