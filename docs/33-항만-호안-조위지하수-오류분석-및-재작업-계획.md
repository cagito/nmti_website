## 1. 문제 요약

`fields/harbor/tide-groundwater` 노드 hero에 **IMG-030**(배면 지반·평지 GL·흙막이 콜아웃의 육상 관측공 Figure)이 바인딩되어 있습니다. 페이지 제목·본문은 **조위·간극수압·해안 수문**을 설명하나, 노출 Figure는 **항만 호안(Seawall/Revetment) 맥락이 전혀 없는** 복사·붙여넣기 수준의 오류입니다.

| 구분 | 현황 |
|------|------|
| hero 바인딩 | ❌ **HAR-01** — `IMG-030` = 센서별 지하수위계, ≠ 항만 조위·지하수 |
| 바다·호안 구조 | ❌ 외해·사석/케이슨·필터매트 **부재** |
| 조위 계측 | ❌ 조위계·H.W.L/L.W.L·소파통 **없음** |
| 침윤선 | ❌ 육상식 **수평 G.W.L** — 조석 잔류 곡선 아님 |
| 관측공 | ❌ **유공관(screen)** 미표현 — 불통 관 |
| IMG-030 자체 | ✅ `sensors/water-level-meter` 용도로는 **PASS** 유지 |

**판정:** hero 바인딩 **REGENERATE** — `IMG-098` 신규 제작·교체 전 **문맥 오류 노출**.

## 2. 발견된 오류 (치명 → 중요)

### 2.1 HAR-01 — 분야 hero에 육상 지하수위계 Figure 재사용 (치명)

**관찰:**

- `fields/harbor/tide-groundwater` → `imageId: 'IMG-030'`
- IMG-030은 INSTRUMENTATION **§3.4** · 흙막이 배면 관측공(벽체 부착 금지) 맥락의 **센서 설치도**
- 제목만 「조위·지하수」이고 단면은 **평지 GL + 매립/모래 + 수평 G.W.L**

**공학적 모순:**

- 항만·호안은 **바다(외해) ↔ 구조물 ↔ 배면 매립지** 경계 문제
- 조위·지하수·간극수압 **연동**이 핵심인데, 육상 굴착 관측공 도면으로는 해석 불가

**조치:**

- hero → **`IMG-098`** (신규) 로 교체
- `IMG-030`은 **`sensors/water-level-meter`** 전용 유지 — **PASS 변경 없음**

**규칙 근거:** BRI-01(교량)과 동형 — **분야 전용 hero 금지 재사용** · 본 문서 §4

---

### 2.2 HAR-02 — 조위(Tide) 계측 실종 (치명)

**관찰:** 본문·purpose에 「조위」 있으나 Figure에 **해수면·조위계·조석 범위 없음**

**올바른 표현:**

| 요소 | 표현 |
|------|------|
| H.W.L | 약최고고조위 (상한 파선/실선) |
| L.W.L | 약최저고조위 (하한) |
| 조위계 | **정수우물(Stilling well)** + 압력/초음파 센서 — 파랑 직접 노출 최소화 |
| 해역 | 좌측 **외해** 영역 파랑·수면 채움 |

---

### 2.3 HAR-03 — 조석 잔류수위·곡선 침윤선 무시 (치명)

**관찰:** 육상식 **일직선 수평 G.W.L** (`─ ─ ─`)

**수리역학:**

- 썰물 후 배면 토사 내부 수두는 바다보다 **늦게·완만하게** 하강 → **곡선 침윤선(phreatic/tidal lag)**
- 바다 쪽으로 **기울어진** 잔류 수두가 정상

**조치:** 배면 매립지에서 **곡선 파선** — 각 지하수위계·간극수압계 팁 고도와 **정합** (DAM-02 원칙 동형)

---

### 2.4 HAR-04 — 관측공 유공관(Screen)·필터팩 누락 (중대)

**관찰:** PVC 보호관이 **불통 실관**처럼 표현 — 변동 구간에 **슬릿/유공관·여과재** 없음

**올바른 표현 (§3.4 + 호안):**

- G.W.L 변동 구간 = **perforated screen** + filter pack + annular seal
- 상부 비변동 구간 = 밀폐 casing + bentonite/grout
- well cap · submerged logger · vented cable (AUTO-01)

**금지:** 간극수압계(§3.5)와 **동형** standpipe 전관 개방 (EXC-03)
