## 2. Figure (IMG-008 v6)

**코드:** `scripts/lib/tunnel_convergence_draw.py`  
**PNG:** `IMG-008_터널-전단면-내공변위-측정시스템_상부아치내공변위.png`

### 2.1 개방형 U자형 단면

- 하부 **노반(도로·철도·인버트)** — 회색 음영, **내공변위 미계측 · Open**
- 계측 체인: **P6(좌 스프링라인) → P3 → P2 → P1 → P4 → P5 → P7(우 스프링라인)**
- P5~P8 바닥 센서·폐합선 **없음**

### 2.2 건축한계 (Clearance Envelope)

- 내공 **점선** 윤곽 — 차량·열차 통행 구역
- Extension Tube·Ball Joint Bracket은 **한계선 외곽 벽면** 밀착 설치 시각화

### 2.3 아치형 거동 프로파일

- 360° 레이더·원형 그래프 **금지**
- 우측 패널: **수평 아치 展개** — 하부 **「노반 Open · 미측정」** 회색 구간
- 양 끝 **← 개방 / 개방 →** — Closed Loop 아님

### 2.4 Kit·현장 H/W 라벨

- **Ball Joint Bracket** (구면 조인트)
- **RS-485 디지털 Multi-drop** (우측 ΔX/ΔY 패널)
