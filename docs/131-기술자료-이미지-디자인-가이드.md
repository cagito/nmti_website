## 0. 공통 생성 원칙

모든 이미지 생성 프롬프트는 다음 5개 항목을 반드시 포함한다.

```text
1. 설치 위치: 계측기가 어디에 설치되는가
2. 측정 물리량: 무엇을 측정하는가
3. 힘·변위·수위·수두 방향: 화살표와 기준점이 어디인가
4. 금지 표현: 어떤 오해 표현을 절대 넣지 말아야 하는가
5. Figure 목적: 이 그림이 대표 단면도인지, 설치 상세도인지, 운영 흐름도인지 명확히 구분
```

### 공통 금지

```text
- AI 생성 원본에 로고·워터마크·불필요한 브랜드 표시 금지 (검수 후 후처리 워터마크는 별도)
- 제품 사진형 데이터로거 금지
- 모바일·서버 UI를 기술 단면도에 강제로 삽입 금지
- 범례에 있으나 그림에 없는 항목 금지
- 그림에 없는 센서명을 표·범례에 기재 금지
- 계측기 종류가 다른데 같은 아이콘으로 표현 금지
- 관리기준 수치를 임의로 일반 기준처럼 표시 금지
- 계측기의 설치 위치와 측정 방향을 생략한 아이콘 나열 금지
```

### 0.1 어스앵커 하중계 절대 원칙 (P0 — 최상위)

> **정본:** [114-모든-도면-어스앵커-공통-표현-절대원칙](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) · [83 ANC-CLOCK](./83-어스앵커-하중계-ANC-CLOCK-정본.md)

어스앵커·앵커 하중계·반력판·노출 강연선이 **한 번이라도** 그려지는 모든 Figure에 적용한다. **미관보다 공학적 기하 관계가 우선**한다.

| 순서 | 규칙 |
|------|------|
| **1** | **축선 먼저** — 부품 전에 `정착장→자유장→관통→반력판→하중계→헤드→노출 강연선` **동일 사선 직선축** |
| **2** | **반력판** = 앵커 축선에 직각(화면 수직 **아님**) |
| **3** | **하중계** = 굴착측 노출 두부 · 사선 원통/링 · **수평 금지** · 지중·자유장·정착장 **금지** |
| **4** | **노출 강연선** = 하중계 축선 **연장** · 수평 꺾임 금지 |
| **5** | **정착장** = 배면 지반 **깊은 곳** (GL 바로 아래 얕은 정착 금지) |
| **6** | **IPI·지하수위**와 앵커 축선 **충돌 금지** |
| **7** | **버팀보·지반앵커·지보재** 하중계 혼입 금지 |

**생성 전:** [114 §10](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) 10문항 자기검수 — **하나라도 아니오면 생성 중단**.

**판정:** [114 §12 PASS](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) 미충족 → **`REGENERATE`** · **PASS 금지**.

## 1. IMG-024 — 댐 안전관리 계측 체계도

**정본 상세:** [39](./39-IMG-024-댐-안전관리-계측-체계도-전면-수정-계획.md) §12 · ImageWorks `IMG-024_댐_계측_개념도.md`

### 1.1 문제점 (기존)

- 침하 그래프 위험 단계가 Y축 방향과 반대
- 침하가 커질수록 더 안전한 것처럼 보임
- 침윤선과 피에조미터 수두 불일치
- 간극수압계를 개방 standpipe처럼 표현
- 누수계를 제체 내부 센서처럼 표현
- 댐 수리역학과 안전관리 로직 불일치

### 1.2 수정 방향

필댐의 **침윤, 간극수압, 누수, 침하, 변위, 데이터 흐름**을 설명하는 체계도. 단순 아이콘 나열이 아니라 **댐체 내부 물 흐름과 안전판정 방향**을 정확히 표시.

### 1.3 개선 프롬프트 (복붙)

[39 §12](./39-IMG-024-댐-안전관리-계측-체계도-전면-수정-계획.md) 및 ImageWorks 프롬프트 `최종 생성 프롬프트 (131)` 참조.

## 5. 최종 검수 규칙 (PASS 게이트)

이미지 생성 후 아래를 통과하지 못하면 **PASS 금지**.

### [공통]

- 그림 목적이 하나로 명확한가?
- 범례에 있는 항목이 실제 그림에 존재하는가?
- 실제 그림에 없는 센서명이 표·범례·캡션에 들어가지 않았는가?
- 측정 방향, 힘 방향, 기준점이 표시되어 있는가?
- AI 원본에 로고·워터마크·불필요한 제품 UI가 없는가?

### [IMG-024]

- 침하 그래프에서 아래로 갈수록 위험도가 증가하는가?
- 침윤선과 피에조 수두가 서로 모순되지 않는가?
- 간극수압계가 폐쇄형 filter tip 센서로 표현되었는가?
- 누수는 하류 배수·집수부에서 표현되었는가?

### [어스앵커 포함 — 공통 · 114 §12]

- [114 §10](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) 10문항 전부 「예」인가?
- 축선을 먼저 정했고, 정착장→노출 강연선이 **동일 사선 직선축**인가?
- 반력판이 **축선 직각**(화면 수직 아님)인가?
- 하중계가 **사선 원통/링**이며 **수평이 아닌가**?
- 노출 강연선이 하중계 축선 **연장**인가?
- 정착장이 배면 **깊은 곳**인가?
- IPI·지하수위와 앵커가 **충돌하지 않는가**?

### [IMG-006]

- 3·4단계 앵커가 **7시 방향(1시~7시)** 으로 벽체를 관통·배면 깊은 곳에 정착하는가?
- **수평(3·9시)·5시·굴착공중·역방향** 앵커가 없는가?
- [114 §9](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) · [58 §3.2](./58-IMG-006-굴착-단계별-계측-흐름도-표현-표준.md) 기하 충족?
- 어스앵커 하중계만 남겼는가?
- 버팀보·지반앵커·지보재 하중계가 제거되었는가?
- 범례·그림 일치?

### [IMG-004]

- [114](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) 전항 + T/P 분리?
- 하중계가 굴착측 외부 노출 두부인가?

### [IMG-005]

- 균열계가 균열을 가로지르는가?
- 구조물경사계가 건물 외벽 표면에 부착되어 있는가?
