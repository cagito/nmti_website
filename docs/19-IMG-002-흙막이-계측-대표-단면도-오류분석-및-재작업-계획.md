## 0. 결론 (Executive Summary)

업로드·배포 중인 IMG-002 v2는 **§14 배면·굴착 물리(C1~C4)** 는 대체로 개선되었으나, **「굴착현장을 설명하는 대표 단면도」** 로는 **부적합**하다.

| 관점 | 판정 |
|------|------|
| **역할** | 계측기 **카탈로그 조합도**에 가깝고, SOE·지보체계·하중 전달 **관계성** 부재 |
| **치명 (P1~P3)** | ⑥ **어스앵커 방향 역전**(굴착측) · ⑥ **앵커 하중계 조립 미표현** · **G.W.L–굴착저 수리학 불완전** |

| **조치** | 부분 수정 **금지** — **인간 CAD 우선** 또는 AI+§19+§3.1.1+**역학 육안 검수** → PNG v3 |

**Cursor 전달용 한 줄:**  
> 앵커가 굴착측으로 향해 구조 성립 불가 · 간극수압=관측공 동형 · G.W.L이 굴착저 위인데 배수/차수 없음 → **REGENERATE**.

## 2. 발견된 오류 (치명 → 중요)

### 2.1 P1 — 어스앵커(⑥) 방향 역전 ⛔ **최우선**

**관찰 (현행 Figure):**

- 좌→우: **배면(좌) | 벽체 | 굴착(우)**.
- ⑥ **앵커 하중계** 부근 **강연선·인장력 T** 가 **벽체에서 우츘(굴착 공동)** 으로 향함.

**공학적 모순:**

- Ground anchor / tieback은 **지반·암반에 정착**하여 벽체를 **당겨 지지** — 강연선은 **배면 지반(좌) 방향**으로 경사 관입해야 함.
- 굴착측 공중으로 나가면 **정착 지반 없음** → anchored system **성립 불가**.

**올바른 표현:**

- **두부 조립체** = **굴착측 노출**(반력판·하중계·앵커헤드).
- **강연선** = 헤드에서 **← 배면 지반** (자유장·정착장·그라우트).
- **인장력 T** = 앵커헤드 → **배면**(정착 방향).

**규칙 근거:** INSTRUMENTATION §3.2 · §3.1.1 P1 · IMG-002 prompt §어스앵커 · FHWA ground anchor 개념

> **엄격 수정 정본:** [82-흙막이-상부지보·앵커축-엄격-수정-요구](./82-흙막이-상부지보·앵커축-엄격-수정-요구.md) — **D-12** · 프롬프트 붙여넣기 블록 **필수**

---

### 2.2 P1b — 앵커 하중계(⑥) 설치 원리 미표현 ⛔

**관찰:**

- 하중계 주변 **반력판–플레이트–헤드–텐던** 순서·접촉면 불명.
- **T(인장)** 만 있거나 조립체가 **장식 아이콘** 수준.

**올바른 표현 (굴착측 → 배면):**

```text
벽체/띠장 → 지지링·반력판 → 하부플레이트 → 원통형 하중계 → 상부플레이트 → 앵커헤드 → 강연선(중심 관통) → 자유장 → 정착장 → 그라우트
```

- **접촉면 A/B**, **T**·**P** **별도** 화살표.
- **인셋 상세** 권장 — `ref-04_어스앵커하중계_참고.png`, IMG-004와 **일치** · **ANC-CLOCK** ([83](./83-어스앵커-하중계-ANC-CLOCK-정본.md))
- **하중계 장축 = 1시~7시 사선** (강연선 7시 방향과 **동축**) — **수평(3~9시) 금지**

**규칙 근거:** §3.2 · IMG-004 · docs/83

---

### 2.3 P2 — 지보체계(SOE) 정체성 불명 ⛔

**관찰:**

- **상부 앵커(⑥)** + **다단 버팀보(⑧)** + **하부 변형률계(⑨)** 가 **역할·시공 단계 없이** 병치.
- 벽체 = RC 슬래브, 띠장·스트럿 = H형강 — **반력 경로·waler 연속성** 미표현.

**올바른 표현 (택 1 — 재작업 시 **반드시 택일·명시**):**

| 옵션 | 단면 메시지 | 필수 라벨 |
|------|-------------|-----------|
| **A (권장)** | **혼합형** — 상부 tie-back + 하부 cross-lot strut | `상부 앵커 지지` · `하부 버팀보` · **띠장 연속** |
| B | **앵커식** 단면 | 버팀보 **생략** 또는 가설 단계 표기 |
| C | **버팀보식** 단면 | 앵커 **생략** |

**금지:** 옵션 불명 + 장비 아이콘만 나열.

**규칙 근거:** §3.1.1 P2 · SOE monitoring 목적(벽체 변형·지보재 축력·배면 거동)

---

### 2.4 P3 — G.W.L·굴착 수리학 불완전 ⛔

**관찰:**

- **G.W.L** 이 **굴착저(굴착 바닥)보다 높음**.
- 굴착 공동은 **건조** — **배수·양수·차수벽 근입·Leakage** 개념 없음.

**공학적 모순:**

- 굴착저 < G.W.L 이면 **굴착부 침수 또는 지하수 제어**가 전제 — 연속벽 **차수·근입**, **Wellpoint/Deep well**, 또는 **굴착저를 G.W.L 아래**로 조정 중 **하나**는 Figure에서 **읽혀야** 함.

**올바른 표현 (택 1):**

| 옵션 | Figure 표현 |
|------|-------------|
| **A (권장)** | G.W.L **유지** + 벽체 **차수/근입** 콜아웃 + (선택) **양수 well** · `지하수위 저하` 점선 |
| B | G.W.L을 **굴착저 아래**로 조정 — **piezometer·관측정 위치** 재정합 |
| C | 굴착 **1단계**(얕은) — G.W.L **아직 굴착저 위** + **배수 계획** 주석 |

**규칙 근거:** §3.1.1 P3 · §3.4~3.5(수위·수압은 **dewatering 추적** 맥락)

---

### 2.5 P4 — 지중경사계(①) Base·유형 ⛔

**관찰:**

- 라벨 **센서형 다단식** — 내부 노드 체인은 있음.
- **안정층 근입 깊이**·**Base(Reference)** 치수 **없음** — IMG-027 **P2** 와 동일 결함.
- (잠재) 수동형 probe + IPI 체인 **혼합** 인상.

**올바른 표현:**

- **센서형 다단식** 고정 — 케이싱 내 **다점 노드** + 케이블.
- **Base** = 안정층 **3~5 m+** 근입(치수·콜아웃).
- 수평변위 **→ 굴착측** (배면 외향).

**규칙 근거:** 공통「지중경사계 근입」·§3.3 · [17-IMG-027 §2.2](./17-IMG-027-지중경사계-설치단면도-오류분석-및-재작업-계획.md)

### 2.9b P8b — 상부 1열 실린더·지보 충돌 ⛔

**관찰 (Strict Engineering Correction):**

- 상부 1열(앵커·띠장) **빨간 원** 영역에 **기계 실린더·유압잭** 형상.
- 상부 앵커 지지가 **버팀보(스트럿) 레이아웃과 충돌** — SOE **읽기 불가**.

**올바른 표현:**

- 상부 = **띠장 + 앵커 두부(⑥)** — **실린더 제거**
- 하부 = **⑤ 버팀보** — **혼합형** 라벨 `상부 앵커 지지` · `하부 버팀보` · **띠장 연속**

**규칙 근거:** [82 §2.1](./82-흙막이-상부지보·앵커축-엄격-수정-요구.md) D-09 · §3.1.1 P2
