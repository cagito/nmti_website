## 지중경사계 — Figure 라벨 (필수 전칭)

> **모든 Figure·범례·Callout·ALT(도면 라벨)** 에서 지중경사계는 **`센서형 다단식 지중경사계`** 로 표기한다.  
> 「지중경사계」단독 라벨 · 침하계형 막대 · 단일 프로브만 · 「지중수평변위계」단독 **금지**.

| 구분 | 규칙 |
|------|------|
| **Figure 라벨** | `센서형 다단식 지중경사계` (번호 ① 등과 병기 가능) |
| **그림 표현** | 수직 케이싱 + **다점 센서 노드** + Base = 영향 심도/활동면 **하부 안정층** (설계·계획서 · [51 §5](./51-계측-도면-검수-공통-원칙.md)) |
| **구분** | **구조물경사계**(외벽·기초 표면) — 「경사계」단독 금지 |

상세: [INSTRUMENTATION §3.3](./INSTRUMENTATION_DRAWING_RULES.md) · [24-가이드 §2.1](./24-토목-계측-개념도-및-구성도-작성-가이드라인.md) · [ImageWorks 09](../ImageWorks/NMTI_Engineering_Image_Prompt_Package_v1/09_지중경사계_센서형-다단식_표현_표준.md)

**본문·SEO**에서는 KDS 용어 **지중경사계**를 분류명으로 쓸 수 있으나, **도면·이미지·프롬프트**에서는 **제목·범례·ALT = 전칭**, **본도 지시선 = `지중경사계` 또는 `IPI`** 허용.

## 지중경사계·지하수위계 — GL 천공 가시 (BORE-GL-01)

> **모든 단면·설치도:** IPI 케이싱·지하수위 관측공은 **지표면(GL)에서 천공** · **천공 개구부·보호함·well cap이 지표면에서 가시**.
