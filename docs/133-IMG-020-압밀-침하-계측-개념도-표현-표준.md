## 0. 강제 지시문

```text
이 그림의 계측 목적은 단 하나다: 연약지반 압밀 침하(시간-침하-간극수압) 계측
침하판 = 성토 하부/원지반 + 연장봉 지표 측정
BM = 성토 영향권 밖 안정지반 (침하판 상단 ≠ BM)
침하량 = BM 대비 침하판 표고 변화
간극수압계 = 연약층 내 압밀 보조 해석 (침하 직접 측정 아님)
그래프: 성토 단계 · u 상승·소산 · S-t (즉시침하·1차압밀·2차압축 구분)
```

## 1. 필수 요소

## 2. 금지 (P0)

- BM을 성토 영향권 안에 배치
- 침하판 상단을 BM으로 표기
- 간극수압계가 침하량 직접 측정
- 「침하 증가+수압 소산」만 — 단계 구분 없음
