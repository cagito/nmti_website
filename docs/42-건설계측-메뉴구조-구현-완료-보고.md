1. 개요 → 2. 측정 목적 → 3. 적용 현장 → 4. 설치 → 5. 측정 원리 → 6. 데이터 해석 → 7. 유지관리·관리기준 → 8. 관련 공종 → 9. FAQ

## 3. 정본 파일 맵

```
js/technology/
├── dictionary.js      # TREE · label · getBreadcrumb · getAllContentNodeIds
├── tree.js              # hidden 노드 스킵
├── content-loader.js    # 센서 전용 SECTION 순서
├── seo.js               # SPA meta·title
└── content-data.js      # build 산출 (수동 편집 금지)

scripts/
├── build-content-data.mjs           # getNode() 라벨 · meta · applications 전파
├── generate-technology-seo-pages.mjs
└── content-data/
    ├── categories.mjs
    ├── sensors.mjs
    └── instruments.mjs

technology/index.html    # SPA 셸
index.html               # 홈 12분야 카드
```

**라벨 변경 시 수정 위치 (1곳만):**

1. `dictionary.js` — `TREE` / `CATEGORY_LABELS`  
2. (선택) `categories.mjs` 등 본문 첫 문단  
3. `npm run build:content && npm run build:seo`

`build-content-data.mjs`에 **별도 LABELS 맵을 두지 않는다** — `getNode(id).label`이 title·meta에 반영된다.
