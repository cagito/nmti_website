# 교량 신축이음계 계측 표현 표준 (BRI-EJ)

**수립:** 2026-06-26  
**우선순위:** **P0 (교량 신축이음 Figure)**  
**대상:** `fields/bridge/expansion-joint` · **IMG-014** · `sensors/joint-meter` · INSTRUMENTATION **§3.23.3**  
**정본:** [INSTRUMENTATION §3.23.3](./INSTRUMENTATION_DRAWING_RULES.md) · [53-IMG-014 수정계획](./53-IMG-014-신축이음계-수정계획.md)

> **한 줄:** 신축이음계 Figure는 **교량 종·횡·수직 3축 변위계**가 아니라, **신축이음부 양측 구조물 간 상대 신축량**(늘음량+/줄음량−)을 계측하는 **신축이음계** 설치 개념도이다.

---

## 1. 목적

교량 신축이음 계측 개념도에서 다음을 **혼동하지 않는다**.

| 유형 | 정체 | 계측 수량 | Figure 역할 |
|------|------|-----------|-------------|
| **① 신축이음량** | 신축이음부 **상대 이동** (계측 **항목**) | 늘음량(+) · 줄음량(−) · **신축량(mm)** | 교량 **종방향** 이음부 간격 변화 — **주계측** |
| **② 신축이음계** | 이음부 양측에 고정된 **전용 센서** | ①과 동일 | 고정 지그·브라켓·보호박스·측정축 |
| **③ 신축이음(장치)** | 핑거형·모듈형 등 **이음 재료** | — | 금속 상판 **톱니형 맞물림** · 양측 상부구조 사이 **간격** |
| **④ 받침부 변위** | 받침 상·하부 **상대 변위** | 별도 항목 | 본 Figure **주항목 아님** |
| **⑤ 절대 좌표변위** | 교량 전체 **GNSS·광파** | 별도 항목 | 신축이음계와 **동일 개념 금지** |
| **⑥ 상부구조 처짐** | 연직 처짐 | **처짐계** 노드 | 본 Figure **주항목 아님** |

**혼동 시 오류:** X/Y/Z 3축 화살표 · 「종방향·횡방향·개폐량」 병렬 표기 → **일반 변위계**로 오인 · 신축이음계를 상판 위 **부유 장비**로 표현.

---

## 2. 용어 (KDS/KCS · 현장)

| 표준·현장 용어 | 정의 | 비고 |
|----------------|------|------|
| **신축이음량** | 신축이음부에서 측정하는 **상대 이동량** (계측 항목) | KCS 24 99 05 교량 유지관리·준공보고서 **「신축이음 이동량」** · 메뉴·필드 노드명 |
| **신축이음계** | 신축이음부에 설치하는 **전용 계측기** | 준공보고서·계측관리계획서 센서명 · `sensors/joint-meter` |
| **신축량** | 계측값 표기 — **늘음량(+)** · **줄음량(−)** | 관리기준은 **현장별** |
| **신축이음** | 구조 **이음 장치**(핑거형·고무·박스 등) | 센서명이 **아님** — 도면에는 **신축이음부**·**신축이음 장치** |
| **신축계** | 문헌·타 계측기 분류의 **일반 명칭** | 본 사이트 교량 맥락에서는 **신축이음계** 우선 · autolink 별칭 유지 |

**삭제·비권장 (독립 메뉴):** **종·횡변위** — 포괄적이며 센서·설치 위치 불명확. 상부구조 절대·상대 변위는 **처짐·받침·교각·GNSS·광파** 등 **별도 항목**으로 분리.

## 3. 측정 체계 (개념)

```text
[좌측 상부구조]  ←── 고정측 브라켓 ── 와이어식 신축이음계(와이어+본체) ── 이동측 브라켓 ──→  [우측 상부구조]
        │                    │              측정축(종방향)              │
        └──── 신축이음 장치(핑거형 상판 맞물림) ──── 간격(신축량) ─────┘

[보조]
  보호박스 · 케이블 보호관 → 데이터 로거(소형)

[금지]
  X/Y/Z 3축 변위 화살표
  횡방향·수직을 주계측으로 동등 표기
  교량 전체 좌표변위·받침부·처짐을 동일 Figure 주항목으로 혼합
  임의 관리기준 mm 표
  대형 데이터 흐름도
```

### 4.1 신축이음계 — 와이어식 (BRI-EJ-13)

NMTI Figure·프롬프트에서 **신축이음계(신축변위계)** 는 **와이어식**으로만 표현한다.

| 요소 | 표현 |
|------|------|
| 고정측·이동측 | 이음 양측 **앵커·브라켓** |
| 와이어 | 양 앵커를 잇는 **인장 와이어** — 측정축 = **종방향 신축** |
| 본체 | 와이어 이격·장력을 읽는 **변위 변환기**(드럼·하우징) |
| 보조 | 보호박스 · 케이블 → 데이터 로거 |

**금지:** LVDT 두 끝 **직결 단축형만** · 와이어 없는 일반 변위계 · 상판 **부유**.
