## 0. 강제 지시문 (프롬프트 맨 위 · Cursor/AI 복붙)

```text
연약지반 성토부 계측기 설치 배치도를 한국 토목·건설 계측관리계획서 스타일의 2D 기술 도면으로 작성한다.

목표:
- 계측 센서를 실제 현장 계측 배치도처럼 표현한다.
- 장식적 인포그래픽, 흐름도, 3D 그림, 캐릭터형 아이콘은 사용하지 않는다.
- 성토층 / 연약지층 / 지표면 / Sand Mat / 지표침하판 / 지표침하핀 / 센서형 다단식 지중경사계 / 간극수압계 / 지중침하계 / 지하수위계 / 토압계가 실제 위치 관계에 맞게 배치된 단면도로 작성한다.

도면 형식:
1. 전체는 흰 배경의 2D 기술 단면도 형식으로 작성한다.
2. 하단 제목은 “[ 계측기 설치 배치도 ]”로 표기한다.
3. 좌측에는 지층명을 세로 방향으로 표시한다. (성토 · 토압계 · 연약지층)
4. 상부에는 성토 사면과 수평 성토면을 표시한다.
5. 성토 상부에는 지표면과 Sand Mat를 구분하여 표시한다.
6. 센서는 지표면 기준으로 실제 설치 심도와 위치가 구분되도록 배치한다.
7. 각 센서는 번호 아이콘보다 실제 계측관리계획서 도면 기호와 라벨 중심으로 표현한다.

필수 배치 요소:
- 지표침하판: 성토 상부 또는 Sand Mat 부근에 수평판 형태로 배치
- 지표침하핀: 지표면 상단에 짧은 핀 형태로 배치
- 센서형 다단식 지중경사계: 성토 하부 연약지층을 관통하여 안정층 방향으로 깊게 설치
- 간극수압계: 연약지층 내 지정 심도에 삼각형 또는 센서 팁으로 표시
- 지중침하계: 중앙부에 수직 관 형태로 설치하고, 심도별 측정점 또는 자석링을 표시
- 지하수위계: 우측 관측공에 설치하고, 관측공 내부 수위선을 표시
- 토압계: 성토 하부 또는 구조물 접촉부의 수평 토압 작용 위치에 표시
- 연직측침하계 또는 연직침하계가 필요한 경우, 명칭을 정확히 검토하고 지중침하계와 혼동하지 않게 표현

센서 배치 원칙:
1. 센서형 다단식 지중경사계는 단순 막대가 아니라 깊이 방향 수직 계측관으로 표현한다.
2. 지중경사계 하단은 연약지층 하부 안정층까지 근입되는 것으로 표현한다.
3. 간극수압계는 수위계처럼 긴 관으로 그리지 않는다. 특정 심도 측정점으로 표현한다.
4. 지중침하계는 지표침하핀과 다르다. 지중침하계는 심도별 침하를 확인하는 수직 계측관으로 표현한다.
5. 지표침하판과 지표침하핀은 센서가 아니라 침하 측점 또는 측정 표식임을 알 수 있게 표현한다.
6. 지하수위계는 개방형 관측공 내부 수위 측정 장치로 표현한다.
7. 토압계는 수평 토압 작용 방향과 감지면 방향을 함께 표시한다.
8. 데이터 로거, 서버, 통신 흐름도는 넣지 않는다. 이 그림은 “계측기 설치 배치도”이며 시스템 구성도가 아니다.

치수 및 기준선:
- 지표면 선을 명확히 표시한다.
- 좌우 지중경사계 하단에는 근입 깊이 예시 치수를 “2.00”처럼 표시할 수 있다.
- 중앙 지중침하계 하단에는 예시 치수 “1.00”을 표시할 수 있다.
- 단, 이 치수는 일반 기준이 아니라 도면 예시 치수처럼 보이게 한다.
- 상부에는 계측기 배치 폭 또는 성토 폭을 나타내는 치수선을 넣는다.

라벨 표기:
- 모든 라벨은 한국어로 작성한다.
- 라벨은 센서와 선으로 정확히 연결한다.
- 센서명: 지표침하판 · 지표침하핀 · Sand Mat · 센서형 다단식 지중경사계 · 간극수압계 · 지중침하계 · 지하수위계 · 토압계
- “지표침하계”는 지표침하핀·지표침하판을 가리킬 때 사용하지 않는다.
- “수위계”와 “간극수압계”를 같은 장치처럼 표현하지 않는다.

그래픽 스타일:
- 계측관리계획서 도면처럼 간결한 2D 선도 · 색상 최소화 (흑·회·남·청록)
- 지층은 연한 회색 또는 베이지 · 그림자·입체·AI풍 장식 금지

절대 금지:
1. 흐름도 형식
2. 데이터 로거·서버·알람·모바일 화면
3. 센서 공중 부유 아이콘
4. 지표침하핀을 지표침하계라고 표기
5. 간극수압계를 지하수위계처럼 긴 관측공으로 표현
6. 지중침하계와 지표침하판 동일 개념
7. 지중경사계 과얕은 설치
8. 토압계 감지면 없이 사각형만
9. 지층명·센서명 겹침
10. 사람·차량·캐릭터·뇌·AI 일러스트

검수 기준:
- 계측사·시공사·감리사가 설치 위치를 이해할 수 있어야 한다.
- 센서와 측점이 구분되어야 한다.
- 수위계·간극수압계·지중침하계·지중경사계 역할이 혼동되지 않아야 한다.
```
