## 0. 즉시 폐기 (P0)

```text
- 상부구조, 교각, 교대, 받침, 기초의 위치 관계가 불명확함
- 처짐계가 주경간 처짐 δ를 설명하지 못함
- 기초침하와 광파 측정이 하나의 삼각형 아이콘으로 혼동됨
- 구조물경사계가 교각 표면 부착형 센서로 보이지 않음
- 케이블장력계가 케이블 축방향 장력 측정으로 보이지 않음
- 사장교와 일반 거더교 구조가 무의미하게 혼재됨
- 내부 검수 문구, BRI-01 같은 관리 코드, 로고, 워터마크가 노출됨
```

## 1. 전체 디자인 방향

- **2D 기술도면** · 흰 배경 · Deep Navy 선 · Cool Gray 구조물 · Teal callout · 최소 강조색
- 사진풍 · 3D 렌더 · 과도한 장식 · 마케팅 그래픽 **금지**
- **교량 구조 계층**이 먼저 읽혀야 함:

```text
상부구조 = 교면, 슬래브, 거더, 주경간
받침 = 상부구조와 교각/교대 사이
하부구조 = 교각, 교대, 주탑
기초 = 교각·교대 하부
```

각 계측기는 **해당 부재에 설치**된 것처럼 표현. 라벨만 붙은 점·의미 없는 원형 아이콘 **금지**.

## 2. 교량 형식

**사장교형 교량**으로 통일 (`⑥ 케이블장력계` 필수).

```text
교면/슬래브 · 거더 또는 PSC 박스 · 주탑 · 사장 케이블
교각 · 교대 · 받침 · 기초 · 지표면 또는 하천/지반 기준선
```

일반 거더교와 **혼재 금지**. 케이블 = 주탑→거더 연결 · 장력계 = **케이블 축선 위**.

## 4. 10종 계측기 표현 (요약)

| # | 계측기 | 위치 | 물리량 | 핵심 표현 |
|---|--------|------|--------|-----------|
| ① | 변형률계 | 거더/PSC 표면 | 축방향 변형률 | gauge plate · 측정축 |
| ② | 온도계 | 교면/거더 | 부재 온도 | 부재 내·표면 센서 (외기 아님) |
| ③ | 가속도계 | 교면/거더 | 진동·가속도 | 박스형 + 진동→ + 파형 |
| ④ | 처짐계 | **주경간 중앙** | 수직 처짐 **δ** | 기준선 + ↓δ (침하계·기초 아님) |
| ⑤ | 신축이음계 | 교대 단부 이음 | 개폐량 ↔ | 틈 + 양측 앵커 + ↔ |
| ⑥ | 케이블장력계 | **케이블 축선** | 장력 **T** | clamp/ring + T→ 케이블 동축 |
| ⑦ | 풍향풍속계 | **주탑 상부** | 풍향·풍속 | 로터/풍향계 + 바람→ |
| ⑧ | 무응력계 | 거더 내·기준 위치 | 크리프 보정 | 변형률계와 **구분** · 보정용 |
| ⑨ | 구조물경사계 | **교각/주탑 표면** | 기울기 **θ** | plate 부착 + θ + 수직 기준 (지중경사계 아님) |
| ⑩ | 기초침하·광파 | 기초 + 외부 시준 | 침하↓ · 프리즘 LoS | **⑩-A 침하 측점** + **⑩-B 프리즘·ATS** 분리 (삼각형 하나 금지) |
