## §0. 최상위 강제 지시문 (모든 P0 공통)

```text
계측은 장비 나열이 아니다. 특정 공학적 질문에 답하기 위해
측정 항목, 계측기 종류, 설치 위치, 측정 방향, 해석 방법을 정한다.
이 그림은 토목 계측 기술도면이다. 홍보용 일러스트가 아니다.
한 그림에는 하나의 계측 목적만 담는다.
센서·측점·계측기·시스템을 혼동하지 않는다.
센서는 실제 설치 위치에만 배치한다.
기준점·측정 방향·측정 물리량을 반드시 표시한다.
AI 생성 원본에 로고·워터마크·NMTI 유사 표기를 넣지 않는다. (후처리 scripts/lib/watermark_stamp)
대외 이미지에 PASS·REGENERATE·P0·BRI-01·DAM-01 등 내부 검수 코드를 넣지 않는다.
범례·캡션에 있는 항목은 본도에 반드시 존재해야 한다.
하단 장문 설명 박스·경고문·조립 순서표는 넣지 않는다. (1~2줄 참고만 허용)
이 중 하나라도 틀리면 그림 전체를 폐기한다.
```

## §1. 그림 목적

- **한 줄:** `이 그림의 계측 목적은 단 하나다: ____`
- One Figure = One 계측 목적 (P0-4)

---

## §2. 구조물/지반 구성

- 단면·입면·평면 **뷰 라벨** 명시 (VIEW-01)
- 가시설: `배면 지반 | 벽체 | 굴착측` (P0-5)

---

## §3. 계측기 설치 위치

- 설계·계획서 위치 각주 (임의 배치 금지)
- [INSTRUMENTATION](./INSTRUMENTATION_DRAWING_RULES.md) 해당 §3.x

---

## §4. 측정 물리량

- 라벨에 **종류명**만 (METHOD-01: VW·진동현식 금지)
- 센서 ≠ 로거 ≠ 측점 구분

---

## §5. 힘·변위·수두·침하·장력·압축력 방향

- **Teal** 측정 방향 화살표 · 감지면 · 기준선
- **PASS 게이트:** 설치 위치 · 물리량 · 방향 **3종 모두** 충족

---

## §6. 필수 라벨

- 본도에 실재하는 항목만
- KDS/KCS [용어기준](../book/KDS-KCS_용어기준.md)

## §8. 최종 PASS 기준

| # | 검수 | FAIL → REGENERATE |
|---|------|-------------------|
| Q1 | 설치 위치가 INSTRUMENTATION·image-knowledge와 일치 | ☐ |
| Q2 | 측정 물리량이 노드·캡션과 일치 | ☐ |
| Q3 | 방향·감지면·기준점 표시 | ☐ |
| Q4 | 범례=본도 정합 | ☐ |
| Q5 | 내부 검수 코드·AI 로고 없음 | ☐ |
| Q6 | 분야 혼합 없음 (MIX-01) | ☐ |

**등록:** `npm run register:figure` 또는 `register-external-figure.mjs` — `visual-grade PASS`는 Q1~Q6 전부 후에만.
