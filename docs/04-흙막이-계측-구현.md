### 1.2 v3 배포본 문제 (2026-06-25 검토 — **v4에서 해소**)

`IMG-002` v3 재검수 시 **불합격**이었던 항목:

| 문제 | 설명 | v4 |
|------|------|-----|
| **좌우 방향 반전** | v3: 굴착측=좌, 인접건물=우 | ✅ 인접건물=좌, 굴착측=우 |
| **하단 원격계측 흐름도** | 센서→로거→서버→대시보드→모바일 스트립 | ✅ 제거 |
| **데이터로거** | 범례에만 존재 | ✅ 단면 내 ⑪ 표시 |

### 1.3 v6 배포본 — 2026-06-25 추가 검토 (**REGENERATE**)

v6 PNG는 어스앵커·버팀보 하중계 등은 반영되었으나, 기술 검토에서 **치명 공간 배치 오류**가 확인되었습니다.

| # | 문제 | 수정 |
|---|------|------|
| C1 | 벽체 우측·건물 측 **굴착 공동**, 건물 기초 아래 뻥 뚫림 | **배면**(벽체↔건물) **연속 토사** — 건물은 토사 위 |
| C2 | 버팀보·굴착저가 배면·건물 아래 | 굴착·버팀보 = **굴착측(맨 우)** 만 |
| C3 | 옥상 「자동광파기 측정(프리즘)」 | **변위 타깃 프리즘**; 광파기 본체 = 부동점 |
| C4 | 구조물경사계 지반 속 모호 | 외벽·기초 **표면 부착** |

→ [INSTRUMENTATION_DRAWING_RULES.md](../docs/INSTRUMENTATION_DRAWING_RULES.md) §3.1 · 프롬프트 IMG-001 v3 · IMG-002 v7

### 1.4 단면 좌→우 (재생성 시 필수)

```text
[좌] 인접 건물 | 배면 지반 | 흙막이 벽체·띠장 | [우] 굴착측(버팀보·굴착저)
         (+ 배면 어스앵커, 지하수위선, 굴착 단계선)
```

**올바른 정의:**

```text
흙막이 계측 = 굴착에 따른 벽체·배면지반·지하수·지보재·인접구조물 거동을 동시에 감시하는 단면 배치도
```

## 2. 설계 원칙

### 2.1 이미지

| 원칙 | 내용 |
|------|------|
| 중심 | 굴착 단면 + 계측기 **실제 설치 위치** |
| 스타일 | CAD 단면도, 흰 배경, Deep Navy / Teal ([00_STYLE_GUIDE.md](../ImageWorks/NMTI_Engineering_Image_Prompt_Package_v1/00_STYLE_GUIDE.md)) |
| 분할 | 한 장에 과밀 시 오류 → 메인(002) + 상세(003~005, 062) |
| 금지 | 벽체 부착 수위계, 장식용 하중계, 터널(천단침하·내공변위) 혼입 |

### 2.2 콘텐츠

흙막이 벽체 계측 본문은 다음을 **통합** 서술합니다.

```text
벽체 수평변위 + 토압 + 지보재 하중 + 지하수위 + 지표침하 + 인접 구조물 거동
```

관리기준 표현은 KDS/KCS 정렬: **설계예상변위**, **최대허용변위** (단독 `1차(주의)` 경보 체계 금지).
