# 지표침하핀 vs 지표침하계 — 구분 및 자동계측 정책

**수립:** 2026-06-26  
**우선순위:** **P0 (최상위)** — 혼동 시 **계측 방식·장비·데이터 파이프라인** 자체가 달라짐  
**정본:** [book/KDS-KCS_용어기준.md](../book/KDS-KCS_용어기준.md) §3a · [INSTRUMENTATION §0.6](./INSTRUMENTATION_DRAWING_RULES.md) · [TERMINOLOGY.md](./TERMINOLOGY.md)

> **한 줄:** **지표침하핀 = 측량 표식** · **지표침하계 = 센서** · **프리즘/ATS = 측량 측점** — **같은 라벨·아이콘으로 쓰지 않는다.**  
> **자동 센서형 침하 Figure** → `지표침하계` · **수준·ATS·프리즘 Figure** → `지표침하 측점` / `프리즘` (로거 직결 **금지**).

---

## 1. 왜 구분해야 하는가

| 혼동 시 오류 | 결과 |
|--------------|------|
| 핀을 「침하계」로 라벨 | **수동 수준측량·ATS** 절차로 오인 |
| 침하계를 핀(T자)으로 그림 | **자동 로거 수집** 파이프라인 누락 |
| Figure에 핀만 표기 | 현장 **센서 설치도**와 불일치 · 견적·시방서 오류 |

**계측 방식이 근본적으로 다르므로** 도면·프롬프트·본문에서 **용어·기호·설치 원리를 혼용하면 안 된다.**

## 2. 정의 (본 저장소)

### 2.1 지표침하핀 (침하핀 · Settlement pin / benchmark pin)

| 항목 | 내용 |
|------|------|
| **정체** | 지표면에 박거나 설치하는 **측정 대상 표식** (철봉·핀·표철) |
| **계측 방식** | **수동** — 수준측량 · 정기 **자동광파기(ATS)** 측량 |
| **데이터** | 현장 측량 후 **기준점 대비 표고 변화** 산정 (간접) |
| **로거** | 없음 (별도 계측 장비·측량 소프트웨어) |
| **KDS 맥락** | 사면·지질 계측에서 「침하핀측량」 등 **수동 계측 방법** (참고·역사 문헌) |

```text
지표침하핀 설치 → 수준측량 또는 ATS 측량 → 기준점 대비 표고 변화 산정
```

### 2.2 지표침하계 (침하계 · Settlement gauge)

| 항목 | 내용 |
|------|------|
| **정체** | 지표·토층에 매설하는 **계측 센서** (침하판·연장봉·보호관·변위계 본체) |
| **계측 방식** | **자동** — 침하량 **직접 계측** |
| **데이터** | **데이터로거** 또는 계측장비로 **연속·주기 수집** |
| **노드** | `sensors/settlement-gauge` (침하계) · `sensors/layer-settlement-gauge` (층별침하계) |
| **KDS** | 표 4.1-1 **지표침하판** 등 — 연직 침하 **센서** 계열 |

```text
지표침하계(센서) 설치 → 침하량 직접 계측 → 데이터로거·계측장비 수집
```

### 2.3 대조표

| 구분 | 지표침하핀 | 지표침하계 |
|------|------------|------------|
| 성격 | **표식·표점** | **센서·장치** |
| 계측 | 간접 (측량) | 직접 (변위) |
| 자동화 | 수동·ATS 주기 | 로거 **자동** |
| Figure 기호 | T자 핀·철봉·**측점** | 침하판·센서 본체·케이블·보호관·**↓** |
| 본 사이트 Figure | **측량·ATS** — `지표침하 측점`·`프리즘` | **자동 센서** — `지표침하계` |

### 2.4 침하계 vs 침하 측점 (교량·구조물 — BRI-FND)

> **정본:** [47-교량-기초-침하-계측-표현-표준](./47-교량-기초-침하-계측-표현-표준.md)

| 구분 | 정체 | NMTI 자동 Figure |
|------|------|------------------|
| **침하 측점** | 표고 변화 **측정 위치** (센서 본체 아님) | **프리즘** + **ATS** · **GNSS** · 측량선 — **핀 아님** |
| **지표침하계** | 침하량 **직접 센서** | 침하판·연장봉·본체 + **지표침하계** 라벨 |
| **「침하계」모호** | 측점·핀·센서 **불명** | **금지** — BRI-FND-01 |

§4.1 「측점」단독 금지는 **지표침하핀(T자)** 혼동 방지. 교량 기초는 **「기초 침하 측점」** + **측정 수단(프리즘·ATS·GNSS)** 또는 **「지표침하계」** 로 구체화.

### 3.1 Cursor용 한 줄

```text
지표침하핀은 센서가 아니라 측량 표식이다.
자동 센서형 침하 Figure에서는 지표침하계로 표현한다.
수준측량·ATS·프리즘 Figure에서는 지표침하 측점 또는 프리즘으로 표현한다.
두 개념을 같은 라벨·아이콘으로 쓰지 않는다.
```

## 4. Figure 표현 규칙

### 4.1 지표침하계 (허용)

- 지표면에 **침하판** 또는 **연장봉·보호관** + **센서 본체**
- **연직 ↓** 침하 방향 화살표
- (권장) **케이블** → 데이터로거·함체 (단면 허용 범위 내)
- 라벨: **`지표침하계`** — 「침하핀」**금지** · 「침하계」모호 단독 **금지**
- 교량 기초: **「기초 침하 측점」** = 프리즘/ATS 또는 **지표침하계** — [47 BRI-FND](./47-교량-기초-침하-계측-표현-표준.md)

### 4.2 지표침하 측점·핀 (측량 Figure — 허용 범위)

- **`지표침하 측점`** · **`지표침하핀`** — **수준·ATS·주변지반(096)·주변건물(005)** 등 **측량 맥락**만
- **로거 케이블 직결 금지** — 측점 → 측량기 / 자동광파기
- **금지:** 위 표식에 **`지표침하계`** 라벨 부착 (P0)

### 4.3 금지 (모든 Figure)

- 자동 **지표침하계** Figure에 **T자 핀만** 그리고 센서 라벨
- EN `settlement pin` on **settlement-gauge sensor** hero
- 핀·프리즘·침하계 **동일 기호·동일 라벨**

### 4.4 예외·별도 (천단·BM)

| 항목 | 처리 |
|------|------|
| 천단침하 **외부 수준점** (IMG-061) | 터널 **수준측량 BM** — [49 TUN-CROWN](./49-터널-천단침하-계측-표현-표준.md) |
