## 0. Executive summary

### 최종 메뉴 (목표 = 달성)

```text
건설 계측 기술 자료
├ 구조물·공종별        (group-field)
├ 계측센서별          (group-sensor)
└ 계측 시스템         (group-system)
```

### 페이지 템플릿 (목표 — 3종 분리)

```text
구조물·공종별  → 현장 계측관리계획서·대표 단면도 중심
계측센서별    → 센서 적용·설치·해석 중심 (OEM·조달 무관)
계측 시스템   → 로거·통신·전원·원격·데이터 운영 중심
```

### 핵심 원칙 (구현 시 준수)

| 항목 | 적용 |
|------|------|
| 메뉴 구조 | 3대 그룹 ID 유지 |

| label | `dictionary.js` TREE·`CATEGORY_LABELS` 단일 정본 |
