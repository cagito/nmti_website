# 기획·구현 계획서

## 배경 및 목표

- **위치**: `d:/website/homepage/`
- **접속 URL**: `https://www.nmti.co.kr/homepage/`
- **기술**: 정적 HTML + CSS + JS only (빌드 도구 없음, IIS 즉시 호환)
- **핵심 파일**: `index.html`, `css/style.css`, `js/main.js`

### 브랜드 이미지

**"기술력 있는 토목·계측 전문 강소기업"**

- 과하지 않은 현대적 B2B 엔지니어링 홈페이지
- 신뢰감 · 정밀함 · 기술력 · 실무 경험
- 스타트업처럼 가볍지 않고, 대기업처럼 과장되지 않은 전문 강소기업 톤
- 장식보다 정보 구조와 정돈된 레이아웃 우선

### 디자인 콘셉트

**"Precision Monitoring for Safer Infrastructure"**

정밀 계측, 현장 데이터, 구조물 안전, 자동화 시스템을 시각적으로 보여주는 방향.

PDF 지명원의 핵심 요소(파란 라인, 계측 단면도, 자동화 시스템 흐름도, 현장 계측 이미지)를 현대적인 웹 UI로 재해석하되, PDF 원본의 공공문서 느낌은 제거한다.

### 10초 이해 목표

1. 토목·계측 전문 엔지니어링 회사
2. 자동화계측과 데이터 분석 역량 보유
3. 2004년 설립된 경험 있는 강소기업
4. 바로 전화·이메일 문의 가능

## 구현 단계

### Phase 1 — HTML (`index.html`)

- 시맨틱 HTML5, 앵커 `#about` `#services` `#strengths` `#contact`
- Hero 대시보드 패널, Strengths 4-step 플로우, Services SVG 아이콘
- SEO 메타태그

### Phase 2 — CSS (`css/style.css`)

- CSS 변수, Pretendard CDN
- Hero 그리드·센서 노드·단면 라인 SVG
- Nav glass, About 라이트, Services hover, Strengths 플로우
- 반응형 768px / 1024px

### Phase 3 — JS (`js/main.js`)

- 스무스 스크롤, nav 스크롤 전환, 햄버거, Intersection Observer fade-up

### Phase 4 — Contact + 지도

- 본사·지사 카드, tel/mailto, 구글맵 iframe

### Phase 5 — 검증

- 10초 이해 목표, 반응형, IIS 접속 확인
