## 1. 강제 지시문 (프롬프트 맨 위)

```text
어스앵커 하중계는 지중에 묻는 센서가 아니다.
반드시 굴착측에 노출된 앵커 두부에 설치한다.
```

## 4. 앵커 두부 조립 순서 (굴착측 → 지반)

```text
흙막이 벽체 또는 띠장
→ 반력판 (앵커 축에 직각)
→ 하중계 (앵커 축과 동축)
→ 앵커헤드
→ 강연선
→ 자유장
→ 정착장
→ 그라우트 정착부
```

※ 상세 현장에서는 **가압판·웨지·콘헤드** 등이 추가될 수 있음 — Figure에 표기 시 **순서·축 정렬** 유지 ([INSTRUMENTATION §3.2](./INSTRUMENTATION_DRAWING_RULES.md))

### 4.1 사선 축 정렬 (ANC-AXIS — **추가 핵심**)

어스앵커는 **배면 지반 방향으로 경사** 설치되므로, **앵커 두부도 앵커 축방향(사선)과 일치**해야 한다.

| 요소 | 표현 |
|------|------|
| **앵커 축** | 사선 · **앵커 설치각** 표기(권장) |
| **반력판** | 앵커 축에 **직각** |
| **하중계** | 앵커 축과 **동축** — 중심축 = 강연선 축 |
| **앵커헤드** | 하중계 **뒤쪽**에서 강연선 정착 |
| **정렬** | 반력판·하중계·앵커헤드·강연선 = **같은 사선 축상** |

**금지 (사선 위반):**

1. 앵커 두부 **수평** 표현  
2. 하중계를 **버팀보 하중계**처럼 수평 압축재로 표현  
3. 반력판·하중계·앵커헤드 **서로 다른 각도**  
4. 강연선은 **사선**인데 하중계만 **수평**  
5. 벽체에 **수직·수평 단순 부착** 장치처럼 표현  

### 4.2 시계방향 사선 (ANC-CLOCK — **필수 · 육안 1순위**)

> **정본:** [83-ANC-CLOCK](./83-어스앵커-하중계-ANC-CLOCK-정본.md) · 영향 Figure: `npm run list:anc-clock-figures`

단면도 관례(좌=배면, 우=굴착)에서 **앵커 두부를 중심**으로 시계를 읽는다.

```text
           12시
      9시           3시 (굴착측)
           6시

강연선(PC strand) = 7시 방향 (← 배면·안정 지반)
하중계 장축      = 1시 ~ 7시 사선 (앵커 축과 동일)
버팀보 LC        = 3시 ~ 9시 수평 (앵커 LC에 복사 금지)
```

| 검수 | 규칙 |
|------|------|
| **ANC-CLOCK-01** | 강연선이 **7시(← 배면)** 가 아니면 **폐기** |
| **ANC-CLOCK-02** | 하중계가 **수평(3~9시)** 이면 **폐기** — **버팀보 LC 오류** |
| **ANC-CLOCK-03** | 하중계 장축 = **1시~7시** = 강연선·헤드·반력판 **동축** |
| **ANC-CLOCK-04** | IMG-002 ⑥·인셋 = IMG-004와 **동일 ANC-CLOCK** |

**프롬프트 한 줄:**  
`강연선 7시 → 하중계 1시~7시 사선. 수평 하중계 금지.`

## 5. 하중 전달 원칙 (T · P)

| 기호 | 물리량 | 방향 |
|------|--------|------|
| **T** | 앵커 **인장력** | **강연선 축방향** (앵커헤드 → 정착장) |
| **P** | 하중계 **압축 반력** | **앵커 축방향** (두부 조립체 내부) |

- 강연선에 **T** 작용 · 하중계에서 **P** 계측  
- **「P = T」만 단독 표기 금지** — 하중계가 **두부 반력**을 계측한다는 **구조**를 보여줄 것  
- **T·P 별도 화살표** · 접촉면 A(반력판–하중계)·B(하중계–앵커헤드) 권장  

**계측 원리 박스 (하단·선택):**

```text
강연선 인장력 T → 앵커헤드 → 하중계 압축 반력 P → 반력판 → 띠장·벽체
```
