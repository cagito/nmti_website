# 사면·옹벽 — 와이어식 변위계 배치 표준 (SLO-WIRE-01)

**수립:** 2026-06-27  
**등급:** **P1 치명** (사면 구조물 변위 Figure)  
**정본:** [127 DISP-ATS-01](./127-변위계측-자동광파기-남용방지-및-와이어식-우선-표준.md) · [INSTRUMENTATION §3.30.2](./INSTRUMENTATION_DRAWING_RULES.md) · [36 §4.3③](./36-AI-이미지-생성-엔지니어링-프롬프트-가이드.md)

> **한 줄:** **와이어식 변위계 = 배면 사면(지표·성토면)** 에 설치한다. **옹벽·낙석방지벽 전면·벽체에 와이어 부착 금지.** 옹벽 본체 거동은 **프리즘(측점)** · (선택) 광학망 · 이음부 LVDT.

---

## 1. 설치 위치 구분

| 대상 | 와이어식 변위계 | 프리즘·광학망 | LVDT·변위계 |
|------|-----------------|---------------|-------------|
| **배면 사면** (옹벽 뒤·성토면) | **✅ 주계측** | (선택) | — |
| **옹벽 전면·벽체** | **⛔ 금지** | **✅ 측점** | 이음부만 (선택) |
| **낙석방지책 전면** | **⛔ 금지** | **✅** | — |

**이유:** 옹벽은 구조물 — 현장에서는 **배면 지반·사면 표면** 변위와 **옹벽 측점(프리즘)** 을 분리한다. 와이어를 벽면에 걸면 설치·유지·물리 표현이 모두 틀린다.

## 2. SLO-WIRE-01 규칙

| ID | 규칙 | 위반 |
|----|------|------|
| **SW-01** | 와이어식 변위계 = **배면 사면 지표** (성토·절토면) | REGENERATE |
| **SW-02** | 기준부 = 사면 **안정권** 암반·앵커 | 불안정 활동면 위만 |
| **SW-03** | 측정 헤드 = **사면 표면 측점** (토사면) | 옹벽 콘크리트 면 |
| **SW-04** | 옹벽 = **프리즘** + (선택) 부동점 ATS 시준선 | 와이어가 벽면 관통 |
| **SW-05** | 라벨 `배면 사면` · `와이어식 변위계` 분리 | 「옹벽 와이어」 혼동 |

```text
옹벽 거동은 프리즘 측점과 (선택) 광학 측량으로 표현한다.
```
