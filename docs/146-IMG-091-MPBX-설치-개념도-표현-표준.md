## 1. Figure 목적

**다점지중변위계(MPBX)** 천공 내 **다점 앵커·로드·헤드**로 **축방향(연직) 상대변위** 프로파일을 설명한다.  
**≠ 센서형 다단식 지중경사계(IPI)** · **≠ 신축이음계(039)** · **≠ 터널 내공변위(008)**.

## 2. 필수 표현

| 요소 | 표현 |
|------|------|
| **GL** | 지표면 라벨 · **well cap·천공 개구부** 가시 (P0-2) |
| **보호관** | MPBX 전용 — **4홈 IPI casing 아님** |
| **앵커** | **≥2~3개** 깊이별 anchor node |
| **로드** | 앵커–헤드 연결 · 그라우트 **≠ rod 본딩**(보호관 내) |
| **헤드** | reference head · LVDT/기계식 변위계 |
| **변위** | **축방향(Teal)** 화살표 — 수평만 금지 |
| **Base** | **안정 지반** 각주 |
| **라벨** | `다점지중변위계`, `앵커`, `로드`, `헤드`, `지표면` |

## 3. 금지 (MPX-01~03)

| ID | 금지 |
|----|------|
| MPX-01 | **4홈 inclinometer casing** · **프로브 휠** |
