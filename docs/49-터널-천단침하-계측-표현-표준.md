# 터널 천단침하 계측 표현 표준 (TUN-CROWN)

**수립:** 2026-06-26  
**우선순위:** **P0 (터널 천단침하 Figure)**  
**대상:** `fields/tunnel/crown-settlement` · **IMG-061** · INSTRUMENTATION **§3.11.2**  
**정본:** [INSTRUMENTATION §3.11.2](./INSTRUMENTATION_DRAWING_RULES.md) · [50-IMG-061 수정계획](./50-IMG-061-천단침하-수정계획.md)

> **한 줄:** 천단침하는 **터널 내부 천단 측점(프리즘·타깃)의 표고 변화 측량**이다. **외부 수준점–천단 와이어·앵커식 침하계·지반 관통 센서**로 그리지 않는다.

---

## 1. 목적

NATM 터널 천단침하 Figure에서 다음 **네 가지를 절대 혼동하지 않는다**.

| 유형 | 측정 대상 | 계측 수단 | Figure·노드 |
|------|-----------|-----------|-------------|
| **① 천단침하** | 터널 **천단부** 연직 변위 | 천단 **측점**·**프리즘** + **자동광파기**·수준·레벨 | **IMG-061** · `crown-settlement` |
| **② 내공변위** | 단면 **벽면·어깨** 상대 변형 | **내공변위계** Kit·측선 | **IMG-008** · `convergence` |
| **③ 지표침하** | **지표면** 연직 변위 | **지표침하계**·ATS·GNSS | **IMG-010** · `surface-subsidence` |
| **④ 지중침하** | 터널 **상부 지반** 내 연직 변위 | **층별침하계**·다점 지중 | IMG-010 복합 · `ground-displacement` |

**핵심 오류(현행 IMG-061 v1):** 천단침하를 **지반 밖에서 수직 와이어·앵커로 당기는 지중침하계·변위계**처럼 표현 — **NATM 천단침하 논리와 불일치**.

## 2. 올바른 천단침하 개념

```text
터널 굴착 단면 (내부 중심)
├ 숏크리트 라이닝 · 록볼트
├ ① 천단침하 측점 (천단부 프리즘·타깃)
├ ② (선택) 좌·우 어깨부 측점
├ ③ (선택) 내공변위 측선 — 별도 항목(IMG-008)
├ ④ 기준점·후시점 (터널 내 또는 영향권 밖 안정 지점)
├ ⑤ 자동광파기 / 레벨 / 수준
└ ⑥ 측량 시준선 (광학·레벨 — 와이어·케이블 아님)

산정: 기준점 대비 천단 측점 표고 변화 → δ 천단침하
```

**외부 수준점:** 수준측량·네트워크 **기준(BM)** 으로 사용 가능 — **천단과 물리적 와이어·앵커·수직 케이블로 연결하지 않음** ([SETTLE-01 §4.3](./42-지표침하핀-지표침하계-구분-및-자동계측-정책.md) · [49](./49-터널-천단침하-계측-표현-표준.md)).

## 4. 절대 금지 지시문 (프롬프트·AI 필수)

```text
절대 금지:
1. 천단침하계를 터널 상부 지반 밖에서 수직으로 내려 설치한 앵커식 센서처럼 그리지 말 것.
2. 외부 수준점과 터널 천단을 와이어 또는 수직 케이블로 직접 연결하지 말 것.
3. 천단침하를 지중침하계, 층별침하계, 앵커식 변위계처럼 표현하지 말 것.
4. 데이터 로거를 천단침하 계측의 필수 구성요소처럼 표현하지 말 것.
5. 터널 라이닝을 관통하는 수직 센서봉을 그리지 말 것.
6. 터널 축방향 다수 지점을 천단침하계 센서열처럼 표현하지 말 것.
7. 외부 수준점을 터널 천단침하 장치의 고정 지점처럼 표현하지 말 것.
8. 천단침하, 지표침하, 지중침하, 내공변위를 혼동하지 말 것.
```

## 5. Figure 표현 (IMG-061 v2)

### 5.1 좌측 주 단면 (필수)

| # | 요소 | 표현 |
|---|------|------|
| A | 터널 **내부** 단면 | 숏크리트·록볼트 · **인버트/노반** (통행 구간) |
| B | ① **천단침하 측점** | 천단부 **프리즘·타깃** · ↓ δ |
| C | ② (선택) **어깨부 측점** | 좌·우 스프링라인 |
| D | ③ **기준점** | 터널 내 후시점 또는 영향권 밖 BM (**와이어 연결 없음**) |
| E | ④ **자동광파기** | 터널 내 안정 브래킷 · **시준선**(점선) → 천단 프리즘 |
| F | (선택) 내공변위 | **별도 라벨** — IMG-008 연계, 혼동 금지 |

### 5.2 금지 (현행 IMG-061 v1)

- 외부 수준점 → 천단 **수직 와이어/광학선** 관통
- **천단침하계(앵커)** 라이닝·지반 관통
- **데이터 로거 함체**·CH-n 유선 계통
- **터널 축방향 다수 지점 계측** 센서열
- 지표침하·지중침하를 동일 단면에 **혼재**(천단 hero 한정 시 최소화)

### 5.3 우측 패널 (v2)

- **천단침하(δ)–시간 그래프** — 캡션 **「예시」** · **현장별 관리기준 적용**
- 임계 mm **보편 수치 금지**
- **데이터 흐름도 없음**

### 5.4 본문 설명 (권장)

> 천단침하는 터널 굴착 후 **천단부에 설치한 측점 또는 프리즘**의 표고 변화를 **기준점 대비 측량**하여 산정한다. 터널 단면별로 천단부·어깨부 측점을 배치하고, **막장거리**·시간에 따른 침하량 및 **수렴 여부**를 확인한다.
