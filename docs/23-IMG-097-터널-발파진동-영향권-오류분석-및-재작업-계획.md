## 2. 발견된 오류 (치명 → 중요)

### 2.1 P1 — **hero Figure 주제 오류** (교량 ≠ 터널 발파) ⛔

**관찰:** SEO·인터랙티브 hero = IMG-041 → **교량 deck**이 대표 이미지.

**공학적 모순:** 발파진동 계측 = **발파원(막장)** · **전파 경로** · **영향권 측점** · **PPV 기준** — 교량 거더가 핵심 대상 아님.

**올바른 바인딩:** `blast-vibration` → **IMG-097** (신규).

---

### 2.2 P2 — **영향권·발파원** 미표현 ⛔

**관찰:** plan/단면에 **막장·발파·진동파** 없음.

**올바른 표현:**

```text
[종단 또는 평면]
  지표 —— 영향권 경계(점선) —— 기준 측점(영향권 밖)
  터널 —— 막장(발파원) —— 숏크리트·암반
  진동계: 지표 / 인접 건물 기초 / (선택) 터널 외곽 지표
```

---

### 2.3 P3 — **측점 배치 논리** (콘텐츠 installation) ⛔

installation 6항 — Figure **미반영:**

- 영향권 도면·민감 시설
- **지표·지중·구조물** + 진동계 **동시**
- 발파 일지·장약·딜레이 **연동**
- 측점별 **관리기준**

**올바른 표현:** 최소 **3종 측점** 아이콘 + 라벨 (과밀 금지).

---

### 2.4 P4 — **3축 정의** (터널 좌표계) ⛔ (097 v2)

교량 Figure: X=종방(교량 축), Y=횡방.

**터널 발파:**

| 축 | 권장 |
|----|------|
| **X** | 터널 **축방향** (굴진 방향) |
| **Y** | **횡방향** (단면 수평) |
| **Z** | **수직** |

콜아웃 또는 소형 3축 도식 **필수** (041 복사 시 교량 축 라벨 **금지**).

---
