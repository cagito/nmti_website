## 2. 목표 Figure (v2)

### 2.1 레이아웃

```text
┌─────────────────────────────────────────────┬──────────────────┐
│  주 단면 (좌 70~75%)                          │ 계측 항목 요약    │
│  · 교각+상부(받침·신축)                        │ (우 25~30%)      │
│  · θ₁·θ₂·Δθ                                  │ 6줄 텍스트       │
│  · 받침부 변위계(상대)                         │                  │
│  · 프리즘+ATS 시준                            │ (선택) 추세축만  │
│  · 기초 지표침하계×n                          │ 「현장별 관리기준」│
│  · BM·영향권 밖                               │                  │
└─────────────────────────────────────────────┴──────────────────┘
```

### 2.2 필수 라벨 (한글)

| 라벨 | 위치 |
|------|------|
| 구조물경사계 (교각 상부) θ₁ | 상부 |
| 구조물경사계 (교각 하부) θ₂ | 하부 |
| Δθ — 교각 회전·기초 변형 | 상·하 사이 |
| 변위계 — 받침부 상대변위 | 받침 |
| 변위 타깃 프리즘 | 교각 상부 |
| 자동광파기 (부동점) | 영향권 밖 |
| 기준점 (BM) | ATS 인접·안정지반 |
| 지표침하계 | 기초 2~3점 |

### 2.3 금지

- 교각 측면 **단독** 「변위계(수평변위)」
- 기초 **직옆** 기준점
