```text
[114 — 축선 우선 · IMG-006 · ANC-7시]
단면 좌→우 = 배면 지반 | 흙막이 벽체 | 굴착측.
3·4단계 어스앵커는 벽체를 반드시 7시 방향(1시~7시 사선, 굴착측→배면 좌하)으로만 관통·정착한다.
수평(3·9시)·5시·굴착공중·배면→굴착 역방향 금지.

부품을 먼저 그리지 말고 어스앵커 축선을 먼저 설정한다.
정착장→자유장→벽체 관통→반력판→하중계→앵커헤드→노출 강연선 = 동일 사선 직선축.

IMG-006에서 하중계는 “어스앵커 하중계”만 남긴다.
버팀보 하중계, 지반앵커 하중계, 지보재 하중계는 넣지 않는다.
어스앵커 하중계는 굴착측 외부 노출 앵커 두부에만 둔다. 지중·굴착공간·와이어 중간 금지.
강연선, 반력판 중심, 하중계 중심축, 앵커헤드, 노출 강연선은 하나의 동일 사선 직선축 위에 둔다.
하중계는 수평 원통·수평 박스가 아니다. 강선 연장선 위 사선 링·원통이다.
반력판은 화면 수직판이 아니다. 앵커 축선에 직각인 기울어진 판이다.
굴착저면 라벨 사용. “굴착면” 단독 표기 금지.
범례에 있는 항목은 본 그림에도 동일하게 있어야 한다.
로고, 워터마크, 데이터로거 제품 사진, 서버·모바일 UI 금지.
```

## 3. 어스앵커 하중계 (ANC-AXIS · ANC-CLOCK)

### 3.1 부품 순서 (굴착측 외부)

```text
흙막이 벽체 외면 → 반력판(축선 직각) → 어스앵커 하중계(축선 방향 사선) → 앵커헤드 → 노출 강연선
```

### 3.2 축선·관통 방향 (ANC-CLOCK · **7시 필수**)

```text
[좌] 배면 지반 | 흙막이 벽체 | [우] 굴착측

관통축 = 1시 ~ 7시 사선 (굴착측 노출 두부 → 벽체 관통 → 배면 깊은 정착장)
강연선·정착장 방향 = 7시 (좌하 · ←배면) — 반드시 7시로만 관통
하중계·헤드 장축 = 1시~7시 (강연선과 동축)
```

| 올바름 | 금지 |
|--------|------|
| 7시로 배면 깊은 정착 | 3·9시 수평 |
| 굴착측 두부 → 벽체 → 배면 | 5시·굴착저면 방향 |
| 3·4단계 동일 7시 각도 | 굴착공중만 돌출 |
| | 배면→굴착 역방향 |

- 배면 강연선 → 벽체 관통점 → 반력판 중심 → 하중계 중심축 → 앵커헤드 → 노출 강연선 = **동일 사선 직선축**
- **수평(3~9시) 폐기**

### 3.3 금지

- 수평 하중계 · 지중 하중계 · 강선과 다른 각도
- **3·9시·5시 앵커 관통** · **7시가 아닌 사선**
- 화면 수직 반력판 · 축선 비직각 반력판
- 정착너트·육각너트 과장

### 3.4 생성 전 · PASS

- [114 §9.5](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) — **7시 관통** 포함
- [114 §8](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) 공통 체크리스트

## 8. Cursor용 생성 프롬프트 (131 · 114)

**먼저 붙일 블록:** [114 §9·§9.4](./114-모든-도면-어스앵커-공통-표현-절대원칙.md) (IMG-006 **7시 관통**)

```text
IMG-006 굴착 단계별 계측 흐름도.

단면 좌→우 = 배면 지반 | 흙막이 벽체 | 굴착측.
3·4단계 어스앵커는 벽체를 반드시 7시 방향(1시~7시 사선, 굴착측 노출 두부에서 배면 깊은 정착장)으로만 관통한다.
수평(3·9시)·5시·굴착공중·배면→굴착 역방향 앵커 금지. 단계마다 동일 7시 각도.

이 그림은 굴착 단계별 계측 운영 흐름도이다. 어스앵커 하중계 설치 상세도는 아니지만, 그림 안에 어스앵커 하중계를 표시할 경우 설치 위치·축선·7시 관통 방향은 반드시 정확해야 한다.

좌측 70%에는 1단계부터 최종 굴착까지 4단계 단면을 표 형식으로 배치한다. 단면은 배면 지반 | 흙막이 벽체 | 굴착측 순서로 구성한다. 단계가 진행될수록 굴착저면이 점점 깊어진다. “굴착면” 대신 “굴착저면”을 사용한다.

하중계는 “어스앵커 하중계”만 표시한다. 버팀보 하중계, 지반앵커 하중계, 지보재 하중계는 넣지 않는다.

어스앵커는 굴착측 노출 앵커 두부에서 벽체를 7시 방향(1시~7시 사선)으로 관통하여 배면 지반 깊은 정착장에 정착한다. (배면→굴착 역방향·수평·5시 금지)
어스앵커 하중계는 지중에 있으면 안 된다. 하중계는 반드시 흙막이 벽체의 굴착측 외부, 노출 앵커 두부에 설치한다.

가장 중요한 축선 규칙:
배면 강연선, 벽체 관통점, 반력판 중심, 어스앵커 하중계 중심축, 앵커헤드, 노출 강연선은 모두 하나의 동일한 사선 직선축 위에 있어야 한다.

하중계는 수평 원통이나 수평 박스로 그리지 않는다. 하중계는 강선 연장선 위에 끼워진 사선 방향의 짧은 링형 또는 원통형 부품으로 표현한다. 하중계의 중심축은 어스앵커 강선 중심축과 완전히 일치해야 한다.

반력판은 화면 기준 수직판이 아니다. 반력판은 어스앵커 축선에 직각인 기울어진 판으로 그린다. 반력판 중심은 어스앵커 축선 위에 있어야 한다.

굴착측 외부 부품 순서:
흙막이 벽체 외면 → 반력판(축선 직각) → 어스앵커 하중계(축선 방향 사선) → 앵커헤드 → 노출 강연선.

우측 30%에는 운영 루프를 작게 배치한다.
굴착 → 계측 → 데이터 수집 → 관리기준 비교 → 경보 판정 → 다음 단계 굴착.

범례는 최소화한다. 범례에 있는 항목은 반드시 그림에도 동일하게 있어야 한다.

금지: 수평 어스앵커 하중계, 지중 하중계, 강선과 다른 각도의 하중계, 7시가 아닌 앵커 관통(수평·5시·역방향), 화면 수직 반력판, 축선과 직각이 아닌 반력판, 버팀보·지반앵커·지보재 하중계, 범례만 있고 그림에 없는 항목, 데이터로거 제품 사진, 로고, 워터마크.

2D CAD, 흰 배경, Deep Navy·Cool Gray·Teal, 한국어 라벨, 1920×1080.
```
