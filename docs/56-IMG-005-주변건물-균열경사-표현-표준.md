## 3. VIEW-01 — 단면·입면 혼합 금지 (치명 · 핵심 오류)

> **창문 방향 이전에** 가장 흔한 실패 = **단면도와 입면도를 한 평면에 설명 없이 혼합**

### 3.1 금지 (폐기)

- 건물 **입면**(창문·외벽 정면)과 **지반 단면**(토층 해칭·굴착 공동)을 **같은 그림 평면**에 겹침
- **시점 혼동**: 원근 입면 + 지질 단면이 **한 장**에 — **뷰 전환 표기 없음**
- 건물을 **3D 입체**로 그리면서 배면 토사를 **2D 단면**으로 관통 표현

### 3.2 허용

| 방식 | 요건 |
|------|------|
| **A. 단일 단면도** | 건물 = **절단면**(단면 컷) · 외벽에 균열계·경사계 부착 · **창문 입면 원근 금지** |
| **B. 분할 패널** | `단면도` / `입면도 (건물 외벽)` **경계·제목·뷰 기호** 명시 |
| **C. 단면 주 + 입면 인셋** | 인셋에 **「입면도」** 라벨 · 단면 본문과 **구분선** |

**검수:** 뷰가 2종 이상이면 **라벨·경계**가 있는가? 없으면 **REGENERATE**.

## 4. 단면 좌→우 (FIG-03)

```text
[좌] 인접 건물 → 배면 지반 → 흙막이 벽체 → [우] 굴착측
```

- 배면 = 건물↔벽체 사이 **연속 토사** (허공 금지 · C1)
- 건물 = **지표면 위** 안착 (C0 · [27](./27-지표면-건물-안착-원칙.md))

## 6. 건물 계측기

| 항목 | 표현 |
|------|------|
| **균열계** | 외벽 균열에 **직교(가로질러)** |
| **구조물경사계** | 외벽·기둥 **표면 부착** · **θ** · **≠ 지중경사계** |
| **변위 타깃 프리즘** | 옥상·측점 — **광파기 본체 아님** |
| **자동광파기** | 굴착 **영향권 밖** 안정 지반(부동점) + **시준선** → 프리즘 |
| **금지** | 건물에 광파기 본체 · 「자동광파기 측정(프리즘)」 복합 라벨 |
