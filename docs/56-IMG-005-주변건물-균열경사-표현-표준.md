## 1. 강제 지시문 (프롬프트 맨 위)

```text
주변건물 균열·경사 계측도는 건물 외벽 계측 위치와 굴착 영향 관계를 설명하는 2D 기술 개념도이다.
단면도와 입면도를 한 평면에 섞지 않는다.
좌측은 “건물 외벽 입면도”, 우측은 “굴착 영향 단면도”로 명확히 분할한다.

건물 창문은 같은 크기의 단순 사각형으로 수직·수평 정렬한다. 창문 방향, 크기, 간격이 다르면 폐기한다. 사다리꼴·기울어진 창문·원근 창문을 그리지 않는다.

건물에는 균열계, 구조물 경사계, 변위 타깃 프리즘(측량 측점)만 표시한다. 자동광파기·Total Station 본체·시준선·광학망을 그리지 않는다.

지표침하는 건물과 흙막이벽 사이 배면 지표면에 “지표침하 측점” 또는 “지표침하핀”으로 표시한다. “지표침하계”라고 쓰지 않는다.

우측 단면도에는 배면 지반, 굴착 영향권, 지표침하 측점, 흙막이벽, 굴착저면만 표시한다. 오른쪽 일반 지반 수직벽, 지반에 직접 연결된 강재, 반력 구조 없는 스트러트는 그리지 않는다.

로고, 워터마크, 하단 설명 박스, 큰 범례 박스, 하단 아이콘 범례를 넣지 않는다.
이 중 하나라도 틀리면 그림은 폐기한다.
```

## 3. VIEW-01 — 단면·입면 혼합 금지

### 3.1 금지

- 건물 입면(창문·외벽 정면)과 지반 단면(토층 해칭·굴착 공동)을 같은 그림 평면에 무라벨로 겹침
- 원근 입면과 지질 단면을 한 장에 섞고 뷰 전환 표기를 하지 않음
- 건물을 3D 입체로 그리고 배면 토사를 2D 단면으로 관통 표현
- 창문이 어느 방향을 바라보는지 불명확한 단일 장면 구성

### 3.2 허용

| 방식 | 요건 |
|------|------|
| **A. 단일 단면도** | 건물 = 절단면 또는 단순화된 외벽면, 창문 입면 원근 금지 |
| **B. 분할 패널** | `건물 외벽 입면도` / `굴착 영향 단면도` 경계·제목 명시 — **권장** |
| **C. 단면 주 + 입면 인셋** | 인셋에 `입면도` 라벨, 단면 본문과 구분선 |

**검수:** 뷰가 2종 이상이면 패널 라벨·경계가 있어야 한다. 없으면 **REGENERATE**.

## 5. 건물·창문 표현

| 항목 | 표현 |
|------|------|
| **건물** | 단순 직사각형 입면, 수직·수평 기준 유지 |
| **창문** | 같은 크기, 같은 방향, 같은 간격, 같은 행·열 정렬 |
| **균열** | 국부 벽체 균열, 과장 금지 |
| **균열계** | 균열을 가로질러 직교 부착 |
| **구조물경사계** | 외벽 또는 기둥 표면에 수직 부착, θ 표현 가능 |
| **변위 타깃 프리즘** | 옥상 또는 외벽 측량 측점 — **ATS 본체 없음** |

**창문 폐기 조건:**

1. 크기가 제각각  
2. 사다리꼴·마름모·기울어진 창문  
3. 같은 층에서 방향이 다름  
4. 원근 왜곡으로 입면 방향이 애매함  
5. 창문이 계측기·균열·라벨과 겹침  

---

## 6. 흙막이·굴착부 표현

- 이 그림은 흙막이 지보 상세도가 아니다.
- 흙막이는 **수직 벽체 + 굴착저면** 중심으로 단순 표현한다.
- 우측에 또 다른 일반 지반 수직벽을 만들지 않는다.
- 스트러트는 원칙적으로 삭제한다.
- 스트러트를 그릴 경우 양 끝에 흙막이벽·띠장·중간파일·반력 구조가 있어야 한다.
- 일반 지반, 토사 절벽, 배면 지반에 강재를 직접 연결하지 않는다.
- 띠장·버팀보는 굴착측 벽체면 또는 굴착공간 안에만 허용된다.

## 7. 지표침하

- **위치:** 건물과 흙막이벽 사이, 배면 지표면(GL) 상의 측점
- **라벨:** `지표침하 측점` 또는 `지표침하핀`
- **금지 라벨:** `지표침하계`
- **금지 위치:** 건물 옥상, 벽체, 굴착측, 배면 지하
- **금지 표현:** 로거 직결 케이블, 자동 센서처럼 보이는 침하핀
