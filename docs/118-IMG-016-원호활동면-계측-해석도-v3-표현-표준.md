## 0. 최상위 강제 지시문 (P0 — 하나라도 위반 시 폐기)

```text
1. 최대 변위 심도를 원호활동면 위치로 단정하지 않는다.
2. 「원호활동면」 확정 표현을 사용하지 않는다.
3. 반드시 「추정 원호활동면」 또는 「잠재 활동면」으로 표기한다.
4. IPI 프로파일만으로 활동면을 확정하지 않는다.
5. 원호파괴 도식은 안정해석 검토 모식도이지 계측 확정 단면이 아니다.
6. 실측선과 추정선을 같은 의미로 그리지 않는다.
7. G.W.L, 간극수압, 지질 경계, 균열, 침하, 현장관찰을 병행 검토 요소로 분리한다.
8. 「단일 IPI만으로 활동면 확정 금지」 문구를 반드시 넣는다.
```

## 1. 기존 오류 반복 금지

- 점선 원호가 **확정 활동면**처럼 보임
- 추정 원호와 **안정해석 모식도** 미분리
- IPI 전단변형 집중 구간 = 활동면 오해
- 간극수압계 **사면 중간 부유** · G.W.L 관계 불명
- IPI **안정층 미근입** · 하부 안정층 **반투명 박스** 가림
- 하단 설명 박스가 **도면 침범**
- 문구는 금지인데 **그림은 확정**처럼 보임
- 생성 시 **로고·워터마크**

## 3. 좌측 사면 단면

**필수:** GL · 토사 사면 · 하강 방향 변위→ · 센서형 다단식 지중경사계(casing·다점 노드) · **최종 관입 심도** · **하부 안정층** · G.W.L · 간극수압 검토 지점 · **추정 원호활동면**(Orange 점선)

- IPI는 **GL 시작 → 잠재 활동면 하부 안정층까지** — 짧은 막대 **금지**
- 안정층 = **지층 구분** — 반투명 덮개 **금지**

**금지:** 활동면 위에서 IPI 종료 · 다점 센서 없는 막대 · 부유 센서

## 4. 추정 원호활동면

- **Orange 점선**만
- 라벨: `추정 원호활동면` · `잠재 활동면` · `안정해석 검토용 원호`
- **금지 라벨:** `원호활동면` · `확정 활동면` · `활동면` · `파괴면` · `실측 활동면` · `IPI 활동면`

**도면 내 필수 문구:**

```text
점선 원호 = 안정해석 검토 모식도
계측 확정 단면 아님
```

## 5. 누적변위–깊이 프로파일 (우 상)

- Y: 깊이 · X: 누적변위
- **실선:** IPI 측정 프로파일
- **음영/점선:** 전단변형 집중 구간
- **별도 점선 박스:** 활동면 추정 구간

**필수 문구:**

```text
전단변형 집중 구간 ≠ 활동면 확정 위치
최대변위 심도는 활동면 위치가 아님
```

**금지:** 최대변위 지점 = 활동면 · 「활동면 위치」 직접 표기 · 실선=점선 동일 의미

## 6. G.W.L · 간극수압

- **G.W.L** — 파란 선
- **간극수압계** — 특정 심도 **filter tip** (폐쇄형) — 개방 관측정 **금지**
- 간극수압 = **보조 판단**만 · G.W.L ≠ piezo

## 7. 안정해석 검토 모식도 (우 중)

- 단순화 사면 · Orange 점선 원호 · 하강→
- `원호파괴 안정해석 검토` · `점선 원호 = 검토 모식도` · `계측 확정 단면 아님`
- 계측 단면과 **동일 선형 금지** · IPI–원호 **직접 연결 확정 금지**

## 8. 하단 병행 검토 · 판정 원칙

**하단 얇은 밴드:**

```text
병행 검토: 지질 경계 · G.W.L · 간극수압 · 균열 · 침하 · 지표변위 · 현장관찰을 함께 판단
단일 IPI만으로 활동면 확정 금지  (강조)
```

**우 하 판정 원칙:**

```text
- 추정·검토·가능으로 표기
- 확정·인과 표현 금지
- IPI 프로파일은 병행 검토 자료
- 활동면 판단은 지질·수위·간극수압·균열·침하·현장관찰과 함께 수행
```

하단 박스가 IPI 하부·안정층·단면을 **가리면 안 됨**.
