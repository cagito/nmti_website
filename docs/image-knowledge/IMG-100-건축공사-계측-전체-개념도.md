# IMG-100 — 건축공사 계측 전체 개념도

> **node:** `fields/building` **hero** · **정본 배치:** [17-건축공사-계측-배치](./17-건축공사-계측-배치.md)  
> **분리:** IMG-099 처짐 상세 · IMG-101 주변건물 · [127 §2](../127-변위계측-자동광파기-남용방지-및-와이어식-우선-표준.md) DISP-ATS-01

## 1. Figure 목적

**KCS 3.9** 건축공사 계측 **5항목**(침하·경사·변위·변형률·균열·기둥축소)을 **한 건물 개요도**에서 **콜아웃**으로 통합 설명한다.  
**단일 센서 설치 상세도 아님** — `fields/building` 분야 **진입 hero**.

## 2. 필수 표현

- **RC/골조 건물** 입면·단면 윤곽 + **지표면** (P0-1: **1층 바닥·출입문 = 지표면**)
- **≥3종** 계측 **콜아웃** (번호·라벨 분리):
  1. **처짐·변위** — **LVDT·와이어식 변위계** (보·슬래브)
  2. **기둥 축소** — **변형률계** + **온도계** 병기
  3. **균열** — **균열계** (일상 신축 회피 위치 각주)
  4. **경사** — **구조물경사계** (선택 콜아웃)
  5. (선택) **부등침하** — **지표침하계** on 주요 부재·지표
- **와이어식·LVDT·변형률·균열·경사** = **주 표현** · **ATS·프리즘** = **「선택·보조」** 작은 inset 또는 생략
- 하단: **KCS 3.9** · **설계도서·계측관리계획** 각주

## 3. 금지 (BLD-H-01 · DISP-ATS-01)
