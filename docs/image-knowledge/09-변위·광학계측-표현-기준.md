# 변위·광학계측 표현 이미지 작성 기준

> **연계 노드(참고):** `sensors/displacement-transducer` · `sensors/automated-total-station` · IMG-040·042  
> **근거 PDF:** `book/` only — KDS §4.2.1.3 · 도시철도 수동·자동계측기 연구보고서

## 1. Figure 목적

- **구조물·지반 변위** 를 측정하는 **센서형 장비** 와 **광학 측량(타겟·광파기)** 의 **역할 분리** 및 설치·표현 원칙을 설명한다.
- 핵심: **와이어식 변위계·LVDT·침하계·내공변위계** = **직결 센서** · **프리즘·타겟·광파기** = **측량 측점·광학망** · **ATS hero = IMG-042 전용**.
- 설명하지 않을 것: GNSS hero(→07), 데이터로거 시스템, 흙막이 전체 단면, 브랜드 광파기·프리즘 상표.

## 2. 기술 대상

- **표현 대상(센서 1순위):** 와이어식 변위계, **LVDT(차동트렌스)**, 지표·층별침하계, 내공변위계, 처짐계, MPBX(별도 주제).
- **표현 대상(광학):** **반사 타겟·프리즘**(측점), **광파기·Total Station**, **기준점·시준선**, ΔX·ΔY(또는 3D) 벡터.
- **측정 대상:** 상대·절대 **변위·침하·처짐** — 목적별 1종.
- **표현하지 말 것:** 로거 직결 센서처럼 그린 **ATS** · 개념도 **만능 ATS** · 진동현식·VW를 변위계 **종류명**.

## 3. 설치 또는 표현 가능 구간

- **LVDT·와이어식 변위계(KDS 4.2.1.3·4.2.1.5):** 구조물 부재·이음부·지지부 **고정-이동점** 사이 **축방향 변위** — 현장계측 일반 변위변환기.
- **지표침하계·지표침하판(도시철도 표 3.1):** 인접 지반 **지표면 절대 변화** · 침하 속도 판단.
- **내공변위계·천단침하(도시철도 표 3.1·그림 3.4):** 터널 **내공·천정** — **핀+타겟+광파기** 수동 측량 방식 **현장 사례** (일반 hero 아님 — 센서·핀·타겟 분리 표기).
- **광파변위 타겟(표 2.5):** 가시설 구간 등 **별도 수량** — **센서 로거 입력과 구분**.
- **자동광파기(ATS):** **지반 하부 공사**(터널·깊은 굴착) · **IMG-042** 전용 — 지상 분야 hero **금지** (ATS-SUB-01 · docs/206).
- **계측 위치 선정(도시철도 §3.2.2·3.3.2):** **대표 단면** · 변형 예측 큰 구간 · 인접 구조물·과다 변위 우려 구간.

## 4. 설치 제한 구간 / 미표현 구간

- **ATS·프리즘** 을 **와이어식 변위계·LVDT** 와 **동일 라벨·동일 실루엣**.
- **전체 개념도** 에 ATS를 **습관적 기본 포함**(표 2.5 — 광파 타겟은 **선택·소수**).
- **광파기** 를 **데이터로거 함체** 안에 · **센서 케이블** 로 로거 직결 표현.
- **옹벽 전면·사면 표면** 에 와이어식 변위계(배면 성토면만 해당 시 별도 주제).
- **신축이음·받침 슬라이드** 에 ATS **단독 hero**(와이어·LVDT·변위계 우선 — repo 정책과 정합, PDF는 항목 분리).
- 브랜드 **KT-40B·Triple Prism** 등 **Figure 인쇄**(도시철도 그림 3.5).

## 5. 반드시 그릴 요소

## 6. 절대 그리면 안 되는 요소

- **ATS** 를 처짐·신축·받침·사면 **일반 hero**.
- **프리즘** 없이 광파기만 · **시준선** 없이 절대좌표 단정.
- **침하핀** 을 **지표침하계(센서)** 와 동일.
- **내공변위계(핀+광파)** 를 **자동 연속 센서** 로 오인되게.
- 카메라형 ATS · **뇌·AI** · 스마트폰 UI.
- 도시철도 **별내선 수량표** 를 모든 현장 **표준**처럼.

## 7. 적합한 Figure 유형

- **IMG-040:** 와이어·LVDT **설치 개념도** (부재 사이).
- **IMG-042:** **광학망 개념도** — 기준점·시준·프리즘·변위 벡터.
- **복합 개념도:** 센서 **주** + 광학 **보조 inset**(공사 중) — 동등 hero 금지.

## 8. 추천 이미지 구성

- **ATS(042):** 좌 굴착/교량 맥락 · 우 상단 광파기 · 하단 프리즘 측점망 · Teal ΔX·ΔY.

## 10. 오해하기 쉬운 표현

- **변위계 ≠ 경사계 ≠ 균열계** — 도시철도 표 3.4 항목 분리.
- **광파변위 타겟 ≠ 로거 센서** — 표 2.5 별도 항목.
- **수동 광파(터널 핀+타겟)** ≠ **ATS 연속망** — 맥락 구분.
- **LVDT** 는 KDS **일반 변위변환기** — 특정 브랜드 아님.
- **불확실 / 현장 조건 의존:** 자동 vs 수동 선택 · 측점 수 · 기준망 유지.
