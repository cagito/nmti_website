# 지하수위계·관측공 설치 이미지 작성 기준

> **연계 노드(참고):** `sensors/water-level-meter` · IMG-030·062  
> **근거 PDF:** `book/` only

## 1. Figure 목적

- **지하수위계(groundwater level meter)** 가 **관측정·보링공** 에 설치되어 **지반 내 지하수위** 를 측정하는 원리·구조를 설명한다.
- 핵심: **개방 관측공** · **GL 천공** · **G.W.L은 GL 아래** · 카사그란데 팁·필터·모래·벤토나이트 밀봉(굴착 맥락).
- 설명하지 않을 것: 간극수압계 전용 밀폐 필터, 데이터로거 시스템 hero, 흙막이 3분할, 브랜드 standpipe 상표.

## 2. 기술 대상

- **표현 대상:** 지표면, 관측공(PVC·보호관), **개방 수면**, 수위 센서(부호), well cap·상부 보호, 지층, **G.W.L** 점선, 배면/굴착 맥락(선택).
- **측정 대상:** **지하수위(관측정 내 수위)** — pore pressure at filter ≠ open standpipe level unless explicitly same purpose.
- **표현하지 말 것:** 벽체 표면 부착형, 공중 부양 관측공, G.W.L 지표면 위.

## 3. 설치 또는 표현 가능 구간

- **관측정 또는 보링공 내** 설치(KDS 표 4.1-1 · KCS 3.1.2.1②).
- **굴착에 따른 지하수위 변화 예상 지층마다** 설치(KCS 3.10.3.1③④).
- **카사그란데 팁** — 측정 목표 **지층 내**; 팁 상단 **0.5 m** 및 주변 **투수성 모래**; 상부 **1.0 m** 패커·불투수(벤토나이트 등) 후 나머지 **원지반과 유사 토사**(KCS 3.10.3.1③).
- **시추공 내경 86 mm 이상**, 연직도 **1° 이내**(KCS 3.1.3.1②).
- **모래 충전:** No.200체 통과 **5% 이하**(KCS 3.1.3.1④) — 간극수압·지하수위 공통.
- **Standpipe piezometer:** filter/well-screen + riser; water level in pipe = pore pressure at intake (guide §7 p.7).
- **지표면:** 천공 개구부·**well cap·보호함 가시** — BORE-GL-01은 repo 정책; PDF는 상부 **우·지표수 유입 방지** 타설(KCS 3.10.3.1③·보호).

## 4. 설치 제한 구간 / 미표현 구간

- **흙막이 벽체 표면**에 지하수위계 부착.
- **간극수압계** 와 **동일 긴 관측공**·동일 라벨.
- **G.W.L** 을 **지표면 위** 또는 **벽체 외부**에만 표시.
- well cap 없이 **지하에만** 관측공.
- **피압대수층 간극수압** 을 지하수위계 Figure에 hero(→ 05 주제).

## 6. 절대 그리면 안 되는 요소

- **G.W.L = 간극수압** 동일 기호·동일 관측공.
- 지하수위계 라벨을 **벽체**에 붙임.
- **수동 sounder·probe** hero(AUTO-01 repo와 정합 — PDF standpipe는 **수위 측정** 개념).
- 브랜드(VW piezo only figure에 혼입).
- 지표면 **위** 수위.

## 7. 적합한 Figure 유형

- **관측공 단면 설치 개념도** — guide standpipe §7 · KCS 3.10.3.1③.
- 복합 IMG-062는 **지하수위 + 간극수압 분리** 콜아웃 필수.

## 8. 추천 이미지 구성

- **제목:** `지하수위계 설치` / `지하수위·관측공`.
- **본도 70%:** 종단 단면 GL~목표 지층.
- **우 30%:** standpipe vs diaphragm piezo **구분** — 본 Figure는 **개방 관측정** 중심.
- **색상:** Teal 수면·G.W.L, Navy 관·지반, Gray cap·범례.

## 10. 오해하기 쉬운 표현

- **지하수위 ≠ 간극수압** — KCS 3.10.3.1③⑤ 분리.
- **Casagrande tip** — 지하수위·간극수압 **공용 설치방법**; **측정 목적**은 Figure마다 하나.
- **Standpipe** water level — intake depth의 pore pressure; **전체 지층 수위선**과 혼동 금지.
- **86 mm·1°** — KCS 시공; guide inch 표기 병기 시 **현장 조건**.
- **불확실 / 현장 조건 의존:** 천공 깊이, 지층 수, 배면 vs 원지반.

## 11. Cursor Agent 도면 지시

- **G.W.L** 점선은 **관측공 내부 수면**과 **수평 연결**.
- **well cap** 지표면에서 **명확히 가시**.
- **벽체 부착 금지** — 천공은 **배면 지반**.
- 간극수압 필요 시 **별도 짧은 밀폐 천공** — 같은 Figure면 **좌우 분리**.
- WebP · 외부 AI/CAD · 인간 검수.
