# 데이터로거·계측시스템 구성 이미지 작성 기준

> **연계 노드(참고):** `instruments/datalogger/static` · IMG-045  
> **근거 PDF:** `book/` only — **KDS §4.2.2** 정본 · 부록2는 **시스템 계층(1·2·3)** 만 참고(브랜드·모델 **Figure 금지**)

## 1. Figure 목적

- **계측자료 획득 장치(데이터로거)** 가 **센서·게이지** 로부터 신호를 받아 **저장·전송** 하는 **현장 구성** 을 설명한다.
- 핵심: **자료 획득 + 전송** · 로거 = **브릿지 전원·증폭·A/D·인터페이스** · **보호함** · **차폐 케이블·접지** · (선택) **다중화기·통신**.
- 설명하지 않을 것: 특정 제조사·모델(CR1000X·Gantner·e.bloxx 등), 서버 랙 hero, 빈 접속함만, 센서 설치 단면, 뇌·AI 회로.

## 2. 기술 대상

- **표현 대상:** 데이터로거 **함체(인클로저)** · 전면 **단자·LCD** · 센서 입력 케이블 · **스위치박스/다중화기**(채널 많을 때) · **차폐 케이블** · **보호함** · **전원(12V·배터리·태양광)** · **유선/무선 통신** · **저장·분석 컴퓨터**(블록).
- **측정 대상:** 정적 계측 **전기 신호** 수집(KDS 1.6.5① · 4.2.2.1).
- **표현하지 말 것:** 개별 센서(지중경사계·간극수압) 설치 hero · 계측서버 실물 랙 상세(부록2 IBM 등).

## 3. 설치 또는 표현 가능 구간

- **계측시스템 구성(KDS 4.2.2①):** 자료 획득(계측기기·스위치박스·변형측정기·**기록기/로거**·저장·제어 PC·분석 PC·주변기기) + **전송**(송·수신).
- **데이터로거 정의(4.2.2②):** **브릿지 전원, 신호증폭기, A/D변환기**, 교환기 제어, **컴퓨터 인터페이스**.
- **정적 로거(4.2.2.1①):** 신호처리·**자료저장 메모리**·**통신 인터페이스** · **스케줄 자동측정** · **자가진단·자동복귀** · 다중화기 **순차 측정**.
- **채널 확장(4.2.2⑪):** 채널 부족 시 **multiplex** 로 순차 주소 지정.
- **보호함(4.2.2.11):** 계측기기·**자동계측장치 보호** · 시건 · 점검 용이 크기.
- **전원·부속(4.2.2.5·8·9):** 전압안정 · **무정전(UPS)** · 항온항습(환경 시) · **접지(4.2.2⑩·13)** · **차폐 케이블(4.2.2⑨)**.
- **수집 방식(1.6.6~7):** 수동·반자동·**자동(유선·무선)** — Figure는 **자동 현장 로거** 중심.
- **부록2 계층:** `데이터로거(1) → 계측서버(2) → 계측센서(3)` — **역할 분리** 만 차용.

## 4. 설치 제한 구간 / 미표현 구간

- **로거 = 센서** 동일 실루엣·라벨.
- **로거 = 서버 랙** 동일 표현(부록2 서버는 **별도 블록**).
- **빈 박스**·「DATA LOGGER」 세로 텍스트만 — **함체·단자 없음**.
- **진동현식·VW** 를 로거·케이블 **종류명**으로(METHOD-01).
- 부록2 **Gantner·IBM·e.bloxx·ERLD** 등 **상표·가격·사양표** hero.
- **동적 고속 DAQ** 를 정적 토목 현장 로거와 **동일** 표기(구분 필요 시 각주).

## 5. 반드시 그릴 요소

- **데이터로거 함체** — 회색 직사각 **인클로저**, LCD·**전면 단자 블록**(IMG-045).
- **센서 입력** 화살표/케이블 ≥2종 — **신호 형식 구분**(브리지·4–20mA·디지털 등, 동형 1선 금지).
- **전원** — 12V·배터리 또는 **보호함 내 전원** 블록.
- **통신** — 유선(RS·이더넷) 또는 무선 → **원격 PC/서버** 블록.
- **저장** — 메모리·SD 또는 「자료 저장」 블록.
- (선택) **보호함** 외곽 · **차폐 케이블** · **접지**.
- 라벨: `데이터로거`, `센서`, `전원`, `통신`, `저장`(또는 `자료 저장`).

- 자동계측 **데이터 수집·저장·전송** 구성 — **방수 함체·산업용 인클로저** (빈 접속함·뇌 아이콘 금지, P0-3) — *KDS-11-10-15 §4.2.2*
## 6. 절대 그리면 안 되는 요소

- **Campbell·CR1000X·Gantner·e.bloxx** 인쇄.
- **빈 접속함**·범례 박스만.
- **서버·PC 본체** 를 Figure **중앙 hero**(로거가 주 피사체).
- **센서 ≠ 로거** 혼동 — 지중경사계 casing 을 로거로.
- **뇌·AI·사이버 회로** · 스마트폰 UI.
- 내부 CPU 다이어그램 **만**(외형 함체 없음).

## 10. 오해하기 쉬운 표현

- **데이터로거 ≠ 계측서버** — 부록2 **1·2** 분리; 서버는 분석·운영(4.2.3).
- **데이터로거 ≠ 센서** — 변환기·게이지는 **입력측**.
- **정적 vs 동적 DAQ** — KDS 4.2.2.1②; IMG-045는 **정적 현장** 기본.
- **반자동 vs 전자동** — 4.2.2.3; Figure 한 장에 **전 과정 UI** 넣지 않음.
- **보호함 ≠ 빈 박스** — 내부 **로거 함체** 가시(P0-3 repo와 KDS 4.2.2.11 정합).
- **불확실 / 현장 조건 의존:** 통신 매체(RS232/485/무선)·UPS 용량·다중화기 유무.
