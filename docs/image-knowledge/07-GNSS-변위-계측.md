# GNSS 변위 계측 이미지 작성 기준

> **연계 노드(참고):** `sensors/gnss` · IMG-043  
> **근거 PDF:** `book/GNSS.pdf` only (제조사·모델명 **Figure 금지**)

## 1. Figure 목적

- **GNSS** 를 이용한 **지표·구조물 변위(침하·3D 변위)** 계측의 **시스템 구성** 을 중립 CAD 개념도로 설명한다.
- 핵심: **기준국(고정)** · **이동국(2점 이상)** · **GNSS 안테나** · **무선 통신** · **중앙 서버** · **3D 변위** · (선택) **태양광 전원**.
- 설명하지 않을 것: 제조사(EGE·EGM-200 등)·가격·특허·앱 UI hero, 자동광파기·프리즘, **GNSS.pdf에 없는** 케이블 RTK 시준선.

## 2. 기술 대상

- **표현 대상:** 기준국 위치(변형 영향권 **밖**), 이동국 측점(#1,#2…), GNSS 안테나, 수신기·LTE·배터리·태양광, 중앙 서버, 위성(도식), ΔX·ΔY·ΔZ.
- **측정 대상:** **3D 변위** · 잔류침하 **모니터링** 개념(p.2).
- **표현하지 말 것:** PDF p.3–4 **제품 비교표·모델명·가격**; p.5 **앱·AI·지도 UI** hero.

## 3. 설치 또는 표현 가능 구간

- **기준국:** **안정 지반·암반** · 변형 영향권 **외부** · **고정 GNSS 안테나**(IMG-043·PDF p.2).
- **이동국:** 계측 대상(사면·댐·연약지반·항만 등) **2점 이상** — `#1`, `#2`, `#3` (p.2).
- **통신:** **LTE/무선** → **중앙 서버(데이터 수집·분석)** (p.2).
- **전원:** **태양광·배터리** — 원격·무인 (p.2).
- **위성:** GPS·GLONASS·BeiDou 등 **도식** (p.3 채널 — **과도한 사실 렌더 금지**).
- **정밀도:** p.3 **1 mm 후처리** 등 — Figure **수치 필수 아님**; **「고정밀 GNSS 변위」** 정도.

## 4. 설치 제한 구간 / 미표현 구간

- **기준국**을 **계측 대상 위**·변형 구역 **내부**에 배치.
- **단일 안테나**만(기준국 없음).
- **프리즘·ATS·광파기**.
- **제조사 로고·EGM-200·Smart Mega Port** 등 PDF **상표**.
- **스마트폰·CCTV·사람**.
- 위성에서 안테나로 **시준 점선** — **보정정보/무선** 과 **혼동** (IMG-043 규칙).

## 5. 반드시 그릴 요소

- **기준국** + **이동국 ≥2** + **GNSS 안테나** 각각.
- **중앙 서버** 블록.
- **무선 통신** 화살표(안테나/수신기 → 서버).
- **3D 변위** 벡터(ΔX·ΔY·ΔZ 또는 동·북·수직).
- **위성** 간단 도식(선택).
- **태양광** (원격 계측 표현, 선택).
- 라벨: `기준국`, `이동국`, `GNSS 안테나`, `중앙 서버`, `3D 변위`.

## 6. 절대 그리면 안 되는 요소

- EGE·EGM-200·NavStar·GMS800·**가격·연락처** (p.3–4).
- **무선침하계** 를 GNSS **동의어**로 — p.2 혼재 설명이나 **Figure는 GNSS 안테나** 중심.
- **AI 분석·앱 화면** hero (p.5).
- **KCS ±1.0 mm** 를 **모든 GNSS** 에 단정(p.4 표 각주 — **특정 제품** 맥락).

## 7. 적합한 Figure 유형

- **블록·개념도** (FT-B) — `book/GNSS.pdf` p.2 **중립 재해석**.
- **단면 설치도** 부적합(hero 아님).

## 8. 추천 이미지 구성

- **좌 65%:** 사면/댐/교량 **하나** + 기준국(부동) + 이동국.
- **우 35%:** 안테나 → 수신기 → 무선 → 서버 (p.2 유사, **브랜드 제거**).
- **16:9** · 흰 배경 · NMTI 중립 CAD.

## 9. 라벨 용어

| 라벨 | 용도 |
|------|------|
| 기준국 | 고정 측점 |
| 이동국 | 변형 측점 |
| GNSS 안테나 | 센서 |
| 중앙 서버 | 수집·분석 |
| 3D 변위 | ΔX·ΔY·ΔZ |
| 무선 통신 | LTE 등 |

## 10. 오해하기 쉬운 표현

- **GNSS ≠ GPS 측량(수동)** — p.3 비교표; Figure는 **자동·연속** GNSS 네트워크.
- **GNSS ≠ 지표침하핀·침하판** — 별도 방식.
- **기준국** 필수 — **상대 변위** 해석.
- PDF는 **제품 브로슈어** — **구조만** 차용, **상표·성능·가격** 제거.
- **불확실 / 현장 조건 의존:** 측점 수·기준국 거리·후처리/실시간.
