# 지중경사계 관측공 설치 이미지 작성 기준

> **연계 노드(참고):** `sensors/inclinometer` · IMG-027·028  
> **근거 PDF:** `book/` only

## 1. Figure 목적

- **지중경사계(inclinometer) casing** 의 관측공(천공)·케이싱·상부 종료·그라우팅 개념을 설명한다.
- 핵심 메시지: 안정 지반까지 casing 연장, 예상 이동 방향 groove 정렬, 수직·직진성, 상부 보호·종료, 그라우트 시 부력 대책.
- 설명하지 않을 것: 횡단면 흙막이 3분할, 데이터로거 통신, MPBX·지하수위계, 브랜드 제품명, traversing probe 조작 절차 전체.

## 2. 기술 대상

- **표현 대상:** 지반 단면, 천공(borehole), inclinometer casing(홈 있는 관), groove 방향, 안정 지반 구간, 상부 cap·보호함, bentonite-cement grout.
- **측정 대상:** 지중 **횡방향 변형** 프로파일(개념만 — 측정 원리도는 별도 Figure).
- **표현하지 말 것:** 가시설 벽체 띠장, 앵커 LC, 간극수압 필터만 단독, 지표면 위 전체 건물, 도로 노면 아래 casing.

## 3. 설치 또는 표현 가능 구간

- Casing은 **천공 내**에 설치; **안정 지반 10~20 ft(약 3~6 m) deeper** 관입을 계획·표현(심도는 `현장 조건에 따라 상이`로 일반화).
- 흙막이·옹벽 뒤 등 **예상 이동 방향**으로 groove 한 쌍을 정렬(대략 10° 이내 오차 허용 — PDF).
- **수직 설치:** 설계 수직에서 **3° 이내**; wavy casing 방지.
- 상부: casing top이 보호 enclosure 상단에서 **1~2 inch(약 25~50 mm) 아래** 또는 pulley 부착 여유 **10 inch clearance** 개념 표기.
- 그라우트: bentonite-cement grout backfill; piezometer 동축 설치 시 grout-in 가능(별도 목적 혼합 주의).
- In-place sensor vs traversing probe: 본 Figure는 **casing 설치 단면**에 집중; probe·logger는 최소화.

## 4. 설치 제한 구간 / 미표현 구간

- **도로·철도·노반·차량 통행면** 내부에 casing·천공을 그리지 않는다.
- Casing을 **상부에서만 눌러 고정**하는 방식을 정상 설치로 표현하지 않는다(부력으로 snaking·압축 발생 — PDF).
- 안정 지반 미관입·천공 개구부 없이 지표면 아래에 casing이 매몰만 된 그림.
- 흙막이 **굴착측 벽면**에 inclinometer를 띠장·앵커와 동일 면에 혼동 배치(SOE Figure와 목적 분리).
- 브랜드명(Slope Indicator 등)·영문 상표 로고.

## 6. 절대 그리면 안 되는 요소

- 지하수위계·간극수압계를 동일 관측공에 무라벨 혼합(G.W.L ≠ piezo).
- 센서형 다단식 **IPI 노드 체인**을 본 casing 설치도에 전부 전개(제품 혼동 — IPI 상세는 별도 주제, PDF 본 문서는 inclinometer casing 중심).
- 데이터로거 함체를 천공 내부에 배치.
- 사람·차량·시공 장비 hero.
- 카탈로그 제품 사진 복제·브랜드 로고.
- 현장 사진 픽셀 추적.

## 8. 추천 이미지 구성

- **제목:** `지중경사계 설치` 또는 `센서형 다단식 지중경사계` — 노드 정책과 충돌 시 용어기준·노드 라벨 확인 후 하나만 사용.
- **본도(70%):** 지표면—지반—천공—casing—grout 종단 단면.
- **우측 패널(30%):** groove 방향 vs 이동 방향; 안정 지반 깊이; 상부 termination; 부력 대책 스케치.
- **색상:** Navy 지반·casing, Teal 이동 방향·groove, Gray grout·보조선.
- **미표현:** 천공 하단 미탐사 구간은 생략 가능.

## 10. 오해하기 쉬운 표현

- **Inclinometer casing ≠ standpipe piezometer:** 동일 천공 그라우트인 경우도 목적·라벨 분리.
- **Traversing probe vs in-place:** 본 Figure는 casing 물리 설치; probe는 상부 접근만 암시.
- **수직 3°·직진성:** 깊은 천공일수록 wavy·snaking 오류 증가 — 과도한 곡선 casing 금지.
- **Settlement:** casing 길이· telescoping은 별도 고려 — 본 Figure에 과도 전개 금지.
- 영문 매뉴얼의 ft·inch는 도면에 **원칙만** 반영, 임의 m 환산 일반화 금지(필요 시 `현장 조건에 따라 상이`).

## 11. Cursor Agent 도면 지시

- **캔버스:** 16:9, ≥1920×1080, WebP.
- **지표면:** 상단 12% 영역에 수평선 + 라벨 `지표면`.
- **천공:** 지표면에서 수직(≤3° 시각적 수직); casing은 천공 중심, 하단은 안정 지반 3~6 m 표기(범위 주석).
- **Grout:** casing 외곽과 지반 사이 채움 — Navy 20% 명도.
- **상부:** 지표면 위 casing cap + 사각 보호함; 천공 개구부 **가시**(well cap).
- **부력:** casing 내부 점선 steel rod, 하단에서 25 mm 이격 suspension.
- **금지:** 브랜드, SVG, PNG 운영본, 흙막이 3분할 혼합.
- **출력:** `assets/images/technology/IMG-027_*.webp` 등 registry 정본.
- **방식:** 외부 AI/CAD + redline 인간 검수.
