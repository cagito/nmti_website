# 간극수압계 설치 이미지 작성 기준

> **연계 노드(참고):** `sensors/piezometer` · IMG-031·062  
> **근거 PDF:** `book/` only

## 1. Figure 목적

- **간극수압계(pore pressure / piezometer at filter)** 가 **보링공 내 지정 심도** 에 설치되어 **간극수압** 을 측정하는 구조를 설명한다.
- 핵심: **밀폐 필터 구간** · **카사그란데 팁 또는 간극수압계** · **지하수위계와 분리** · 피압대수층 · 모래·벤토나이트 충전.
- 설명하지 않을 것: 개방 standpipe 전체 수위 hero, 댐 침윤선 전체, VW 진동현식 라벨, 브랜드.

## 2. 기술 대상

- **표현 대상:** 지표면, 천공, **필터 팁**, seal·패커, 모래(80 μm 규격), 벤토나이트 상부 1 m, 지층, **간극수압** 측정 방향(수압).
- **측정 대상:** **간극수압 변동** — 활동면·피압층·굴착 영향 심도(KDS 표 4.1-1: 활동면 예측 부근).
- **표현하지 말 것:** G.W.L 단일선으로 간극수압 대체, 지하수위계 동일 관.

## 3. 설치 또는 표현 가능 구간

- **보링공 내 활동면 예측 부근**(KDS 표 4.1-1) · **피압대수층**(KCS 3.10.3.1③⑤).
- **카사그란데 팁** 위치 = 측정 **지층**; 팁 상단 0.5 m 모래; 상부 1.0 m 불투수; 잔여 원지반 유사 토사(KCS 3.10.3.1③).
- **필터 간극 ≤80 μm**, 필터 내 모래 **≥80 μm**(KCS 3.1.3.1⑤⑥).
- **측정범위** 설계 간극수압 고려, **기본 1 MPa 이상**(KCS 3.1.2.1⑤).
- **VW·pneumatic·strain-gauge** — guide §8–9; Figure에는 **「간극수압계」** 종류명만.
- **86 mm 천공**, **1° 연직**(KCS 3.1.3.1②).

## 4. 설치 제한 구간 / 미표현 구간

- **개방 관측공 전 구간**을 간극수압계로 표현(→ 지하수위계).
- **G.W.L** 을 간극수압 **측정값**처럼 라벨.
- **벽체 표면** 부착.
- **well cap 없이** 지하만 — 상부 **벤토나이트 펠렛 1 m** (그라우트 혼입 방지, KCS 3.1.3.1⑦) 누락(해당 시).
- 댐 **침윤선 전체** 도면(→ 댐 전용 Figure) — **filter tip 고도에 침윤선 맞추기 금지** ([152](../152-IMG-024-댐-피에조-침윤선-정합성-수정계획.md)).

## 5. 반드시 그릴 요소

- **지표면** · **GL 천공**.
- **짧은 필터 구간** + **seal/패커** (개방 관과 **시각적 구분**).
- **간극수압계** / **카사그란데 팁** 라벨.
- **모래 충전**(필터 주변) · **벤토나이트·불투수** 상부.
- **측정 심도** 콜아웃(활동면·피압층 — `현장 설계`).
- **G.W.L** 점선은 **참조만** — hero 아님, **≠ 간극수압**.
- 라벨: `간극수압계`, `필터`, `지표면`, `천공`.

## 6. 절대 그리면 안 되는 요소

- **지하수위계** 와 **동일 긴 open pipe** + `G.W.L` 만.
- **진동현식·VW** 측정방식 라벨(METHOD-01).
- **Piezometer = standpipe** 단일 표현(guide는 유형 구분).
- 브랜드·카탈로그 제품 사진.

- **G.W.L(자유수면) ≠ 간극수압** — 수위선·piezo 응답을 **동일 기호·동일 측점**으로 표현 금지 — *KCS-11-10-15 §3.10.3.1②*
- **G.W.L(자유수면) ≠ 간극수압** — 수위선·piezo 응답을 **동일 기호·동일 측점**으로 표현 금지 — *KCS-11-10-15 §3.10.3.1②*

- **밀폐 필터 단면 설치도** — KCS 3.10.3.1③ + guide diaphragm §8–9.

## 8. 추천 이미지 구성

- **제목:** `간극수압계 설치`.
- **본도:** 필터·seal·지층 강조; 수압 방향 **짧은 화살표**.
- **범례:** standpipe(개방) vs pore pressure(밀폐) **대비 1줄**.

## 10. 오해하기 쉬운 표현

- **G.W.L ≠ piezo** — repo·KCS 공통; Figure 필수.
- **활동면** — KDS 표 4.1-1 간극수압 **설치 위치**; IPI **Base** 와 다른 개념.
- **1 MPa** — 성능 하한; Figure에 수치 필수 아님.
- **불확실 / 현장 조건 의존:** 필터 심도·피압층 위치.

## 11. Cursor Agent 도면 지시

- **개방 관측공**과 **짧은 sealed tip** 동시에 그릴 때 **라벨·색 분리**.
- G.W.L은 **점선**, 간극수압은 **필터 부호 + u** 또는 `간극수압`.
- WebP · AI/CAD · 검수.
