# 통신·게이트웨이 역할 이미지 작성 기준

> **연계 노드:** `instruments/communication/iot-gateway` · `instruments/communication/lte-remote` · IMG-046·048·058  
> **근거 PDF:** `book/KCS 11 10 15` §3.1.2 · §3.2.4 · KDS §4.2.2·4.2.3  
> **정책:** **GW-ROLE-01** · **ARCH-01** (058)

## 1. Figure 목적

- **현장 데이터로거** 와 **IoT 게이트웨이·원격 통신** 의 **역할 분리** 및 **센서→로거→게이트웨이→서버** 데이터 경로를 설명한다.
- 핵심: 로거 = **현장 수집·저장·1차 변환** · GW = **중계·프로토콜 변환·보안·재전송** · GW **≠ 관리기준·이상 판정**.
- 설명하지 않을 것: GW=로거 동일 장치 · **뇌·사이버 구름 hero** · 브랜드 모뎀 · 센서 설치 단면.

## 2. 기술 대상

- **표현 대상:** 데이터로거 함체 · IoT 게이트웨이 · LTE/유선 통신 · 서버/클라우드 **블록** · 시간 동기 · (선택) 버퍼.
- **측정 대상:** **계측 데이터 전송** — 변위·하중 **물리량 아님**.
- **표현하지 말 것:** CR1000X·특정 제조사 · **게이트웨이가 경보 단계 판정**.

## 3. 설치 또는 표현 가능 구간

- **계측시스템(KCS 3.1.2):** 현장 **자동 수집** + **원격 전송** — 블록 분리.
- **로컬 저장(KDS 4.2.2):** 로거 **메모리·SD** vs GW **버퍼** — **별 아이콘**.
- **게이트웨이:** 프로토콜 변환 · **재전송** · **보안** callout.
- **시간 동기:** NTP/GPS **기준** — TS·로거·GW 정렬.
- **LTE(048):** 현장 GW → **무선** → 서버 — **유선 병행** 가능(각주).
- **058 아키텍처:** sensors → logger → power → comm → modes — **뇌 금지**.

## 4. 설치 제한 구간 / 미표현 구간

- **로거 = 게이트웨이** 동일 박스·동일 라벨.
- **GW = 관리기준 판정** · **경보 단계 결정** UI on GW.
- **서버 hero** — 로거·GW **중앙** 아님.
- **센서 케이블** 이 GW에 **직접** (로거 우회) — except MUX 확장 명시.
- 사례집 **특정 통신사·상표** hero.

## 5. 반드시 그릴 요소

- **순서 화살표:** 계측 센서 → **데이터로거** → **IoT 게이트웨이** → 서버.
- **로거:** 「수집·저장·1차 변환」 · **함체** 실루엣(P0-3).
- **GW:** 「중계·변환·재전송」 · **버퍼** (로컬 저장과 구분).
- **시간 동기** · **GW ≠ 판정** callout.
- (048) **LTE** · (058) **전원·통신·모드** 계층.

## 6. 절대 그리면 안 되는 요소

- **뇌·AI 회로** · **사이버펑크 구름**.
- **빈 접속함** = 로거.
- **게이트웨이 = 데이터로거** 라벨.
- **관리기준선** 을 GW 블록 **내부**에.
- **브랜드** CR1000X·Cisco·特定 LTE 모듈.

## 7. 적합한 Figure 유형

- **FT-C 블록·흐름** — 046·048·058.
- 단일 **센서 단면** — 본 주제 아님.

## 8. 추천 이미지 구성

- **046:** 좌→우 4블록 + **버퍼·시간동기** · **X on GW 판정**.
- **048:** 로거-GW-LTE-서버 + **유선 대안** 각주.
- **058:** 5계층 hub — comm = 29 규칙 준수.

## 9. 라벨 용어

| 라벨 | 용도 |
|------|------|
| 데이터로거 | 현장 수집 |
| IoT 게이트웨이 | 중계 |
| 서버 | 원격 저장·분석 |
| LTE | 무선 통신 |
| 시간 동기 | NTP/GPS |
| 재전송 | GW 버퍼 |

## 10. 오해하기 쉬운 표현

- **로거 ≠ GW ≠ 서버** — KDS 4.2.2·부록2 3계층.
- **경보 전송(KCS 3.2.4) ≠ GW가 판정** — **알림 경로**만.
- **058 ARCH** — comm 블록은 **역할도** not **제품 실사**.
- **불확실:** 유·무선 · 이중화 · 버퍼 용량 = **현장**.
