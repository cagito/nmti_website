# 기상·환경 보조계측 이미지 작성 기준

> **연계 노드:** `sensors/weather-station` · `fields/environmental-impact/*` · IMG-044·093·018(상관)  
> **근거 PDF:** `book/KDS 11 10 15` §4.2.3 · 도시철도 수동·자동계측기 연구보고서  
> **정책:** **WX-SITE-01** · **INTERP-01** (상관≠인과)

## 1. Figure 목적

- **강우·풍속·온습·기압** 등 **환경·기상 보조 자료**의 **센서 구성·설치 조건**을 설명한다.
- 핵심: 기상 = **보조 입력** — **단독으로 구조 변위·안정을 인과 단정하지 않음** · 지체(lag) 검토 callout.
- 설명하지 않을 것: 기상 hero로 변위·침하 Figure 대체 · 브랜드 기상대 · 도시철도 특정 노선 수량 일반화.

## 2. 기술 대상

- **표현 대상:** 강우량계(수평·장애물 이격) · 풍속·풍향계(표준 높이·차폐) · 온습도·기압(복사 차폐·환기) · 데이터로거 연결.
- **측정 대상:** 환경 변수 — **구조물 직접 변위 아님**.
- **표현하지 말 것:** 기상만으로 **관리기준 판정** · ATS·프리즘 · IPI casing.

## 3. 설치 또는 표현 가능 구간

- **강우량계(KDS 4.2.3):** **수평** · 주변 **장애물·요철 이격** · 배수·투명 수집구.
- **풍속계:** **권장 설치 높이** · 주변 **차폐·난류** callout(예시) — 고정 m 일반화 금지(DIM-01).
- **온습도·기압:** **복사 차폐·자연 환기** 실드.
- **데이터 흐름:** 센서 → 로거 → (게이트웨이) → 서버 — [29](./29-통신·게이트웨이-역할.md)와 역할 분리.
- **상관 분석 Figure(018):** 강우·수위·변위 **시계열 병렬** · **점선 지체** · 「상관 분석 — 인과 단정 금지」.

## 4. 설치 제한 구간 / 미표현 구간

- 기상 센서를 **흙막이 단면 hero** 로 — 단면 목적 Figure에 **inset·보조**만.
- **건물 옥상** 에 풍속계만 두고 **지표면 기상대 구성** 주장.
- **교육자료·지명원** PDF의 임의 설치 높이를 **모든 현장 표준**처럼.
- 기상 spike → 변위 spike **직선 인과** 화살표.

## 5. 반드시 그릴 요소

- **기상계측기** 제목 · **강우·풍속·온습·기압** 중 Figure 목적에 해당 항목.
- **장애물 이격** 또는 **설치 높이** — 점선·callout(예시).
- **보조 자료** · **인과 단정 금지** (018·044 공통).
- (018) **지체(예시)** 점선 · **관리기준(예시)** — 보편 기준 아님.

## 6. 절대 그리면 안 되는 요소

- **기상 = 변위 원인 확정** 라벨.
- **고정 지연시간** 모든 현장 공통값.
- **브랜드** 기상대 · **실제 노선·공구명**.
- **뇌·AI** · **사이버 구름 hero**.
- 지명원 PDF **교육용 스케치** 를 **KDS 설치도**처럼.
