# IMG-111 터널 건설중 계측 개념도

## 목적

터널 완성 후 유지관리 이미지가 아니라, 굴착·지보가 진행되는 **건설중** 상태에서 수행하는 대표 계측 항목과 위치를 설명한다.

## 산출 파일

- 운영 WebP: `assets/images/technology/IMG-111_external.webp`
- 생성 스크립트: `scripts/render-img111-construction.py`
- 실행 방식: `git-sync.bat` 또는 `git-sync` 실행 시 자동 렌더링

## 필수 표현 요소

- NATM 계열 터널 단면
- 굴착 진행 방향과 막장
- 숏크리트·강지보 등 지보 개념
- 내공변위 측점
- 천단침하 측점
- 지표침하 계측선 및 지표 측점
- 지중변위 또는 지중침하 보조 측정
- 로거/수집·전송 박스
- 굴착 → 지보 설치 → 계측 확인 단계

## 금지 오류

- `건설기간`이라는 표현 사용 금지. 반드시 `건설중` 사용.
- 완성 터널 유지관리 그림처럼 표현 금지.
- 내공변위와 천단침하를 동일 측선으로 표현 금지.
- 지표침하 측점을 터널 내부에 배치 금지.
- 내공변위계를 지표면에 배치 금지.
- 천단침하를 하부 노반 침하처럼 표현 금지.
- 굴착·지보 단계 없이 완성 단면만 표현 금지.
- 대시보드·서버·모바일 화면을 주제로 표현 금지.
- 계측 위치와 관계없는 센서 아이콘 남발 금지.

## 검수 기준

1. 제목·패널·주석에 `건설중`이 표시되어야 한다.
2. 그림 안에 굴착 진행 방향과 막장이 보여야 한다.
3. 내공변위, 천단침하, 지표침하 위치가 명확히 구분되어야 한다.
4. 지표침하 계측선은 지표면에 위치해야 한다.
5. 터널 내부 계측은 터널 내면 또는 천단 주변에 배치되어야 한다.
6. 한글 라벨이 축소 화면에서도 읽혀야 한다.
