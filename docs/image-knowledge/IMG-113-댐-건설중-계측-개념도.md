# IMG-113 댐·제방 건설중 계측 개념도

> **정본 용어:** [153-건설중-계측](../153-계측문서-용어-정본-건설중-계측.md) · **node:** `fields/dam/construction-phase`  
> **보조:** [15-댐-계측-배치](./15-댐-계측-배치.md) · [30-하천제방](./30-하천제방-계측-배치.md) · [152 filter tip·침윤선](../152-IMG-024-댐-피에조-침윤선-정합성-수정계획.md) (운영기 IMG-024와 구분)

## 목적

준공 후 **운영기 안전관리**(누수·간극수압·홍수기 패턴, IMG-024)가 아니라, **기초·제체·성토·제방 축조** 시공 중 침하·수압·변위·로거를 통합하는 **건설중 계측** 개념도를 설명한다.

## 산출 파일

- 운영 WebP: `assets/images/technology/IMG-113_external.webp` (호환)
- **hero:** `fields/dam/construction-phase`

## 필수 표현 요소

- **댐·제방 단면** — 축조 **진행 중** (성토층·콘크리트 타설·굴착 단계 중 하나 이상 가시)
- **공정 단계** — 기초·제체·성토 / 제방 축조 (콜아웃 또는 화살표)
- **건설중** 라벨 — 제목·패널
- **≥2종** 계측 — 역할 분리 예: **층별침하계** · **간극수압계**(filter tip) · **경사계** (전부 hero 아님 — 대표 2~3종)
- **데이터로거·함체** — 산업용 인클로저 · 통신→원격 개념(블록, UI hero 금지)
- **「설계도서·계측관리계획」** 각주
- (제방) **하천제방** 맥락 각주 가능 — [30](./30-하천제방-계측-배치.md)

## 금지 오류

- **`건설기간`** — **건설중** ([153](../153-계측문서-용어-정본-건설중-계측.md))
- **만수·홍수·운영기** 안전관리 패턴 (→ IMG-024)
- **filter tip = 침윤선** · G.W.L=간극수압 단일 곡선 ([152](../152-IMG-024-댐-피에조-침윤선-정합성-수정계획.md))
- **누수계**를 제체 내부에 박힌 센서처럼
- **흙막이 3분할** SOE
- **ATS·GNSS** 주 hero
- **PC·서버·모바일 대시보드** 주제
- **전 계기 나열** — One Figure = **건설중 통합 개념** (세부는 callout·별도 Figure)
- 침하핀(측량) = 층별침하계(매설) 동일 표현

## 5. 반드시 그릴 요소

- **댐·제방 단면** + **축조 진행** 맥락(성토층·타설면 등).
- **건설중 계측** 제목·공정 콜아웃.
- **≥2종** 센서 — 라벨 분리(`층별침하계`, `간극수압계`, `경사계` 등).
- (piezo) **filter tip·천공** — **침윤선 곡선 주 hero 아님**.
- **데이터로거(함체)** + 측점 연결.
- **「설계도서 위치」** 각주.

## 6. 절대 그리면 안 되는 요소

- **`건설기간`** · **만수·홍수 운영기** 표현.
- **filter tip 고도 = 침윤선** · **G.W.L=간극수압** 단일 곡선.
- **흙막이 3분할** · **ATS·GNSS** 기본 hero.
- **누수계** 오표현 · **브랜드·제품 UI**.
- 운영기 **7단계 흐름도**(IMG-024) **그대로 복사**.

## 검수 기준

1. **건설중** 표기 · 운영기 누수·침윤선 Figure와 목적 분리.
2. 공정 단계(축조·성토) 가시.
3. piezo = **측정 tip** · 침윤선은 **운영 해석**(본 Figure 주제 아님).
4. 로거 = **함체** · 빈 접속함 금지.
5. 16:9 · 한글 라벨 · NMTI 워터마크(후처리).
