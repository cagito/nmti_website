# 데이터로거 이미지 가이드 (레거시 산업용)

> **P0-3 (최우선):** Figure에 등장하는 **데이터로거 = 함체(인클로저)** — 회색 직사각 소형 산업용 로거(LCD·전면 단자) 또는 **방수보호함 내부** 함체. 범례·빈 박스·접속함만 **C0e · REGENERATE**. [TECHNICAL §0.0](../../docs/TECHNICAL_IMAGE_STANDARD.md)

> **2026-06-22 개정:** CR1000X 유형 기준을 **폐기**하고, `scripts/lib/datalogger_draw.py`의 **레거시 산업용 로거**(`draw_legacy_datalogger_front`)를 표준으로 고정합니다.  
> Campbell Scientific·CR1000X 등 **제3자 브랜드·모델명** Figure 인쇄 **금지**.

> **최종 갱신:** 2026-06-22  
> 공통 스타일: [00_STYLE_GUIDE.md](./00_STYLE_GUIDE.md) §데이터로거

## 1. 목적

기술자료 삽도·히어로 Figure에 등장하는 **정적 현장 데이터로거**를 토목·지반 계측 현장에서 흔한 **범용 산업용 소형 로거**(LCD·전면 단자 포트·측면 핸들)로 통일한다.

| 문제 (기존) | 목표 |
|-------------|------|
| 접속함·「DATA LOGGER」 세로 박스만 | **레거시 산업용** 정면 인클로저 |
| Figure마다 로거 모양 불일치 | IMG-045를 **표준 실루엣**으로 고정 |
| CR1000X·브랜드 인쇄 | **금지** — 형태만 참고 |

**용어:** 웹 본문·캡션은 「데이터로거」. 프롬프트·가이드에서도 브랜드명 대신 「레거시 산업용 로거」를 사용한다.

---

## 2. 외형 기준 (레거시 산업용)

Pillow `draw_legacy_datalogger_front` 기준:

| 요소 | 표현 |
|------|------|
| 외형 | 밝은 회색 직사각 인클로저, 전면 LCD·조작부·하단 원형 단자 포트 |
| 비율 | 가로:세로 ≈ **2.2:1 ~ 2.5:1** |
| 전원 | 12V 표기(녹색 포트 또는 POWER 라벨) |
| 설치 | 지표 **방수 보호함** 안 또는 개방, 센서 케이블 수렴 |

### 2.1 금지

- NEMA 대형 캐비닛, 정사각 금속 접속함만
- PC 본체·서버 랙·PLC 랙
- **Campbell Scientific**, **CR1000X** 등 브랜드·모델명
- 내부 CPU/저장/통신 **블록 다이어그램만** (IMG-045 구 버전)
- `draw_cr1000x_front` 신규 합성 (**deprecated**)
