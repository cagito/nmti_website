## 이미지 목적

흙막이 **배면 지반**에서 지하수위계·간극수압계가 **관측공·천공** 내부에 설치되는 위치와 수위선·수압 측정 방향을 보여주는 단면도.

## 필수 요소

- **「지표면」** 또는 **GL** 라벨·경계선
- 배면 지반 (연속 토사)
- 흙막이 벽체 (참조용, 센서 부착 금지)
- 지하수위선 (점선)
- **지하수위계** — **GL에서 천공** · **well cap·개구부 지표면 가시** (BORE-GL-01) · 관측공 내부 수위(▽)
- **간극수압계** — 지정 심도 **밀폐** 천공·짧은 필터 + seal (개방 관측공 **≠** 지하수위계)
- 굴착 영향 심도 (점선 또는 음영)

## 금지

- 지하수위계를 벽체 표면에 부착
- **well cap·개구부 없이** 지하에만 관측공 표시 (BORE-GL-01 위반)
- 지하수위계·간극수압계 **동일 긴 관측공**
- 굴착 바닥에만 센서 표시
- 수위선 없이 센서만 나열
- 터널·천단침하 요소
