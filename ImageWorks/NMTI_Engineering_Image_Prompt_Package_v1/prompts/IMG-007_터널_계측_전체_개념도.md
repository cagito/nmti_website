# IMG-007 - 터널 계측 전체 개념도

> **AI (docs/36):** [§4.2](../../docs/36-AI-이미지-생성-엔지니어링-프롬프트-가이드.md) · Prefix §2.1


## 1. 이미지 목적
NATM 터널 단면과 주요 계측 항목 표시

## 2. 사용 위치
- NMTI 홈페이지 신규 페이지: 건설계측 기술자료
- TreeView 항목: 분야별/터널
- 본문 설명 이미지, 기술자료 삽도, 제안서 재사용 가능 이미지

## 3. 제작 이미지 설명
터널 계측 전체 개념도를 기술 매뉴얼에 들어갈 수 있는 정밀한 엔지니어링 삽도로 제작한다.
현장 실사 사진이 아니라, 사실적인 CAD 기반 단면도/개념도/시스템 구성도 스타일로 표현한다.
불필요한 배경 장식 없이 핵심 계측 대상, 센서 위치, 데이터 흐름, 측정 방향이 명확해야 한다.

## 실행 규칙

> **book/image-knowledge:** `docs/image-knowledge/01-터널-내공변위-측정.md` · §5·§6

**반드시 그릴 요소:**
- 터널 단면 윤곽(전단면 또는 반단면 중 하나로 목적 고정).
- **측정 단면 위치**(종방향, 간격 20 m / 10 m 구간 구분 가능 시).
- **측선** 및 **측점**(그림 4.2-1에 따른 3·4·6측선 예 — 단면 유형 명시).
- **막장면(굴진면)** 과 측점의 근접 관계.
- 동일 단면에서 **천단침하 측점**이 있음을 보조 표기(천단 상세는 별도 Figure).
- **지반조건에 따른 간격 조정** 가능 구간(좁힘/생략) — 일반화 시 `현장 조건에 따라 상이` 주석.
- 라벨: `내공변위`, `측선`, `측점`, `측정단면`, `막장면(굴진면)` 등.
- **지표·지중침하**는 토피·지상시설물 조건에 따라 **가감·생략 가능** — 내공변위 Figure에서 침하를 **동등 hero**로 혼합 금지 — *KDS-11-10-15 §4.1.5*

**절대 금지:**
- 천단침하계·지중경사계·지하수위계를 내공변위 측점으로 오인되게 동일 위치에 배치.
- 지표침하핀·프리즘을 터널 내공 측점처럼 표현.
- 측점을 순환 체인으로 잘못 연결(폐곡선 측선) — PDF는 단면 내 측선 배치 예를 제시.
- 노반·궤도·차량·보행자.
- 서버·대시보드·모바일 앱(시스템 Figure 혼합).
- 360° 전 둘레 등간격 측점(상부 아치 대표 측선 목적과 혼동).
- 현장 특정 프로젝트 도면의 측점 번호를 일반 규칙처럼 고정 표기.
- **철도구조물 직하부** 통과 구간의 지표침하·발파진동을 **내공 단면만**으로 대표 — 종단 callout 또는 **별도 Figure** — *KDS-11-10-15 §4.1.5.1④*

## 4. 필수 포함 요소
- 짧고 명확한 한글 라벨: 천단침하, 내공변위(내공변위계), 지중변위, 지표침하, 록볼트, 숏크리트, 데이터로거
- **내공변위**는 테이프·LVDT 2점이 아니라 **상부 아치 Kit 체인**으로 구분 표기 (상세는 IMG-008)
- 화살표, 치수선, Callout 박스
- 계측 대상 구조물 또는 지반 단면
- 센서 위치를 작은 아이콘 또는 실제 장비 외형에 가까운 단순 도형으로 표시
- 관리기준 또는 경보 관련 이미지인 경우 색상 단계: 정상 Green, 주의 Orange, 경고 Red
- 현장명이 필요한 경우: "00 복선전철 제00공구"만 사용

## 5. 구도 지시
- 16:9 웹 배너형 또는 본문 삽도형
- 좌측에는 구조물/지반/센서 배치
- 우측에는 핵심 원리 또는 데이터 흐름 요약
- 단면도는 평면적인 기술 도면으로 표현
- 그래프가 필요한 경우 축, 기준선, 범례를 포함하되 숫자는 예시값으로 단순화

## 6. 스타일 지시
- 흰색 또는 아주 밝은 회색 배경
- Deep Navy, Cool Gray, Teal 중심
- 선 두께는 일정하게 유지
- 매뉴얼 Figure 스타일
- 과도한 입체감 금지
- 실제 장비 매뉴얼 삽도처럼 신뢰감 있게 제작

## 7. 텍스트 규칙
- 긴 문장 금지
- 라벨은 2~8자 중심
- 예시 문구 사용 가능:
  - "센서"
  - "데이터로거"
  - "관리기준"
  - "경보"
  - "변위"
  - "침하"
  - "수위"
- 실제 위치명, 실제 노선명, 실제 공구명 금지

## 8. Negative Prompt
AI풍, cartoon, cute, mascot, human, worker, face, brain, biological organ, fantasy, cyberpunk, neon, excessive gradient, dramatic 3D render, cluttered background, unreadable Korean text, distorted labels, actual city map, real station name, real project name, random numbers, messy dashboard, advertising banner style
