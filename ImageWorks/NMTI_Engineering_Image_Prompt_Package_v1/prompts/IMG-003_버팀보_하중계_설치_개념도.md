## 이미지 목적

흙막이 **버팀보(Strut) 하중계**가 **띠장–버팀보 접합부(끝단)** 에 설치되는 방식을 기술 매뉴얼 Figure 수준으로 설명한다.  
**보 정중앙 하중계는 금지** — IMG-002 v6·KDS 가시설 계측 기준과 동일.

## 필수 포함 (v2)

| 항목 | 규칙 |
|------|------|
| 단면 좌→우 | `[좌] 배면 지반 \| 흙막이 벽체·띠장 \| [우] 굴착측` |
| 버팀보 위치 | **굴착측(우)** 공간, 수평으로 양쪽 띠장 연결 |
| 하중계 위치 | **띠장–버팀보 접합부(끝단)** 만 — 양쪽 끝단 각 1개 또는 한쪽 상세 확대 |
| 축력 화살표 | **띠장/벽체 → 보 끝단** 방향 압축(축압축력) — 보 축 내부 `→ ←` 수축형 **금지** |
| 우측 패널 | 하중 전달 원리(띠장→하중계→버팀보), 데이터로거 연결 |
| 하단/우측 | 관리기준·경보 그래프(정상 Green / 주의 Orange / 경고 Red) |
