## 강제 지시문

```text
연약층 전체가 균일하게 좌우로 밀리는 것처럼 그리지 않는다.
침하계·지표침하 측점은 측방유동을 측정하지 않는다.
IPI는 연약층 하부 견고 지지층까지 근입한다.
중앙 IPI는 연직·침하 검토 — 측방유동 주역이 아니다.
측방 검토 위치 = 성토 어깨·TOE 부근.
위반 시 전면 폐기.
```

## 도면 구성

| 영역 | 내용 |
|------|------|
| **좌** | 성토 단면 · 연약층 · 견고 지지층 · IPI 3소(중앙·어깨·TOE) |
| **우** | 측방변위 프로파일 · 변위속도·성토단계·간극수압 병행 callout |
| **하** | 침하 측점(연직) vs IPI(측방) 역할 분리 |

## 금지

대칭 측방 화살표 · 침하계=측방 · 근입 생략 · 흙막이 굴착 혼동
