## 이미지 목적

**「가시설 계측 전체 개념도」** — 흙막이 가시설 계측의 대표 항목을 **굴착 단면 한 장**에 개념적으로 표시.  
센서 아이콘 나열·원격계측 흐름도가 아니라 **계측관리계획서 수준의 단면도** 스케치.

## ⛔ 선행 — 3영역 (생성 게이트)

흙막이 단면은 **먼저** 다음 3영역으로 나눈다.

```text
[배면 지반] | [흙막이 벽체] | [굴착측]
```

인접 건물 포함 시 좌→우 전체:

```text
[좌] 인접 건물 | 배면 지반(연속 토사) | 흙막이 벽체·띠장 | [우] 굴착측(버팀보·공동)
```

**3영역을 구분하지 못하면 이미지를 생성하지 않는다.**

## 절대 배치 규칙 (G-01~G-15)

| # | 규칙 |
|---|------|
| G-01 | **띠장**·**버팀보** = 반드시 **굴착측 벽체면**에만 |
| G-02 | 띠장·버팀보를 **배면 지반 속**에 넣지 않음 |
| G-03 | **버팀보 하중계** = **띠장–버팀보 접합부** 또는 **축력 전달부** (보 정중앙 금지) |
| G-04 | **어스앵커 하중계** = 반드시 **굴착측 앵커 두부** |
| G-05 | 어스앵커 하중계를 **자유장·정착장·그라우트·배면 지반** 내부에 넣지 않음 |
| G-06 | 어스앵커 = 벽체 관통 → **배면 지반** 방향 — **자유장**과 **정착장** 구분 |
| G-07 | 두부 조립 = **`반력판 → 하중계 → 앵커헤드 → 강연선`** |
| G-08 | **토압계** = 벽체 **배면·지반 접면** — 감지면 **배면 지반 → 벽체** |
| G-09 | **지하수위계** = **지표면(GL)에서 천공** — **well cap·개구부 지표면 가시** · 관측공·보호관·**스크린**·**G.W.L** (BORE-GL-01) |
| G-10 | **간극수압계** = 특정 심도 센서 — **짧은 필터** + 상·하부 **차수(seal)** |
| G-11 | 지하수위계 ≠ 간극수압계 — **동일 긴 관측공** 금지 |
| G-12 | 침하 **핀·측점** → **`지표침하 측점(침하핀)`** — **`지표침하계` 라벨 금지** |
| G-13 | **지표침하 측점(침하핀)** 은 **데이터로거에 직접 연결하지 않음** |
| G-14 | **자동 센서형 지표침하계**만 **케이블 → 현장 계측함** 연결 표현 |
| G-15 | **데이터로거 ≠ 센서** — **`현장 계측함 / 데이터로거`** 작게 표시 · **센서 범례와 분리** |
| G-16 | **자동광파기·Total Station·시준선 금지** — 인접건물=프리즘 측점만 (ATS-SUB-01 · [206](../../../docs/206-자동광파기-지하공사-전용-표현-통합-적용-계획.md)) |

## 필수 표기 (개요 수준)

| 항목 | 위치·규칙 |
|------|-----------|
| **센서형 다단식 지중경사계** | 벽체 배면 천공, **GL에서 케이싱·보호함 가시**, 케이싱+다점 센서, 안정층 근입 (BORE-GL-01) |
| **지하수위계** | G-09 — **GL 천공·well cap 가시** · 개방 관측공·스크린·G.W.L |
| **간극수압계** | G-10 — 밀폐 filter·seal (②와 이형) |
| **토압계** | G-08 — 배면 접면 |
| **버팀보 + 하중계** | G-01~03 — 굴착측, 띠장 접합부 |
| **어스앵커 + 하중계** | G-04~07 — 굴착측 두부 |
| **지표침하** | G-12~14 — **핀 측점** 또는 **자동 지표침하계** 중 택1·구분 라벨 |
| 구조물경사계·균열계 | 인접 건물 — 외벽 표면 부착 |
| 변위 타깃 프리즘 | 건물 옥상(선택) |
| **현장 계측함 / 데이터로거** | G-15 — 배면 지표, **소형 함체**, 센서 목록 **밖** |

## 우측 패널 (25%)

- **가시설 계측 항목** — 번호 없는 간략 범례 (지중·수위·지보재·지표·인접구조물)
- **데이터로거는 센서 목록과 분리** — 「현장 계측함」 별도 소제목 권장
- **원격계측 흐름도·서버·모바일·경보 스트립 금지**

## 금지

- 3영역 미구분 · 배면·굴착 공간 역전 (C1·C2)
- G-01~G-16 위반 전항
- 옥상 자동광파기 본체·「자동광파기 측정(프리즘)」 혼동 라벨
- 하단/우측 센서→서버 흐름도
- 사람·굴삭기·터널 내공
