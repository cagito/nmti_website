## 목적

교각 **경사·상대변위·절대좌표변위·기초침하**를 **구분** 표현 — 측면 단독 변위계·데이터 흐름도 **금지**.

## Figure 범위

| 포함 | 제외 |
|------|------|
| 상·하부 구조물경사계 + Δθ | 데이터 흐름도 (센서→서버) |
| 받침·신축 **상대** 변위계 | 교각 측면 단독 수평 변위계 |
| 프리즘 + 영향권 밖 ATS/GNSS | 기초 직옆 기준점 |
| 기초 지표침하계(복수) | 일반 관리기준 수치 그래프 |
| 우측 **계측 항목 요약** 6줄 | 흙막이·굴착 배경 |

## P0 (BRI-PIER 요약)

1. 측면 단독 변위계 **삭제** → 받침·신축 **상대변위**
2. 절대변위 = **프리즘 + ATS**(영향권 밖) 또는 GNSS
3. 기준점 = **안정지반** (기초 옆 금지)
4. 상·하 경사계 + **Δθ** 회전 개념
5. 기초 **지표침하계** — [SETTLE-01](../../../docs/42-지표침하핀-지표침하계-구분-및-자동계측-정책.md)
6. 그래프: **「현장별 관리기준 적용」** — mm·° 고정값 금지
