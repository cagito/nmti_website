## 이미지 목적

1단계 굴착부터 최종 굴착까지 **굴착 깊이 증가**에 따른 지보재·계측기 추가와 **계측→판정→다음 굴착** 운영 흐름을 한 장에 설명한다.

## 좌측 — 4단계 굴착 단면도 (각 미니 단면)

단면 좌→우: `[좌] 배면 \| 벽체·띠장 \| [우] 굴착측`

| 단계 | 내용 |
|------|------|
| ① 1단계 | 얕은 굴착, 지중경사계·지하수위계 |
| ② 2단계 | 1차 버팀보, **띠장 접합부 하중계** |
| ③ 3단계 | 어스앵커·추가 버팀보 |
| ④ 최종 | 최종 굴착선, 전 계측기 가동 |

- 수평변위 화살표 → **굴착측(우)**
- **버팀보 하중계 = 띠장 접합부(끝단)만** — 정중앙 금지
- 어스앵커 하중계 = 굴착측 노출 두부

## 우측 상단 — 계측 운영 루프 (본 이미지 허용)

계측 → 데이터로거 → 관리기준 비교 → 경보 판정(정상/주의/경계) → 다음 단계 굴착 → (반복)

## 우측 하단 — 누적 변위 그래프

굴착 단계별 누적 변위(mm), 정상·주의·경계 기준선

## 금지

- IMG-002와 동일한 11종 대표 단면도 (역할 중복)
- 버팀보 정중앙 하중계
- 터널·천단침하 요소
- 사람 얼굴·홍보 배너
