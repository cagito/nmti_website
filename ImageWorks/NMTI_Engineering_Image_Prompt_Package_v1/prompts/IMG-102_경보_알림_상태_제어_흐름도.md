# IMG-102 — 경보·알림 상태 (Alarm Trigger Control Flow)

## 메타

| 항목 | 값 |
|------|-----|
| ID | IMG-102 |
| 제목 | 경보·알림 상태 제어 흐름도 |
| nodeId | `instruments/modes/alarm-status` |
| 용도 | 3단계 threshold 돌파 → 경광등 · SMS/Push · ≠ SF 경고창 |

## 이미지 목적

Data chart — **3 horizontal dotted threshold lines** (Caution · Warning · Action) · plot **breaks above Action** · logic vector → **strobe siren** + **SMS/Server alert block**.

## 필수 포함

- Data chart with **Caution / Warning / Action** horizontal threshold lines (dotted)
- Escalating data plot crossing **Action** line — inflection emphasized
- Logic signal vector → **outdoor strobe siren horn** symbol
- Parallel route → **automated SMS / Push notification / Server alert** block
- Pure **engineering schematic** · white background

## 금지 (치명)

- SF movie warning popup · neon hologram · abstract red alert illustration
- brain · cyber · 3D render
- IMG-054 process-only without threshold chart detail (102 = threshold+trigger focus)

## Negative Prompt

SF warning popup, neon hologram, brain, cyber matrix, abstract red alert art, 3D render, explosion
