## 강제 지시문 (프롬프트 맨 위)

```text
센서형 다단식 지중경사계 측정 원리도 — 초기 프로파일(기준)과 현재 프로파일을 반드시 구분한다.
변위 증분 = 현재 − 초기. θ만 적분하여 누적변위라고 단정하지 않는다.
A축·B축 정·역 왕복 측정을 표시한다. 누적변위는 기준 심도부터 상대 누적이다.
기준 심도 불확실 시 절대변위 단정 금지. 누적오차·반복성 검토 주석 필수.
하나라도 위반 시 전면 폐기한다.
```

**반드시 그릴 요소:**
- **지표면** 선과 명시 라벨 `지표면`.
- **천공** 단면과 **inclinometer casing**(내부 groove 표현 가능).
- **안정 지반** 구간(하단 고정·기준 구간).

- 상부 **보호 cap / surface box / enclosure** 개념(잠금·우수 유입 방지).
- 부력 대책 중 하나: casing 내부 **suspended steel pipe/drill rod**(상부 누름만으로 고정 ❌).
- 라벨: `지중경사계`, `관측공`, `케이싱`, `그라우트`, `안정 지반`, `지표면`.
- 지중경사(수평변위) 계측은 **배면·벽체 영향권** 천공 — **벽체 표면 부착형** IPI hero 금지 — *KCS-11-10-15 §3.10.3.1①*

**절대 금지:**

- 데이터로거 함체를 천공 내부에 배치.
- 사람·차량·시공 장비 hero.
