## 이미지 목적

**공사 경계 펜스** 한쪽 · **인접 주택/아파트** 반대편 레이아웃 — 펜스 상단 또는 **독립 계측주**에 **소음 마이크(풍향막)** + **분진 흡입 노즐** + **데이터로거** · 신호선 도해.

## 필수 포함

- **Construction boundary fence** (left or right) vs **residential buildings** (opposite side)
- **Outdoor monitoring pole** — rugged steel mast
- **Sound level meter microphone** with wind screen
- **Optical dust sensor inlet** / sampling nozzle on same pole
- **Visible data logger enclosure** — cables from sensors converging (§2.1 P0)
- Leader lines · technical labels · light grey background

## 금지 (치명)

- generic air pollution graph · city skyline poster
- abstract PM2.5 infographic only
- brain · 3D marketing render · empty legend box for logger
- tunnel PPV geophone layout (IMG-097)

## Negative Prompt

cityscape poster, pollution chart only, brain, 3D render, empty logger box, geophone triax, apartment marketing aerial
