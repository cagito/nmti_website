## 이미지 목적

**케이슨·안벽·옹벽** 등 항만구조물과 **육측·해측 주변 지반**, **조위·지하수** 계측을 **횡단 단면 한 장**에 개념적으로 표시한다.  
교량(IMG-011)·댐(IMG-024)·가시설(IMG-001)과 구분되는 **해안 횡단** 도면이다.

```text
[좌] 육측(뒤채움·매립) | 항만구조물(케이슨/안벽) | [우] 해측(바다·사석마운드)
```

## 필수 표기 (개요 수준)

| 항목 | 위치·규칙 |
|------|-----------|
| 항만구조물 | 케이슨 또는 중력식 안벽 단면 — 육측 배면·해측 전면 구분 |
| 변위계 | 구조물 표면(상부·모서리) |
| 구조물경사계 | 구조물 외벽 또는 상부 |
| 반력계 | 구조물 **하부 바닥 모서리**(KDS 4.1.8.4③) — 개요 수준 1~2개 |
| 육측 지표침하계 | 육측 지표 |
| 육측 **센서형 다단식 지중경사계** | 육측 **GL 천공** · **케이싱 상단·보호함 지표면 가시** (BORE-GL-01) · 뒤채움 지반 |
| 구조물 하부 층별침하 | 기초 하부 지층 |
| 해측 **센서형 다단식 지중경사계** | 해측 **GL 천공** · **케이싱 상단 가시** (BORE-GL-01) · 지반·사석마운드 |
| 지하수위계 | 육측 **GL 천공** · **well cap 지표면 가시** (BORE-GL-01) · 관측공, G.W.L 점선 |
| 간극수압계 | 배면·해측 지반 천공(관측공과 구분) |
| 조위계 | 해안 대표 위치, **조위선** 점선(해수면) |
| 데이터로거 | 육측 지표 보호함 내 **소형 로거**(CR1000X 유형, 선택) |

## 우측 패널 (25%)

제목 **「항만·호안 계측 항목」** — 번호 없는 간략 리스트:

- 구조물 변위·경사·반력
- 육측·해측 지반 침하·지중변위
- 조위·지하수·간극수압

**원격계측 흐름도·서버·모바일 스트립 금지**

## 금지

- 교량 상부구조·교각(IMG-011 혼입)
- 댐 누수관·제체(IMG-024 혼입)
- 흙막이 굴착·버팀보(IMG-001·002 혼입)
- 터널·내공변위
- 하단 센서→서버→대시보드 흐름도
- 사람·굴삭기·실사 사진
