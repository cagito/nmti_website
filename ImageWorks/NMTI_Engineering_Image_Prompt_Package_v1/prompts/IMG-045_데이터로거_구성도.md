## 이미지 목적

현장 **정적 데이터로거**의 외형·입출력 구성을 기술 매뉴얼 Figure로 설명한다.  
중심 장비는 **소형 측정·제어 데이터로거**(CR1000X와 유사한 배선판형 인클로저)이다.

## 필수 포함

### 중앙 — 데이터로거 정면 (주 피사체)

- 밝은 회색 직사각 인클로저, 앞면 **탈착식 나사 단자 블록** 2~3열
- 채널·H/L·GND·`12V POWER`(녹색 포트) 표기
- 라벨: **데이터로거**
- **브랜드 로고·실제 모델명 인쇄 금지**

### 좌측 — 센서 입력

- 변위·하중·수위·경사 등 **센서 케이블**이 단자 블록으로 수렴
- 라벨: 계측 센서, 센서 입력, 아날로그·디지털 채널(간략)

### 우측 — 전원·통신·저장

- **12V 전원**(태양광·배터리 또는 AC 어댑터 화살표)
- **통신** 포트(USB/Ethernet/RS-485 또는 LTE 모뎀 점선)
- **저장**: 내장 메모리·SD 카드 슬롯
- 라벨: 전원, 통신, 저장, 계측 데이터

### 하단(선택)

- 센서 → 데이터로거 → (통신) → 서버 방향 **단순 화살표** — 흙막이 단면·대시보드 UI 금지

## 구도

- 16:9, 1920×1080, 흰색 배경
- 중앙 50%: 로거 정면 배선판 상세
- 좌 25%: 센서·케이블
- 우 25%: 전원·통신·저장 블록
- CAD 매뉴얼 Figure, Deep Navy / Cool Gray / Teal

## 금지

- 지반·흙막이 **단면도를 주 화면**으로 사용 (IMG-002 역할과 혼동)
- 정사각 접속함, PC 본체, 서버 랙, NEMA 대형 캐비닛
- 사람·홍보 대시보드·실제 현장명
