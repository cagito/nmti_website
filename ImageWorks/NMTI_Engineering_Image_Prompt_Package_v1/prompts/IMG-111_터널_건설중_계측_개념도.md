# IMG-111 — 터널 건설중 계측 개념도 (v2 — TUN-CONST-01)

> **AI (docs/36):** [§4.3](../../docs/36-AI-이미지-생성-엔지니어링-프롬프트-가이드.md) · [153 건설중](../../docs/153-계측문서-용어-정본-건설중-계측.md) · Prefix §2.1

**node:** `fields/tunnel/construction-phase` **hero**  
**목적:** NATM 터널 **건설중** 내공·천단·지표 변위 **통합** (운영기·완성 단면 아님)

## 1. 이미지 목적

굴착·지보 진행 중 터널에서 **내공변위·천단침하·지표침하** 위치를 **한 개요도**에 콜아웃. 단일 센서 상세(IMG-008/061/010) 아님.

## 2. 사용 위치

- `fields/tunnel/construction-phase` hero
- 하위: `convergence`→IMG-008 · `crown-settlement`→IMG-061 · `surface-subsidence`→IMG-010

## 3. 필수 포함

- **NATM 단면** — 막장·미지보 구간 · **굴착 진행 방향**
- **숏크리트·강지보** 1종 이상
- **내공변위** 측선·측점 (터널 **내면**)
- **천단침하** 측점 (천단·아치 — **내공과 측선 분리**)
- **지표침하** 측점·계측선 (**지표면**)
- **건설중** 제목·패널 · **데이터로거 함체**
- (철도 직하부) 지표침하·발파 — **작은 종단 callout**만

## 4. 금지 (TUN-CONST-01)

- **`건설기간`** · 완성 라이닝만 · 막장 없음
- 내공=천단 **동일 측선** · 지표침하 **내부** · 내공변위 **지표면**
- 흙막이·앵커 LC · 360° 측점 · 서버 UI
