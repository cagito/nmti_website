## 1. 이미지 목적

교량 **신축이음부 양측 구조물 사이 상대 신축량**을 **신축이음계**로 계측하는 설치 개념도.  
교량 전체 종·횡·수직 3축 변위계가 **아님**.

## 2. 사용 위치

- **노드:** `fields/bridge/expansion-joint` (신축이음량)
- NMTI 기술자료 교량 · 신축이음계 hero

## 3. 제작 이미지 설명

한국 토목 유지관리·준공보고서 스타일 **2D 기술 도면**.  
평면도와 단면도를 함께 구성. 핑거형(톱니 맞물림) **신축이음 장치** 실형 반영.
