## 1. 이미지 목적

교량 **신축이음부 양측 구조물 사이 상대 신축량**을 **신축이음계**로 계측하는 설치 개념도.  
교량 전체 종·횡·수직 3축 변위계가 **아님**.

## 2. 사용 위치

- **노드:** `fields/bridge/expansion-joint` (신축이음량)
- NMTI 기술자료 교량 · 신축이음계 hero

## 3. 제작 이미지 설명

한국 토목 유지관리·준공보고서 스타일 **2D 기술 도면**.  
평면도와 단면도를 함께 구성. 핑거형(톱니 맞물림) **신축이음 장치** 실형 반영.

## 5. 절대 금지 (BRI-EJ)

1. X/Y/Z 3축 · 종방향·횡방향·개폐량 병렬 주계측
2. 「교량 종·횡변위 계측도」 제목
3. 상판 위 부유 센서 (지그·브라켓 없음)
4. 단순 직선 갭만 있는 이음 (핑거형 미표현)
5. 임의 관리기준 mm 수치
6. 대형 데이터 흐름도
7. 사람·AI 장식
8. **와이어 없는** LVDT 단축형·부유 신축이음계 (BRI-EJ-13)
