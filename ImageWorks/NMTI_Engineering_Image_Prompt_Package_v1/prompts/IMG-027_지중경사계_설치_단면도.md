## 필수 포함

- **센서형 다단식 지중경사계** — 연속 **케이싱** + **내부 다점 센서 노드**(심도별) + 케이블
- **보링공** / **GL에서 천공** (지표~안정층) — **케이싱 상단·지표 보호함 지표면 가시** (BORE-GL-01 · [111](../../../docs/111-지중경사계-지하수위계-지표면-천공-가시-표준.md))
- **그라우트** — 케이싱 **외주부**·보링공 사이 annulus (**내부 센서 구간 전체 충전 금지**)
- **안정층** — Base = **기준 심도 확보** (≠ 물리 절대고정) · 근입 = **현장 설계** ([51 §5](../../../docs/51-계측-도면-검수-공통-원칙.md))
- **활동면** 또는 **변위 집중 심도** — 점선·콜아웃 (**Base 아래**에 Base 위치)
- 지표 **보호함**·초기치 측정 위치
- **수평변위** 화살표 — **굴착·활동·사면 외향** (수직 침하 X)
- **설치 맥락** 1개+: 흙막이 **배면** 천공+벽체 / **사면** 절토 / **굴착** 저면
