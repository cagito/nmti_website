# IMG-100 — 건축공사 계측 전체 개념도 (v5 — BLD-H-01 · DISP-ATS-01)

> **AI (docs/36):** [§4.4](../../docs/36-AI-이미지-생성-엔지니어링-프롬프트-가이드.md) · [127 DISP-ATS-01](../../docs/127-변위계측-자동광파기-남용방지-및-와이어식-우선-표준.md) · Prefix §2.1

**node:** `fields/building` **hero**  
**목적:** KCS 3.9 처짐·축소·균열·경사·응력 **통합 콜아웃** (단일 센서 상세 아님)

## 1. 이미지 목적

건축공사 **KCS 3.9** 계측 항목을 **한 건물 개요도**에서 **번호 콜아웃**으로 통합 설명.  
**LVDT·와이어식·변형률·균열·경사** = 주계측 · **ATS** = 선택·보조 inset 또는 생략.

## 2. 사용 위치

- `fields/building` hero (IMG-022·성토·침하판 Figure **대체** — BLD-H-01)
- 하위 노드: `deflection`→IMG-099 · `adjacent-building`→IMG-101

## 3. 필수 포함

- **RC/골조 건물** 입면·단면 + **지표면** (P0-1: **1층 바닥·출입문 = 지표면**)
- **≥3종** 콜아웃 (라벨 분리):
  1. **처짐·변위** — **LVDT** 또는 **와이어식 변위계** (보·슬래브)
  2. **기둥 축소** — **변형률계** + **온도계**
  3. **균열** — **균열계** (일상 신축 회피 각주)
  4. (선택) **경사** — **구조물경사계**
  5. (선택) **부등침하** — **지표침하계**
- 하단 각주: **KCS 3.9** · **설계도서·계측관리계획**

## 4. 금지 (BLD-H-01 · DISP-ATS-01)

- **연약지반·성토·침하판** 단면 · **흙막이 3분할** · **터널·교량** 혼입
- **ATS·프리즘** 주 hero (작은 「선택·보조」만)
- **침하핀(측량)** = **지표침하계** · **진동현식·VW** · 브랜드 · **서버 UI**

## 최종 생성 프롬프트

그림의 목적은 KCS 3.9 건축공사 계측(처짐·기둥축소·균열·경사·부등침하)을 한 건물 개요도에서 번호 콜아웃으로 통합 설명하는 것이다.
