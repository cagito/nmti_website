## 4. 필수 포함 요소

### 지질·단면

- 매립층·풍화토·연암(기반암) 층 구분
- **잠재 활동면(추정)** 점선
- **G.W.L** (지하수위선) 파란 점선
- 사면 상단·하단(밸리) 방향 표기

### 계측기 (번호·라벨)

1. **센서형 다단식 지중경사계** — **GL에서 천공** · **케이싱 상단·보호함 지표면 가시** (BORE-GL-01) · 사면 관통, 케이싱+다점 센서, 하단 기반암 깊은 근입, 수평변위 → 화살표
2. **지표침하계** / 층별침하 — 필요 시
3. **지하수위계** — **GL에서 천공** · **well cap 지표면 가시** (BORE-GL-01) · 관측공·보호관·**개방 수면**·필터/유입 표현
4. **간극수압계** — 토층 내 필터부·케이블, G.W.L **아래**
5. **강우량계** — 사면 상부 개활지
6. **프리즘 타깃** — 사면 표면(대표 측점), **광파기 아님** — **(선택·공사중)**
7. **(선택·보조) 자동광파기** — 사면 **밖** 안정 부지 — **주계측 아님** · [127 DISP-ATS-01](../../docs/127-변위계측-자동광파기-남용방지-및-와이어식-우선-표준.md)
8. **(선택) 시준선** — ATS → 프리즘
9. **데이터로거** — 방수함

### 우측 패널 (선택)

- 범례: 구조물 변위·지중경사·침하·수위·수압·강우
- 짧은 캡션: 「기준점은 부동, 측점은 사면」
