## 강제 지시문 (프롬프트 맨 위 — 필수)

```text
이 그림의 계측 목적은 단 하나다: 필댐/석괴댐 안전관리 계측 체계(저수위→응답→해석→판단→조치)를 한 장에 표현한다.
댐 단면은 필댐 또는 석괴댐만 그린다. 콘크리트 중력식 댐·Plumb line hero를 넣지 않는다.
해석 침윤선은 파란 점선으로 저수위·piezometric head·배수조건과 정합하는 해석 곡선이다. filter tip 고도에 맞추거나 tip 수두를 침윤선으로 연결하지 않는다.
간극수압계는 filter tip+seal로 그린다. 개방 standpipe·전관 수면으로 그리지 않는다.
우측에 6항목(저수위·간극수압·누수탁도·침하·수평변위·보조) 카드와 하단 7단계 데이터 흐름을 반드시 넣는다.
침하 그래프는 아래(음수)일수록 위험도가 높게 표시한다.
틀릴 것 같으면 그리지 말고 별도 그림으로 분리한다.
```

## 실행 규칙

> **book/image-knowledge:** `docs/image-knowledge/15-댐-계측-배치.md` · §5·§6

**반드시 그릴 요소:**
- **댐 단면** + **저수·기초** 맥락.
- **≥2종** 계측기(유형에 맞게) — 역할 라벨 분리.
- (piezo) **Sand Cell·천공·밀봉** 개념.
- (침투) **V-노치** 또는 측정실 콜아웃.
- **「설계도서 위치」** 각주.
- 라벨: `간극수압계`, `지하수위계`, `경사계`, `층별침하계`, `침투량`, `G.W.L`, `댐체`, `기초`.
- (침윤) **filter tip** 위치 + **piezometric head** 수두점 — **침윤선(해석 곡선)** 은 저수위·수두와 **정합** ([152](../152-IMG-024-댐-피에조-침윤선-정합성-수정계획.md)).

**절대 금지:**
- **G.W.L = 간극수압** 단일 곡선.

- **침하핀(측량)** = **층별침하계(매설)** 동일 표현.
- **filter tip 고도에 침윤선 맞추기** · tip 수두선 = 침윤선 ([152](../152-IMG-024-댐-피에조-침윤선-정합성-수정계획.md)).
