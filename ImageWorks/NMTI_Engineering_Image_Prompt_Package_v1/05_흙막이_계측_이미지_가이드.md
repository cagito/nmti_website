**AI·도면 생성 [조건/제약사항] (물리적 현실성 4원칙):**  
[docs/14-흙막이-굴착-계측-개념도-AI-생성-가이드라인.md §2](../../docs/14-흙막이-굴착-계측-개념도-AI-생성-가이드라인.md) — 프롬프트에 **그대로 포함**

```text
흙막이 계측 ≠ 센서 아이콘 나열 ≠ 원격계측 흐름도
흙막이 계측 = [좌]인접건물|배면|벽체|[우]굴착 단면 + 11종 계측기 실제 위치

P0-4: One Figure = One 계측 목적
P0-5: 배면 지반 | 벽체 | 굴착측 3분할 — 굴착 공동은 굴착측만
P0-6: 띠장·버팀보·앵커 두부 = 굴착측 벽면만 (배면 지반 속 금지)
```

용어: `book/KDS-KCS_용어기준.md`

### BORE-GL-01 (P0 — 지중경사계·지하수위계)
