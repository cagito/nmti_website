## 0. 레이아웃 (16:9, 1920×1080)

```
┌────────────────────────────────────────────────────┬──────────┐
│  좌 75% — 굴착 단면                                   │ 우 25%   │
│  [좌]인접건물|배면토사|흙막이벽체·띠장|[우]굴착공동      │ 범례①~⑪ │
│  하단: 여백만 (흐름도 금지)                            │          │
└────────────────────────────────────────────────────┴──────────┘
```

**좌→우 필수:** 인접 건물(맨좌) → 배면 지반(연속 토사) → 벽체·띠장 → 굴착측(맨우, 유일한 공동)

## 1. C0 — 지표면·건물 (치명)

| 항목 | 검수 |
|------|------|
| 「지표면」 라벨 | ☐ |
| 1층·출입구 = 지표면 높이 | ☐ |
| 지층(매립·모래·풍화암) = 지표면 **아래만** | ☐ |
| 건물이 토사층 **내부에 묻히지 않음** | ☐ |
| 배면(벽↔건물) = **연속 토사** (빈 공동 없음) | ☐ |

## 2. SOE·앵커·버팀보 (치명 P1~P3)

| 항목 | 검수 |
|------|------|
| **상부 앵커 + 하부 버팀보** 혼합형 라벨 | ☐ |
| 강연선·T → **배면(좌)** 방향 | ☐ |
| 앵커 두부(하중계) = **굴착측(우) 노출** | ☐ |
| 버팀보 = 굴착측(우)만, 띠장 접합부 하중계 | ☐ |
| G.W.L > 굴착저 시 **차수/근입·배수** 콜아웃 | ☐ |
| 굴착 공동이 건물 기초 아래 **없음** | ☐ |

### 앵커 두부 인셋 (권장, 하단 우)

지지링·반력판 → 하부플레이트 → 원통형 하중계 → 상부플레이트 → 앵커헤드 → 강연선(좌측 배면)

## 3. 계측기 11종 (단면 내 + 범례)

| # | 라벨 (전칭) | 위치 |
|---|-------------|------|
| ① | 센서형 다단식 지중경사계 | 수직 케이싱, 다점 센서, 안정층 근입 |
| ② | 지하수위계 | 개방 관측공 (② ≠ ③) |
| ③ | 간극수압계 | 필터·그라우트 밀폐 천공 |
| ④ | 토압계 | 벽체 배면, 토압→벽체 화살표 |
| ⑤ | 버팀보 하중계 | 띠장–버팀보 **끝단** |
| ⑥ | 어스앵커 하중계 | 굴착측 노출 두부 |
| ⑦ | **지표침하 측점** / **지표침하핀** | 배면 지표면 측점 (≠ 지표침하계) |
| ⑧ | 구조물경사계 | 건물 1층 외벽 표면 부착 |
| ⑨ | 균열계 | 건물 균열 수직 교차 |
| ⑩ | 변형률계 | 띠장·버팀보 |
| ⑪ | 데이터로거 | 배면 지표 방수함 (LCD·단자) |

- G.W.L 수평 점선
- 수평변위 화살표 → **굴착측(우)**
- 옥상: **변위 타깃 프리즘** (광파기 본체 금지)
- 자동광파기: 영향권 밖 부동점 + 시준선
