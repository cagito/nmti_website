## 2. 좌측 단면 — 필수 요소

### 2.1 측점 P1~P5 (상부 아치만)

| 측점 | 위치 | 각도 가이드 (12시=0°) |
|------|------|------------------------|
| P1 | 천단 | 90° |
| P2 | 좌 상부 | 120°~135° |
| P3 | 좌 스프링라인 **위** | 150°~160° (Envelope **외측**) |
| P4 | 우 상부 | 45°~60° |
| P5 | 우 스프링라인 **위** | 20°~30° |

- **P6/P7 하부 스프링라인 배치 금지**
- Kit = Bracket + 길이변위센서 + 경사센서 (라이닝 밀착)
- Extension Tube로 **아치 따라** 연속 체인 (인버트 관통 금지)

### 2.2 노반·건축한계

- 평면 **노반 (도로·철도)** — 라벨 「내공변위 미계측 구간」
- 주황 점선 **건축한계 Envelope** — 통행·궤도 점유 공간
- 캡션: 「건축한계 — 통행·궤도 구간 (계측기 배치 금지)」
- Envelope **내부**에 Kit·튜브·MUX **없음**

### 2.3 형상·신호

- 점선 = **초기 형상** (아치만)
- Teal 실선 = **현재 형상** (아치만)
- 케이블: 체인 종단 → MUX → **데이터로거** (중앙 허브 별선 금지)

## 3. 우측 패널 — 필수 채움 (빈 패널 = 불합격)

### 3.1 Kit 구성 (상단)

박스 3분할 또는 스택:

1. **길이변위센서** — 축방향 변형 측정
2. **경사센서** — 접선/경사
3. **Bracket** — 라이닝 정착

### 3.2 좌표 (중단)

- **ΔX (반경방향)** — 화살표 + 단위 mm
- **ΔY (접선방향)** — 화살표 + 단위 mm
- 간단 수식 또는 「P_i 기준 좌표 변화」

### 3.3 전단면 변위 프로파일 (하단)

- 제목: 「전단면 변위 프로파일 (상부 아치 P1~P5)」
- **반원형** 극좌표 또는 U자 프로파일 — **완전 원(360°) 금지**
- P1~P5 라벨 + 예시 변위 곡선 1종 (점선 초기 / 실선 현재)
