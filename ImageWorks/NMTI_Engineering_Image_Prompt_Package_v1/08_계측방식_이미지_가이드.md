# 계측 방식 이미지 가이드 (IMG-070~075)

> **계획:** [docs/IMG-070_MEASUREMENT_MODES_IMAGE_PLAN.md](../../docs/IMG-070_MEASUREMENT_MODES_IMAGE_PLAN.md)

## 1. 목적

`instruments/modes/*` 각 노드에 **계측 방식 정의**에 맞는 전용 Figure를 부여한다. 장비(IMG-045)·플랫폼(IMG-058) Figure와 역할을 분리한다.

## 3. 운영 조건 표현 (MOD-01 · DP-11) — **진화 사다리 금지**

> **정본:** [docs/59 §1](../../docs/59-계측-운영-모드-구조-환경-AI-표현-표준.md) · [51 DP-11](../../docs/51-계측-도면-검수-공통-원칙.md)

**⛔ 금지:** `수동 → 자동 → 원격 → 스마트 → AI` 를 **유일한 상하 등급·진화 화살표**로만 표현.

**필수:** 각 방식 블록에 **`적용 조건` · `장점` · `한계` · `검증 필요성`** 콜아웃.

| 방식 | 역할 |
|------|------|
| 수동 | 기준·검측·정밀 점검 |
| 자동 | 반복·상시 취득 |
| 원격 | 통신 수집 |
| 스마트 | 경보·조치이력·대시보드 |
| AI | **HITL 보조** · 이상탕후 (판정 확정 금지) |

### 3.1 요소 혼입 금지 (하위 Figure에 상위 전용 요소)

| 금지 | 예 |
|------|-----|
| 수동에 서버·로거 | IMG-070 |
| 자동에 웹 대시보드 | IMG-071 |
| 원격에 AI·뇌 | IMG-072 |
| AI에 「기준 대체」 문구 | IMG-074 — **보조**만 |
