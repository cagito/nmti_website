## 1. PDF에서 추출한 핵심 개념 (이미지에 반영)

### 1.1 시스템 개요

GNSS 기반 변위·침하 계측은 **기준국 + 이동국(복수 측점) + 중앙 서버 + 무선 통신**으로 구성된다.

```text
[기준국] ──RTK/차분──> [이동국 #1 · #2 · #3 …] ──LTE/무선──> [중앙 서버]
     │                        │                              │
  안정 지반              계측 대상(사면·댐·교량·지반)      수집·분석·경보
  GNSS 안테나            GNSS 안테나·수신기
```

### 1.2 현장 구성요소 (PDF p.2 도식 기준)

| 구성 | 한글 라벨 | 표현 요지 |
|------|-----------|-----------|
| 기준국 | **기준국** | 변형 영향권 **밖** 안정 지반·암반. 고정형 GNSS 안테나·수신기 |
| 이동국 | **이동국** · 측점 #1~#n | 사면·제체·교량·연약지반 등 **거동 대상** 위 안테나 |
| 안테나 | **GNSS 안테나** | 기준국·이동국 각각 — 광파기·프리즘·일반 GPS 핸드헬드와 구분 |
| 수신·전송 | **GNSS 수신기** · **데이터로거**(선택) | 안테나 하부 또는 방수함; LTE/무선 모뎀 연결 표현 가능 |
| 전원 | **태양광** · **배터리**(선택) | 원격·무인 계측 현장 — PDF 특징 반영 |
| 통신 | **LTE** · **무선 통신** | 현장 → 서버 데이터 전송 화살표 |
| 서버 | **중앙 서버** · **데이터 수집** | 원격 모니터링·분석 (운영자 PC는 아이콘 수준) |
| 해석 | **실시간 계측** · **3D 변위** · **잔류침하량**(선택) | ΔE·ΔN·ΔU 또는 ΔX·ΔY·ΔZ 벡터 |

### 1.3 측정 원리 (개념도 우측 패널)

- **RTK / 차분 GNSS:** 기준국 좌표 고정 → 이동국 상대 변위 산출
- **다중 위성:** GPS · GLONASS · BeiDou · Galileo · QZSS (위성 궤도·신호선은 **도식 수준**)
- **정밀도 구분:** 후처리·실시간 정밀도는 별도 캡션 (PDF: 후처리 mm급·실시간 cm급 예시 — 숫자는 예시값으로 단순화)
- **KCS 연계:** 지표침하 계측기 **측정오차 ±1.0 mm 이내** (KCS 11 10 15 지반계측) — 침하 중심 Figure에 각주 가능

### 1.4 응용 분야 (배경 단면 선택)

PDF 및 건설계측 맥락:

- 연약지반·항만·공항 **침하**
- **사면**·**댐**·**교량** 구조물 **3D 변위**
- **지반조사**·**구조물 계측** 장기 모니터링

## 3. 구도 (16:9)

```text
[좌 65%]  단면 또는 평면도 — 계측 대상(사면/댐/교량 중 1) + 기준국(부동) + 이동국 측점
[우 35%]  시스템 블록도 — 안테나→수신기→LTE→중앙 서버 + RTK 차분 화살표
```

- **좌:** 물리적 설치 위치 (기준국은 영향권 밖)
- **우:** PDF p.2와 유사한 **데이터·통신 흐름** (제조사 제품 사진 금지)

## 4. 필수 라벨 (KDS/KCS)

- 기준국, 이동국, GNSS 안테나, GNSS 수신기

- 3D 변위, ΔX·ΔY·ΔZ (또는 동·북·수직)
- RTK, 실시간 계측 (선택)

## 5. 치명적 금지 (REGENERATE 사유)

```text
- 기준국을 사면·구조물 변형 구간에 배치
- 이동국을 부동점·기준국 위치에 배치 (역할 혼동)
- 자동광파기·프리즘·레이저스캐너·일반 스마트폰 GPS로 표현
- 기준국 없이 이동국만 나열 (차분·RTK 원리 누락)
- 제조사 로고·EGM-200 등 상표·가격·연락처·실제 회사 URL
- 위성·안테나 없이 "GNSS" 텍스트만 있는 아이콘 콜라주
- 사람·얼굴·실제 현장 사진 합성
```
