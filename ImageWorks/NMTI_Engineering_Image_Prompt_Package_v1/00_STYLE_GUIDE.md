## VW·진동현식·측정 방식 (METHOD-01 · P0)

**Figure 라벨·제목·범례·프롬프트**에 **「진동현식」「VW」「vibrating wire」** 및 **특정 센서 인터페이스·측정 방식**을 **넣지 않는다.**

- 도면은 **계측기 종류**(간극수압계·변형률계·신축이음계 등)·**설치 위치**·**측정 물리량**만 표현
- 진동현식(VW) 인터페이스는 **고가·현장 채택 감소** — OEM·방식이 다양하므로 Figure에서 **방식 단정 금지**
- 정본: [TECHNICAL P0-10](./TECHNICAL_IMAGE_STANDARD.md) · [IMAGE_REJECT M-01](./IMAGE_REJECT_CHECKLIST.md)

## 터널 내공변위 (IMG-008) 전용

**터널 전단면 내공변위** — 제목·라벨에 진동현식·VW **금지** (본 Figure 센서 유형과 무관하게 METHOD-01 적용).

**⛔ 타사 제품·브랜드(ACE, ACE-TCS, Campbell 등)는 이미지·웹·프롬프트에 절대 넣지 않는다.**

**🚨 치명 — 하부(도로·철도) 간섭 금지**  
터널 바닥은 **노반·궤도**가 점유한다. **360° 원형 폐합**·invert까지의 Kit/튜브/프로파일은 **설치 불가·현장 간섭** — Figure에 그리지 않는다.  
→ **상반·측벽 아치형(U자형)** 5~7지점만. 「노반 (도로·철도)」「내공변위 미계측 구간」 필수.

**표현해야 할 것**
- **상부 아치** 라이닝에 연속 배치된 센서 Kit (P1~P5 등)
- 길이변위센서 + 경사센서 + Bracket + Extension Tube (아치를 따라 체인)
- **하단 노반(도로·철도)** — 평면, **내공변위 미계측** 라벨
- 초기 형상(점선) vs 현재 형상(Teal) — **아치 구간만**
- X/Y 좌표 변위, 상대·절대좌표 변위량, 온도 보정

**표현하지 말 것**
- **완전 원형** 단면 + invert·노반까지 360° 센서·튜브·변위 프로파일 (**도로·철도 간섭 — 치명**)
- **ACE, ACE-TCS** 등 제3자 상표·제품명
- 테이프 익스텐소미터, 변위핀 2점 거리 측정, 천단침하 중심, 터널 중앙 부유 센서

## Cursor 배치 생성 QA (docs/36 §6.1)

수십 장 연속 생성 시 AI가 일반 일러스트로 회귀하기 쉽다. **출력 1장마다** 확인:

1. **지반·구조 배치** — 가시설(좌=배면→우=굴착) · 터널(지하→지상 침하) · 댐(좌=상류→우=하류) · 항만(좌=매립→우=바다) · 철도(노반 단면)
2. **센서 부착** — 변형률=용접 · 하중계=관통 도넛 · 경사계=L브래킷 · MPBX=다점 강봉+앵커(≠ inclinometer)
3. **네거티브** — 사람·스마트폰·뇌·SF 빛무리 없음
4. **라인 아트** — 사실적 흙·바위 텍스처·그라데이션 최소 · CAD blueprint

부분 오류: docs/36 **§5.1 Feedback Loop**
