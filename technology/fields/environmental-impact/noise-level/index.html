소음

환경·민원 소음의 측정 목적, 적용 현장, 설치, 데이터 해석, 유지관리 및 관련 센서를 정리한 기술 자료입니다.

1. 계측 개요

  • 목적소음 계측은 공사 경계에서 소음계(Class 1)로 Leq·Lmax·주간/야간 dB를 연속 기록합니다. 굴착·항타·절단·발파·야간 작업 등 공종별 이벤트와 시간 태그를 연동합니다.
  • 적용 범위민감 수용체(주택·학교) 방향 측정주 배치가 핵심이며, 풍속·강우 보정과 방풍막 설치가 데이터 품질을 좌우합니다.
  • 주요 구성소음 계측은 민감 수용체 방향 경계 측정주에서 Leq·Lmax를 연속 기록합니다. 공종 일지·야간 기준과 시간 동기화가 필수입니다.
  • 운영 포인트계측 데이터는 계측책임자가 검토하는 계측관리계획서의 계측 빈도·설계예상변위·최대허용변위와 연계하여 운영합니다. 환경·민원 현장에서는 추세·변화속도·인접 센서의 동시 변화를 우선 검토하며, 현장별 상세 시방·허용값은 설계도서와 발주처 기준을 따릅니다.

2. 주요 계측 항목

  • 연속 dB1분·15분 Leq 기록
  • 이벤트공종·장비별 소음 peak를 식별합니다.
  • 민원민원 시간대 데이터로 원인 분석을 지원합니다.

3. 대표 단면·원리

  • 측정 대상IEC 61672 Class 1 마이크는 A-weighting·Fast/Slow time constant로 dB를 산출합니다.
  • 측정 원리경계 펜스·독립 주에 설치하고 데이터로거로 원격 전송합니다.

4. 설치·운영

  • 민감 수용체 방향 대표 측점 선정
  • 마이크 · 풍향막을 규격 높이에 설치
  • 방수 로거 · LTE를 주기에 연결
  • 주간/야간 · 휴일 기준 설정
  • 공종 일지와 시간 동기화
  • 착공 전: 배경 소음 — 기준선
  • 주간 공사: Leq · Lmax — 법규 · 계약
  • 야간 · 휴일: 강화 기준 — 작업 중지
  • 이벤트: 공종 태그 — 장비 · 공법
  • 민원: 시간대 상관 — 조치 · 보고

  • 기준 출처소음 허용 기준은 환경정책기본법·소음·진동 관련 법규·계약 특별조건을 따릅니다. 주간·야간·휴일 한계를 구분하고 Leq·Lmax 초과 시 공정·작업 시간을 조정합니다. KCS에 따라 현장 계측책임자가 계측관리계획서·측정·보고·기준 초과 대응을 총괄합니다.

자주 묻는 질문

진동계(041)와 같나요?

진동계는 PPV·가속도(mm/s)입니다. 소음은 dB 음압 레벨로 별도 센서·기준입니다.

터널 발파(097)?

발파는 지진파 PPV 영향권입니다. 소음은 경계 dB 연속 측정입니다.

Class 1이 필요한가요?

법규·협약·환경영향평가에 따라 등급을 정합니다. 민감 수용체 구간은 고등급·연속 기록이 일반적입니다.

야간 작업은?

주간·야간 한계가 다릅니다. 공종 일지와 Leq·Lmax를 시간 태그로 연동합니다.
