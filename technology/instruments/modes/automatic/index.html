자동 계측

자동 계측의 구성, 설치, 운영, 데이터 흐름 및 현장 유지관리를 정리한 기술 자료입니다.

개요

자동 계측은 데이터로거가 센서 신호를 설정 주기로 자동 수집·저장하는 방식입니다. KCS에서는 구간 특성에 따라 수동계측에서 자동계측으로 전환할 수 있으며, 고위험 구간에는 자동화 계측방법 적용을 검토합니다.

측정 원리

센서 → 데이터로거 → (로컬 저장) → 필요 시 통신 모듈 순으로 구성합니다. 샘플링 주기·필터·캘리브레이션이 데이터 품질을 결정합니다. 무인 현장 전원은 태양광·배터리 설계가 핵심이며, AC 가능 구간은 전원 체계 전체를 함께 검토합니다.
