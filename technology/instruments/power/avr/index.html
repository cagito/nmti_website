AVR

AVR의 구성, 설치, 운영, 데이터 흐름 및 현장 유지관리를 정리한 기술 자료입니다.

1. 시스템 개요

  • 목적AVR(Automatic Voltage Regulator, 자동전압조정기)는 디젤 발전기·약전압 배전·임시 전력처럼 입력 전압이 변동할 때 출력을 220V 정격 근처(±1~3%)로 유지하는 장치입니다. 계측 함체·통신 장비 전단에 두어 로거·모뎀 오동작을 방지합니다.

2. 구성 요소

  • 전압 안정입력 ±20~30% 변동 흡수
  • 장비 보호민감 전자부 보호
  • 품질노이즈·고조파 완화(기종별)
  • 연속성UPS 전단 조건 정비

3. 현장 구성·원리

  • 구성 흐름불안정 AC 입력(발전기·임시 배전) → AVR → (선택) 절연변압기·UPS → 계측·통신 장비. 출력 220V 정격 근처 유지가 목표입니다.
  • 측정 대상서보·탭체인저 또는 고체소자 방식으로 출력 전압을 제어합니다. 고정밀 계측 현장에서는 AVR → 절연변압기 → UPS 3단 구성으로 접지 루프·공통모드 노이즈를 줄이기도 합니다.

4. 설치·운영

  • 입력 변동 범위 · 부하 kVA 산정
  • AVR 정격 · 응답 속도 선정
  • 접지 · 단락 용량 · 배선
  • 출력 측 UPS · 정류 연결
  • 전압 이력 모니터링(선택)

자주 묻는 질문

UPS와 차이?

AVR은 전압 안정화, UPS는 정전 시 무정전 백업. 병행 사용이 일반적입니다.

태양광 DC에 AVR?

AVR은 AC용입니다. DC는 충전제어기·DC-DC regulator가 해당 역할입니다.

계측 함체에 직결?

AVR·UPS 후단 DC를 로거·통신에 분배합니다. Surge·접지를 함께 설계합니다.

발전기 구간은?

입력 전압 변동이 큰 경우 AVR을 전단에 두는 것이 일반적입니다.
