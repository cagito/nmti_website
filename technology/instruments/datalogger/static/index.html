정적 데이터 로거

정적 데이터 로거의 구성, 설치, 운영, 데이터 흐름 및 현장 유지관리를 정리한 기술 자료입니다.

1. 시스템 개요

  • 목적정적 데이터로거는 변위·수위·하중·간극수압·온도 등 느리게 변하는 신호를 분~시간 단위로 스캔·저장하는 현장 계측 장치입니다. 토목·지반 자동계측에서 가장 널리 쓰이며, 본 기술자료 Figure는 범용 산업용 소형 로거(LCD·전면 단자 포트) 외형을 표준으로 합니다.
  • 적용 범위아날로그·디지털·SDI-12·RS-485 입력, 내부 스케줄러, SD·플래시 저장, 12V 전원, 이더넷·셀룰러 통신 옵션으로 구성됩니다. 지중경사계·간극수압계·하중계·침하계 등 다채널 정적 센서와 직접 연동하며, 채널 부족 시 멀티플렉서로 확장합니다.

2. 구성 요소

  • 주기 수집1분~1일 스캔으로 추세·변화량 산정
  • 다채널수십 채널 아날로그·디지털 동시 관리
  • 현장 저장SD·내부 메모리로 통신 단절 대비
  • 경보한계치·이상값 현장 판별
  • 가시설 · 터널 · 사면 다채널 자동계측
  • 댐 · 교량 · 철도 장기 모니터링 현장
  • 연약 지반 성토 · 압밀 추세 수집
  • 원격 모니터링 전 단계 현장 저장 · 백업

3. 현장 구성·원리

  • 구성 흐름센서 체인 → (선택) 멀티플렉서 → 정적 데이터로거 → 현장 SD·플래시 저장 → LTE·이더넷 → 서버. 진동·발파 구간은 동적 DAQ를 별도 두고 시간 동기화합니다.
  • 측정 대상정적 로거는 프로그램(스캔 테이블)에 따라 채널을 순차 또는 병렬로 읽습니다. ADC 해상도·입력 범위·여기·필터가 측정 정밀도를 좌우하며, 전원·온도 드리프트는 캘리브레이션·보정으로 관리합니다. MUX 연결 시 스캔 한 주기에 모든 채널을 읽는 데 수 초~수 분이 소요될 수 있어, 계측 주기 설계에 MUX 지연을 반영합니다.
  • 측정 원리외형은 밝은 회색 배선판형 인클로저·12V 전원 포트·나사 단자 블록이 특징이며, 방수 함체 안에 설치해 야외 현장에서 운용합니다.

4. 설치·운영

  • 채널 · 센서 · MUX 매핑 및 스캔 주기 설계
  • 12V 전원 · 태양광 · 배터리 · 접지
  • 방수 함체 · 케이블 글랜드 · 서지 보호
  • 초기 영점 · 캘리브레이션 · 안정화 측정
  • 통신(이더넷 · LTE) · 원격 설정
  • 수동계측 백업 절차 유지

자주 묻는 질문

대표 외형은?

본 기술자료 Figure는 범용 산업용 소형 로거(LCD·단자 포트)를 표준으로 합니다. 선정 시 채널·프로토콜·전원·통신 호환을 우선합니다.

동적 로거와 병행?

발파·진동 고속 계측은 동적 DAQ를 별도 두고, 침하·수위 등은 정적 로거로 운영하는 혼합 구성이 일반적입니다.

MUX 없이 가능한 채널 수?

로거 내장 아날로그 채널(통상 8~16)과 디지털·버스 센서 수에 따릅니다. 초과 시 MUX 또는 추가 로거를 검토합니다.

SD 카드 백업?

통신 단절·서버 장애 대비 로컬 저장 후 재전송합니다. 카드 용량·보존 기간을 설계에 반영합니다.
