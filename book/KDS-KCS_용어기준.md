## 1. 적용 범위

- 기술자료 SPA (`scripts/content-data/`, `js/technology/`)
- ImageWorks 프롬프트·이미지 라벨·ALT
- `AGENTS.md`, `README.md`, SEO·autolink

KDS/KCS에 없는 용어를 신규 정의하지 않는다. **웹·이미지·SEO에는 타사 제품명·상표(ACE, ACE-TCS, Campbell 등)를 절대 쓰지 않는다** — KDS 표준 용어(**내공변위계** 등)만 사용.

## 2. 터널 계측 항목 (KDS 4.1.5)

| 표준 용어 | 정의 요약 | 대표 계측기 |
|-----------|-----------|-------------|
| **지표 및 지중침하** | 터널 상부 연직 변위 (토피·지상시설물 조건) | 지표침하계, 지중침하계 |
| **내공변위** | 굴착에 따른 **터널 벽면(상반·측벽 아치)** 변위 | **내공변위계** |

**IMG-008 Figure 주의:** 터널 **하부 노반(도로·철도)** 은 계측 대상이 아니다. 360° 원형 폐합 배치는 현장 불가 — Figure·웹에 표현 금지.
| **천단침하** | 천단부 **연직 침하**; **터널 외부 수준점** 기준, 지표침하와 비교 | **천단침하계** |
| **지중변위** (터널) | **굴착면 주변** 심도별 **반경방향** 변위 | **지중변위계** |
| **막장전방 선행변위** | 막장 전방 지반 침하·불량구간 사전 인지 | 선행변위계 |
| **록볼트 축력** | 반경방향 변위에 수반되는 축력 | 록볼트 축력계 |
| **숏크리트 응력** | 라이닝 응력·배면 지반압 | 숏크리트 응력계 |
| **강지보 응력** | 강지보(스틸 세트 등) 축력·변형 | 변형률계, 축력계 |

**공통:** 내공변위·천단침하 **동일 단면**; 지중변위·록볼트·숏크리트·강지보도 동일 단면 종합 해석.

## 3. 가시설·사면 등 (KDS 표 4.1-1)

| 표준 용어 | 용도 |
|-----------|------|
| **지중경사계** | 심도별 **수평변위** 프로파일 (굴착·사면·영향권) |
| **지중변위계** | 지중 연직·수평 변위 (문맥별; 터널은 4.1.5.2③) |
| **로드셀** | 앵커 등 **장력** (KDS 표 4.1-1) |
| **하중계** | 버팀보·록볼트 등 축력 (로드셀과 혼동 금지; 문맥 명시) |
| **지표침하판** | 지표 연직 침하 **센서** (KDS 표 4.1-1) |
| **지표침하계** | 지표침하판과 동일 **자동 센서** 계열 — Figure·노드 `settlement-gauge` |

**금지:** 「지중경사계 = 지중변위계 = 인클리노미터」동일시 정의.

### 3a. 지표침하핀 vs 지표침하계 (SETTLE-01)

> 상세: [docs/42](../docs/42-지표침하핀-지표침하계-구분-및-자동계측-정책.md)

| 구분 | 지표침하핀 | 지표침하계 |
|------|------------|------------|
| 정체 | 측량 **표식** | **센서** |
| 계측 | 수준측량·ATS → 표고 변화 | 침하량 직접 → 로거 |
| 본 사이트 | Figure·hero **금지** | **표준** |

**금지:** Figure에서 지표침하계 라벨을 침하핀(T자)에 붙이기 · 「침하핀/연장봉」을 지표침하계 설명에 혼용.

## 4. 관리·시공 용어 (KDS 1.3, KCS 1.3)

| 표준 용어 | 설명 |
|-----------|------|
| **설계예상변위** / **설계예상값** | 설계상 예상 변위 |
| **최대허용변위** / **설계최대허용값** | 안전 확보 범위 내 허용 상한 |
| **관리대상물** | 보호·관리 대상 시설 |
| **굴착심도** | 계측 시점 굴착 깊이 |
| **막장거리** | 막장(굴착면)으로부터의 거리 (KCS, 터널 직경 배수 D) |
| **계측책임자** | 계측 수행·분석 책임 기술자 |

실무 **관리 단계**(주의·조치 등)는 위 용어 **뒤에** 설명할 수 있다. **「1·2·3차 경보」단독 표기 금지.**

### 4a. 출처 표기 (웹·기술문서)

- **레지스트리:** `book/kds-kcs-citation-registry.json` · 실행 계획 [docs/40](../docs/40-KCS-KDS-출처-표기-통합-계획.md)
- **웹:** 기술자료 각 페이지 하단 **「근거 기준」** 블록 (빌드: `npm run build:content`)
- **검증:** `node scripts/validate-citations.mjs` · `npm run verify:local`
- **인용 형식:** `KDS 11 10 15:2025 §4.1.5` — 조항은 PDF 확인 후만 등급 A·B로 표기

|------|------|

## 6. 금지·대체 표

| 금지·비표준 | 표준 대체 |
|-------------|-----------|
| 내공변위(수렴) | **내공변위** |
| 천단침하 = 단면 수축 | **천단침하** = 연직 침하 (외부 수준점) |
| 천단에 침하계만 (단독) | **천단침하계** + 외부 수준점 |
| 터널 지중변위 = 지중경사계만 | **지중변위계**(굴착면 주변) + 영향권 **지중경사계** 구분 |
| 1·2·3차 경보 (단독) | **설계예상변위**·**최대허용변위** + 관리 단계 |
| 굴착면 거리 (단독, 터널) | **막장거리** 병기 |
| LVDT·ACE-TCS (정의문 단독) | **내공변위계** (+ 제품명 괄호) |
